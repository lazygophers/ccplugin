# 已完成任务 (DONE)

本会话或最近完成的任务。任务包含：
- 任务标题 - 清晰准确的任务描述
- 优先级 - P0 / P1 / P2 / P3
- 完成时间 - 任务完成的时间
- 类别 - 功能、文档、重构、bug等
- 说明 - 任务的说明和结果
- 关键交付物 - 重要文件或变更

## 使用规范

参考 [@plugins/task](../../plugins/task/skills/SKILL.md) 中的完整指南。

### 已完成任务格式

```markdown
### [P{N}] {任务标题} ✓

- **完成时间**: {YYYY-MM-DD HH:mm}
- **类别**: {功能/文档/重构/bug}
- **说明**: {任务说明和结果}
- **关键交付物**: 
  - 文件1
  - 文件2
- **关联提交**: {commit hash 或 PR链接}
```

### 存档任务

定期将已完成的任务移动到 `archive/` 目录，按项目/模块组织。

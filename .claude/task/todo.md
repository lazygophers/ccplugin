# 待完成任务 (TODO)

待完成的任务清单。任务应当包含以下信息：
- 任务标题 - 清晰准确的任务描述
- 优先级 - P0 (紧急) / P1 (高) / P2 (普通) / P3 (低)
- 类别 - 功能、文档、重构、bug等
- 描述 - 任务的详细说明

## 使用规范

参考 [@plugins/task](../../plugins/task/skills/SKILL.md) 中的完整指南。

### 添加任务

在此列表下添加任务：

```markdown
### [P{N}] {任务标题}

- **类别**: {功能/文档/重构/bug}
- **描述**: {详细说明}
- **相关文件**: 如需要，列出相关文件或PR
```

### 开始任务

1. 从 TODO 移动到 IN-PROGRESS
2. 记录开始时间（可选）
3. 按照任务说明执行

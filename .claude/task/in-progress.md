# 进行中任务 (IN-PROGRESS)

当前正在执行的任务。每个任务应当包含：
- 任务标题 - 清晰准确的任务描述
- 优先级 - P0 (紧急) / P1 (高) / P2 (普通) / P3 (低)
- 开始时间 - 任务开始的时间
- 类别 - 功能、文档、重构、bug等
- 描述 - 任务的详细说明
- 进展 - 当前的进展情况

## 使用规范

参考 [@plugins/task](../../plugins/task/skills/SKILL.md) 中的完整指南。

### 添加进行中的任务

在此列表下添加任务：

```markdown
### [P{N}] {任务标题}

- **开始时间**: {YYYY-MM-DD HH:mm}
- **类别**: {功能/文档/重构/bug}
- **描述**: {详细说明}
- **进展**: {当前进展百分比 或 具体进展描述}
- **相关文件**: 如需要，列出相关文件或PR
```

### 完成任务

1. 更新任务状态，标记为完成
2. 从 IN-PROGRESS 移动到 DONE
3. 记录完成时间

## 会话进行中的任务

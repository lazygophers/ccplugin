# 命令交互模式

## 交互方式

命令支持通过 `AskUserQuestion` 工具与用户交互：

```yaml
---
name: create
description: 创建新项目
---

# 创建新项目

我将帮您创建新项目。

首先，请回答几个问题：

1. 项目类型
   - Web 应用
   - CLI 工具
   - 库/包

2. 项目名称

3. 选择的技术栈
```

## AskUserQuestion 使用

### 简单选择

```yaml
---
name: configure
description: 配置项目设置
---

# 配置项目设置

请选择配置选项：

- 语言: [Python, JavaScript, Go, Rust]
```

### 多项选择

```yaml
---
name: setup
description: 设置开发环境
---

# 设置开发环境

请选择要安装的功能：

- [x] 测试框架
- [ ] 代码格式化
- [x] 类型检查
- [ ] CI/CD 配置
```

### 文本输入

```yaml
---
name: new
description: 创建新文件
---

# 创建新文件

请提供文件名（不含扩展名）：
```

## 交互流程

```yaml
---
name: interactive-command
description: 演示交互模式
---

# 交互命令示例

首先，请选择操作类型：
- [创建] 创建新资源
- [更新] 更新现有资源
- [删除] 删除资源
- [查询] 查询资源
```

## 交互后处理

根据用户回答执行不同操作：

```yaml
---
name: process
description: 处理用户选择
---

# 处理选择

根据您的选择，我将：

1. 如果选择"创建"：
   - 收集必要信息
   - 创建资源
   - 返回结果

2. 如果选择"更新"：
   - 获取现有数据
   - 收集更新内容
   - 应用更改
```

## 交互指南

1. **分步提问**：避免一次性问太多问题
2. **清晰选项**：选项描述要明确
3. **提供默认值**：合理的默认值加速交互
4. **确认操作**：危险操作需要二次确认

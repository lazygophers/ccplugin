# 背景整面蓝金光斑交融 — PRD

## 目标
skein-bg-ocean-blend-gold 的上下分割波浪效果不对 (只有中间一条波纹带, 上下纯色块)。用户要**整面背景都是蓝金交融** — 多块柔和光斑交错互混 (类似大理石纹/水彩晕染), 无上下分割线, 接近参考图。成功: 浏览器刷新后整面背景呈现蓝金多块光斑散布交融, 无明显色块分区, 无单一波纹带。

## 用户确认
- 交融方式: **光斑交错互混** (蓝金多块柔和 radial-gradient 散布全屏, 非上下分割, 非对角, 非全屏波浪)

## 根因 (前版问题)
- 基底 `linear-gradient(180deg, 蓝 0-40%, 金 60-100%)`: 上下大色块, 只中间 40-60% 渐变 → 大片纯色
- 波浪层 `circle 240px` 只在 `repeat-x` (横向一行, 垂直 50% 位置): 只出一条横向波纹带, 不铺满
- 两者叠加 = 上下纯色 + 中间一条波纹, 非"整面交融"

## 方案
body 背景改为**多块 radial-gradient 光斑散布全屏** (蓝斑 + 金斑交错, 不同位置/大小/透明度), 去掉上下分割 linear 基底 (改纯底色)。body::after 流沙微光增强交融流动感。

## 边界
- 范围内: `src/input.css` body 背景 (L98-109) + wave-flow keyframes (L124-128, 若不再用波浪层可删或保留); `dist/app.css` 重建
- 范围外: body::after 流沙微光 (L110-118, 保留增强交融); body::before 织纹 (L148-154); 暗模式金沙 (L132-146); 状态色/surface-bg/--skein-gold (已 #E8C264 不动); logo/card/顶栏配色; legacy board/themes/skein.css
- 约束: 纯 CSS, 无 JS, GPU 友好 (光斑数适中, 避免过多层 radial-gradient 拖性能)

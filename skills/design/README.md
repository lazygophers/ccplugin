# design 技能分类

设计类 skill 合集: 以 HTML 为唯一媒介的视觉产出工作台。

| skill | 用途 |
|---|---|
| [html-design](html-design/) | HTML-first 设计工作台——UI/UX 界面、数据可视化、图像设计、演示排版四模式，所有产出以 HTML 落地，其余格式从 HTML 打包导出 |

## 路由

- App / 网页原型 / 布局 / 组件 / 交互 → UI/UX 界面模式
- 架构图 / 流程图 / 时序图 / 信息图 / 数据图 → 数据可视化模式
- 封面 / 配图 / 小红书卡片 / 海报 → 图像设计模式
- PPT 幻灯片 / PDF 报告 / 微信图文 → 演示排版模式

## 设计哲学

所有视觉产出先落到 standalone HTML，再按需导出为 PNG / SVG / PDF / 幻灯片 / 微信图文。不为每种目标格式各开独立流程——它们都是 HTML 的下游。

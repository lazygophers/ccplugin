# 图表选型 · 25 类数据可视化决策表

> 来源：ui-ux-pro-max-skill 的 `charts.csv`，25 种图表类型覆盖趋势/对比/占比/分布/地理/流程/KPI/层级/金融/网络/实时/NLP/流程挖掘。
> 每条给出：**数据类型 / 最佳图表 / 备选 / 适用场景 / 禁用场景 / 数据量阈值 / 配色指引 / 无障碍等级 / 降级方案 / 库推荐 / 交互级别**。
> 用法：用户描述数据特征 → 本表给图表选型；配色具体走姊妹 skill `design-color`；UX 规则走 [rules.md §10](rules.md)。

## 无障碍等级说明

| 等级 | 含义 |
|------|------|
| **AAA** | 最佳，文字标签默认可见，色盲安全 |
| **AA** | 良好，需注意色盲补充(图案/形状) |
| **B** | 中等，必提供表格/列表降级 |
| **C** | 差，必提供表格替代作主视图 |
| **D** | 极差，不可作产品主图表，必提供完全替代 |

## 全局规则

- **a11y ≤C 的图表**(饼/树图/旭日/桑基/网络/3D/词云)：必提供**表格/列表作主视图**，图表仅作视觉补充
- **a11y D 级**(网络图/3D 空间)：**永不作产品主图表**，必完全替代
- **色盲**：永不只靠颜色——加图案/形状/线型/标签冗余
- **数据量**：超阈值必先聚合/采样/分页，禁全量渲染卡性能
- **交互**：hover/缩放/下钻等交互必同时提供键盘可达路径(配合 [rules.md §1 §10](rules.md))

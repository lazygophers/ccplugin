# 行业推理规则 · 100 垂直行业设计决策表

> 来源：ui-ux-pro-max-skill 的 `ui-reasoning.csv`（v2.0 旗舰特性），覆盖 100 个垂直行业的产品类型。
> 每条给出：**推荐模式 / 风格优先级 / 配色氛围 / 字体氛围 / 关键效果 / 决策规则 / 反模式**。
> 用法：用户说出产品领域 → 本表给完整设计系统骨架（配色具体 hex 走姊妹 skill `design-color`）。

## 使用方法

1. **按领域查表**：找最接近用户产品的行（如「美容院」→ Beauty/Spa/Wellness）
2. **取七要素**：模式定信息架构，风格定视觉语言，配色/字体定调性，效果定动效，决策规则定 if-then，反模式定禁区
3. **跨 skill 协作**：配色具体 hex 走 `design-color` 的 themes-catalog / palettes-catalog；布局骨架走 `design-uiux` 各媒介 layout.md
4. **三方向门不豁免**：即使用本表，仍要在该行业语境内出 3 个差异化方向（见 SKILL.md 硬门）
5. **配合 scenarios.md**：14 主流场景的深度 UX 建议在本目录 [scenarios.md](scenarios.md)；本表是行业→风格/配色的快速决策，scenarios 是场景→页面/流程的深度指南，两者互补

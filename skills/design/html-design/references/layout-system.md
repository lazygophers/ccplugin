# 布局系统 · 响应式与场景自适应

布局是骨架。好的布局在不同屏幕、不同密度、不同信息量下都不塌。

## 间距尺度（Spacing Scale）

用固定阶数，禁任意数值。推荐 4px / 8px 基数：

```
--space-1: 4px    (微距：图标内、字间距)
--space-2: 8px    (紧凑内边距)
--space-3: 12px
--space-4: 16px   (标准内边距)
--space-6: 24px
--space-8: 32px   (区块间距)
--space-12: 48px
--space-16: 64px  (大留白)
--space-24: 96px  (段落级分隔)
```

所有 padding / margin / gap 取自这个阶。一致性 > 创意。

## 栅格（Grid）

```css
.container { max-width: 1200px; margin: 0 auto; padding: 0 16px; }
.grid { display: grid; gap: var(--space-4); }
```

常用列数：

| 场景 | 列数 | 列宽 |
|------|------|------|
| 内容阅读 | 1 / 12 | 文行长 60-80 字符 |
| 卡片墙 | auto-fit + minmax | 自适应填充 |
| 后台表格 | 12 栅格 | 自由跨列 |
| 移动 | 4 栅格 | 简化 |

`repeat(auto-fit, minmax(280px, 1fr))` 是响应式卡片墙的银弹——容器变宽自动加列，无需媒体查询。

## 响应式断点

采用移动优先（min-width 上叠）：

| 断点 | 宽度 | 布局变化 |
|------|------|---------|
| 默认 | <640px | 单列、堆叠、抽屉导航 |
| sm | ≥640px | 双列开始、横排按钮 |
| md | ≥768px | 侧栏出现、栅格展开 |
| lg | ≥1024px | 三列、固定侧栏、桌面密度 |
| xl | ≥1280px | 多列大屏、放宽 max-width |

每个断点改的是「布局结构」，不是「换个颜色」。结构变化：列数 / 导航形态 / 侧栏显隐 / 密度切换。

## 场景自适应（同一产物的多语境切换）

### 亮 / 暗模式

```css
:root { --bg:#fff; --text:#111; --card:#f5f5f5; }
[data-theme="dark"] { --bg:#0d1117; --text:#e6e6e6; --card:#161b22; }
body { background:var(--bg); color:var(--text); }
```

暗模式要点：降饱和、提文字对比、阴影改柔（暗底上硬阴影看不见，用亮边框 / 微亮区分层）。

### 密度切换（紧凑 / 舒适）

```css
[data-density="compact"] { --space-unit: 0.75; }  /* 所有间距 ×0.75 */
[data-density="cozy"]     { --space-unit: 1; }
[data-density="comfortable"] { --space-unit: 1.25; }
```

数据密集型后台给「紧凑」选项；消费 App 默认「舒适」。

### 容器查询（组件级自适应）

组件比视口更适合用容器查询自适应——同一个卡片在侧栏窄、主区宽，应自动改形态：

```css
.card { container-type: inline-size; }
@container (min-width: 400px) { .card .media { display:block; } }
@container (max-width: 200px) { .card .meta { display:none; } }
```

## 视觉层级（布局的灵魂）

布局的目的是引导视线。层级工具（从强到弱）：

1. **尺寸**：大 > 小，差距要明显（1.5× 以上才读得出）
2. **留白**：相关元素贴近，不相关拉开（格式塔接近性）
3. **对比**：色 / 重 / 背景差异
4. **位置**：左上 > 右下（阅读起点 > 终点）

自检：眯眼看布局，焦点是否落在你想要的位置？多个等权元素会互相打架——让一个赢。

## 常见布局模式

| 模式 | 何时用 |
|------|--------|
| 圣杯（顶栏+侧栏+主+尾） | 后台、文档站 |
| F 型 | 文字密集页（扫描习惯） |
| Z 型 | 落地页（引导转化动线） |
| 卡片墙 | 等权内容集合（商品、文章） |
| 分屏 | 对比 / 双主区 |
| 居中聚焦 | 单任务（登录、表单、海报） |

## 布局自检清单

- [ ] 所有间距取自尺度阶，无魔法数
- [ ] 移动 → 桌面断点结构合理，无横向滚动
- [ ] 视觉焦点唯一明确
- [ ] 亮 / 暗模式均可读（如项目要求）
- [ ] 密度切换不破坏对齐（如项目要求）

# UI 风格总清单 · 67 种风格选型

> 来源：ui-ux-pro-max-skill 的 `styles.csv`（67 种 UI 风格）+ README 风格目录。
> 分三类：**49 通用风格** + **8 落地页风格** + **10 BI/仪表板风格**。
> 每条给出：风格名 / 适用场景 / 禁用场景。高频风格附关键属性（主色 / 亮暗模式 / 性能 / a11y / 移动端 / 转化）。
> 用法：根据产品类型查 [industry-rules.md](../../design-uiux/references/ui-ux/industry-rules.md) 得推荐风格 → 本表查风格详情 → 配色具体 hex 走 [themes-catalog](themes-catalog.md) / [palettes-catalog](palettes-catalog.md)。

## 风格属性图例

| 标记 | 含义 |
|------|------|
| 亮/暗 ✓Full | 完整支持该模式 |
| 亮/暗 ◐Partial | 部分支持，需额外适配 |
| 性能 ⚡Excellent/⚡Good/⚠Good/❌Poor | 渲染性能等级 |
| a11y ✓WCAG AAA / ⚠需注意 / ❌不友好 | 无障碍等级 |
| 移动/转化 ✓High / ◐Medium / ✗Low | 移动端友好度 / 转化导向 |

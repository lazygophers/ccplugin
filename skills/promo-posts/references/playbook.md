# 通用文案方法论 playbook

跨平台共用的"把素材榨成帖子"的手艺。先提内核, 再按平台调性套结构。

## 1. 提内核(改写前必做)

把素材榨成 5 槽位(SKILL.md 第一动作), 再补两条:
- **一句话价值** / **目标受众** / **可信证据** / **CTA 目标(单一)** / **禁忌**
- **+ 差异点**: 它和现成替代品比, 凭什么(技术社区尤其要这个)。
- **+ 真实细节**: 一个具体的数字、踩坑、使用场景——这是"去 AI 腔"和"建立可信"的共同原料。

## 2. 两种文案骨架(按平台选)

| 骨架 | 适用 | 结构 |
| --- | --- | --- |
| **PAS** | 种草/问答/痛点驱动(小红书/知乎/即刻) | Problem(戳痛点)→ Agitate(放大代价)→ Solution(给方案=你的东西) |
| **AIDA** | 技术发布/thread(X/HN/GitHub) | Attention(hook)→ Interest(怎么work/差异)→ Desire(demo/数据/证据)→ Action(低摩擦 CTA) |

## 3. 标题 / 开头 hook 公式

**技术阵营(具体、不党)**:
- `Show HN: 工具名 – 一句话做什么`
- `我写了个 X 来解决 Y(附 demo)`
- 里程碑+教训: "Week 12 building X: hit $2K MRR, learned [意外教训]"
- 反共识/具体数字 + "here's how"

**种草问答阵营(钩子、痛点前置)**:
- 痛点直击: "终于解决了 [具体痛点], 后悔没早知道"
- 数字+方案: "3 步搞定 X" "5 个 X 神器"
- 反差/身份: "零基础也能 X" "打工人必备"
- 知乎回答开头: 先给反常识结论或边界("先说结论: …""先问是不是")

> 每次给 2–3 个标题备选(标题决定打开率), 让用户挑。

## 4. CTA 三级(按平台容忍度选)

| 级 | 形态 | 用在 |
| --- | --- | --- |
| **L1 邀互动** | "欢迎拍砖/求反馈/评论区聊聊/收藏" | 所有平台通吃, 技术社区只用这级 |
| **L2 轻引导** | "try it free, no signup""star if useful""阅读原文""看我主页" | X(链接放 reply)/GitHub/知乎(站内)/公众号(阅读原文) |
| **L3 强转化** | "立即购买/下载/注册" | 仅大众种草平台**软**用; 技术社区禁用; 中文平台禁站外导流 |

**铁律**: 中文平台(小红书/知乎/公众号)CTA **一律站内化**, 绝不出现微信/QQ/二维码/裸外链(违规)。

## 5. 改写流程(一稿 → 多平台)

1. 提内核(§1)。
2. 每平台: 选骨架(§2)→ 写 hook(§3)→ 填正文(按该平台 `references/` 详档的体量与调性)→ 选 CTA 级(§4)。
3. 套该平台格式硬约束(字数/标签/图片比例)。
4. 过禁忌: 反 spam 红线 + 极限词(§6)+ 去 AI 腔(`anti-ai-detection.md`)。
5. 标注交付: 平台 / 2-3 标题备选 / 正文 / 标签 / 配图建议 / CTA。

## 6. 广告法极限词黑名单(中文平台强制过滤)

《广告法》第九条禁绝对化用语, 小红书/公众号/电商类尤其查。生成中文推广文案**必须扫一遍**:

- **"最"系**: 最、最佳、最好、最大、最高、最高级、最低价、最便宜、最受欢迎、最先进、最新科技
- **排名/权威系**: 第一、唯一、全球领先、全网最低、国家级、世界级、顶级、王牌
- **资质系**: 中国驰名商标、特供、专供、老字号、质量免检、国家免检
- **抢购煽动系**: 秒杀、抢爆、再不抢就没了、错过就没机会、万人疯抢
- **欺骗系**: 恭喜获奖、全民免单、点击有惊喜、点击领取
- **夸大功效系**(小红书重点): 一分钟见效、7 天美白、根治、无副作用 等无依据承诺

> nuance: 并非所有含"最/第一"都违法——绝对化用语**未指向所推销商品、不误导消费者**时不适用(《广告绝对化用语执法指南》)。但生成时**从严**: 命中即换体验化表达("我用下来最顺手"也慎用, 改"我用下来很顺手")。

## 7. 通用反 spam 共性(所有平台)

无论哪个平台, 这几条触发器一致: 索要点赞/投票、多账号刷量、跨平台复制粘贴同一文案、把商业引流伪装成个人分享、营销夸大腔。生成时全部规避。

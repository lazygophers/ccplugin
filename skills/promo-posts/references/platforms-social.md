# 大众种草 / 问答阵营详档

通用调性: 口语、共情、利他、第一人称真实体验、场景化。硬广腔与官方通稿腔失分。
> 中文平台普遍受《广告法》极限词约束 + 严禁站外导流。红线标「官方」, 算法数字标「经验」。核对至 2026-06。

## 小红书(RED)

- **受众**: 偏年轻、女性主导、一二线高线城市、消费决策前会搜索。语言生活化、精致、攻略向。
- **格式硬约束**: 标题约 ≤20 字(建议 ≤18, 手机两行, 前 18 字嵌 2 个核心关键词); 正文约 ≤1000 字(长文功能需邀约); 图片 ≤18 张, 封面 **3:4(1080×1440)**且全组比例统一; 话题标签 3–10 个相关词(官方无硬上限数字, 勿编造上限)。
- **算法(CES, 官方披露)**: `点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8`——**关注权重最高, 评论/转发次之**。"收藏多但无真实评论"可能被判低质限流。完播率/深读率是隐性生死线。
- 🔴 **官方红线**:
  - **广告法极限词**(最/第一/唯一/国家级/全网最低/顶级/秒杀类/无依据功效承诺)→ 限流删帖罚款。**强制过滤**(见 playbook 极限词表)。
  - **严禁任何站外导流**(《交易导流违规管理细则》2025-03-12): 笔记/评论/主页禁留或暗示微信/QQ/二维码/口令/他平台账号/外链; 成交只能走商城/直播购物车/官方留资组件。违规 → 限流→限发→封号。
  - 商业内容须走**蒲公英报备** + 申明利益相关; 对比/效果图须标"原相机+拍摄时间"。
  - AI 辅助创作未主动标识违反《社区公约 2.0》。
- **CTA 得体(口诀: 引导互动可以、引导离开平台不行)**: "姐妹们觉得有用记得**收藏**""**评论区**告诉我你想看啥""**关注**我蹲后续"。**禁**"加微信/私聊领取/扫码"。
- 调性: 第一人称真诚体验、低门槛友好("学生党/打工人友好")、痛点钩子开头。禁夸张猎奇标题、禁炫富擦边伪科学。

## 知乎

- **受众**: 一二线高学历知识型中产, 懂行、会查证、反感被当韭菜。推广帖要**经得起追问**。
- **载体选择**: 专业长文首选**回答**(吃问题长尾+搜索流量+高赞排序, 无自定义标题靠开头钩子)或**文章**(沉淀); **想法**(>50 字)做轻量预热/站内引流。富文本图文(无标准 Markdown 保证)。
- **调性**: 「专业、认真、友善」。"先问是不是, 再问为什么"——推广若建立在夸大/错误前提上会被直接拆台。反"抖机灵"(纯段子无信息易折叠)。开头数十字定生死, 配数据/案例/引用。
- 🔴 **官方红线**(《社区管理规定》): 禁未注明的淘宝客链接/跳转外链诱导; 禁伪造经历冒充他人; 禁多账号配合发广告; 禁刷量造假; 禁大量 SEO 推广链接博取曝光。处罚: 禁言 1/7/15 天至永封 + 删除/折叠/限流。**瓦力 AI 折叠**触发: 答非所问、雷同重复、凑字数、过短。
- **利益相关必须前置披露**; 严格商业推广须显著标"广告"(法律口径, 仅写"利益相关"可能不合规)。
- **CTA 得体(仅站内化)**: "觉得有用点个赞, 方便更多人看到""想了解更多看我主页其他回答/专栏""持续更新 XX 领域, 感兴趣可关注"。**禁**留微信/QQ/二维码、裸贴购买链接、"私信发资料"。

## 微信公众号

- **受众**: 主动关注的已有信任私域用户, 长文容忍度高; 公域推荐弱, **转发率是命脉**。人群以本号后台「用户分析」为准, 不写死。
- **格式硬约束(官方)**: 标题上限 **64 字**(列表/封面遮挡, 经验 ≤13 字不被遮、正标题 ≤20); 摘要 ≤120 字; 正文+图片合计 ≤50000 字; 图片 ≤500 张, 群发每次 ≤8 篇。封面头条 **900×383(2.35:1)**, 核心信息居中(防 383×383 裁切)。
- ⚠️ **无 # 话题标签机制——不要给公众号帖生成话题标签**。选题热度用「微信指数」。
- 🔴 **官方红线(《运营规范》)**:
  - **诱导分享**: 禁"分享后才能看/解锁"、集赞、助力、抽奖、"不转不是中国人/转发平安"等话术(即使不出现"转发"字眼)。
  - **诱导关注**: 禁"关注才能阅读/解锁/扫码点餐"、关注得红包。
  - **外链限制**: **仅"已认证+开通微信支付的服务号"正文可插外链**; 订阅号/普通号正文外链无效。变通: 放底部**「阅读原文」**(无此限制)或二维码长按识别。
  - 处罚: 文章禁转发/删文/删粉/封能力至封号。
- **CTA 得体**: 柔性无门槛——"觉得有用欢迎转发给需要的朋友""点个在看""评论区聊聊""想了解更多点阅读原文"。跨平台引导用"去公众号搜 XXX / 点名片"比贴外链稳妥。**禁**任何"转发才能…/集赞/关注解锁"。

## 即刻(Jike)

- **受众**: 年轻、互联网/产品/创投/技术圈, 重真诚人际连接, 反感商业味。运营会亲自下场互动, 人情味重。
- **格式**: 短动态(moment)为主, 类朋友圈; 字数上限无官方公开数字(需核实)。**圈子机制=主要曝光渠道**: 发动态必选对口圈子(不加圈子很难被发现)。
- **调性**: 真诚、轻松、口语化、沙雕幽默、第一人称真实分享。**禁端着/官腔/硬广腔/通稿**。
- **CTA 得体**: "我做了个 X, 解决了我自己的 Y 问题, 感兴趣的可以聊聊" + 发进对口圈子 + 提问引发评论("大家会怎么做?")。**禁**"重磅发布!点击下载!"式通稿、与圈子无关的硬链接刷屏。**把推广写成真实使用故事 + 邀请讨论**最得体。

## 掘金(juejin)— 补充(技术写作, 跨阵营但归技术调性)

- 互动权重: **评论/收藏/分享 > 点赞 > 阅读**(单刷阅读无效); 发布后黄金数小时窗口集中拿互动; 看率(完读/点赞/评论率)非绝对数; 账号"掘力值"决定新文起跑线。
- 容忍 GitHub/开源链接(价值延伸非硬广); 禁付费课/拉群/公众号导流硬广、抱团互赞刷量(降权)。
- CTA: "点赞收藏吃灰""评论区交流方案""代码已开源: GitHub 链接""有错误欢迎指正"。

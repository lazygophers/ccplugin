# code-quality 技能分类

代码质量类 skill 合集: 架构设计、代码整洁、性能优化。

| skill | 用途 |
|---|---|
| [architecture-design](architecture-design/) | 用「正交分解」方法论做架构设计与评审 |
| [clean-code](clean-code/) | 基于《代码整洁之道》的代码审查与改写顾问 |
| [perf-optimization](perf-optimization/) | 跨栈性能优化方法论框架 |

## 路由

- 架构选型 / 模块边界 / 接口设计 → architecture-design
- 已有代码挑违规 / 写代码时实时整洁化 → clean-code
- 为什么慢 / P99 高 / 该不该优化 → perf-optimization

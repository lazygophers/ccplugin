Retro - 复古怀旧设计风格规范

复古风格设计融合 80-90 年代审美，采用温暖色调、曲线排版和复古字体，唤起怀旧情感。
## 核心特征

**色彩系统**：
```css
/* 温暖调色板 */
--retro-red: #ff6b6b;      /* 复古红 */
--retro-orange: #ffa600;   /* 橙色 */
--retro-yellow: #ffde00;   /* 黄色 */
--retro-purple: #c06c84;   /* 复古紫 */
--retro-cream: #f4e4c1;    /* 奶油色 */
--retro-brown: #6f5d46;    /* 棕色 */
```

```css
/* 复古字体 */

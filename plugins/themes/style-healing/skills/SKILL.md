# 治愈系极简实用风 (Healing Minimalist Practical Style)

## 设计哲学

**核心理念**：每件物品应是生活的点缀，而不是未来的枷锁

治愈系极简实用风是一种融合了：
- **极简扁平**（基底）：视觉无冗余、布局清晰、元素扁平
- **轻量情感化**（温度）：手绘图标、柔和线条、轻量动效
- **实用主义**（核心）：操作便捷、信息清晰、流程简化
- **治愈系细节**（差异点）：圆润设计、柔和色彩、温和反馈

### 圆角规范

```

中等组件       8dp

```

### 间距规范

```
页面内边距     16dp
组件间距       ≥8dp
内容区域留白   ≥30%
卡片内边距     12dp
```

### 字体规范

**字重**：仅允许两种
```
常规 (Regular)    400
半粗 (SemiBold)   600
```

**字体家族**：
- 英文/数字：MapleMono / SF Mono / JetBrains Mono
- 中文：苹方 / 思源黑体 / Noto Sans CJK SC

### 动效规范

**时长限制**：
```
快速反馈       150ms  (按压缩放)
标准过渡       200ms  (淡入淡出)
页面切换       300ms  (滑动动画)
```

**缓动函数**：
```
ease-out       大多数场景
ease-in-out    页面切换
```

**允许的动效**：
- ✅ 按压缩放动画（缩放至98%）
- ✅ 滑动动画（从右向左）
- ✅ 淡入淡出（透明度渐变）

**禁用的动效**：
- ❌ 旋转动画
- ❌ 闪烁效果
- ❌ 放大超过105%
- ❌ 复杂过渡效果

### 阴影规范

```
模糊半径       4dp
透明度         10%
偏移量         0dp (无偏移，扁平化)
```

**原则**：极轻量阴影，保持扁平感

## 设计原则

### 1. 留白优先

内容区域留白占比 ≥ 30%，确保呼吸感

### 2. 功能至上

每个设计元素必须有明确功能，避免纯装饰性元素

### 3. 温和反馈

所有交互都应有温和的视觉反馈（缩放、透明度、颜色变化）

### 4. 一致性

全局使用统一的圆角、间距、字重、动效参数

### 5. 情感温度

通过柔和色彩、圆润设计、轻量动效传递温暖感

## 适用场景

✅ **适合**：
- 生活类应用（物品管理、日记、习惯追踪）
- 效率工具（任务管理、笔记、知识管理）
- 健康应用（冥想、睡眠、运动）
- 社交应用（轻量社交、兴趣社区）

❌ **不适合**：
- 游戏/娱乐（需要强视觉冲击）
- 金融/企业（需要严肃专业感）
- 儿童应用（需要更鲜艳色彩）

## 设计验证清单

- [ ] 色彩100%使用莫兰迪色系（低饱和、灰色调）
- [ ] 圆角遵循16dp/8dp/4dp三级规范
- [ ] 留白占比≥30%
- [ ] 字重仅使用400/600两种
- [ ] 动效时长≤300ms
- [ ] 无高饱和度色彩
- [ ] 无尖锐棱角（最小圆角4dp）
- [ ] 无过度阴影（模糊≤4dp，透明≤10%）
- [ ] 无复杂装饰元素

# 治愈系极简实用风 (Style Healing)

## 设计理念

**核心哲学**：每件物品应是生活的点缀，而不是未来的枷锁

治愈系极简实用风是一种融合了极简扁平、轻量情感化、实用主义和治愈系细节的现代UI设计风格。

基于 [encount](https://github.com/lazygophers/encount) 项目的设计系统提炼而成。

## 设计特征

### 视觉设计

- **莫兰迪配色**：低饱和度、灰色调、温暖感
- **极简扁平**：视觉无冗余、布局清晰、元素扁平
- **圆润设计**：16dp/8dp/4dp 三级圆角规范
- **充足留白**：内容区域留白占比 ≥ 30%

### 交互模式

- **轻量动效**：时长 ≤ 300ms，缓动函数 ease-out/ease-in-out
- **温和反馈**：按压缩放（98%）、淡入淡出、滑动动画
- **极轻阴影**：模糊4dp、透明10%、无偏移

### 设计原则

1. **留白优先** - 确保呼吸感
2. **功能至上** - 每个元素必须有明确功能
3. **温和反馈** - 所有交互都应有视觉反馈
4. **一致性** - 全局统一的设计参数
5. **情感温度** - 通过柔和色彩传递温暖感

## 适用场景

✅ **适合**：
- 生活类应用（物品管理、日记、习惯追踪）
- 效率工具（任务管理、笔记、知识管理）
- 健康应用（冥想、睡眠、运动）
- 社交应用（轻量社交、兴趣社区）

❌ **不适合**：
- 游戏/娱乐（需要强视觉冲击）
- 金融/企业（需要严肃专业感）
- 儿童应用（需要更鲜艳色彩）

## 使用方式

安装插件后，设计规范将自动激活。当进行UI设计时，Claude Code会参考以下内容：

1. **色彩选择** - 使用莫兰迪色系
2. **圆角规范** - 遵循16dp/8dp/4dp三级
3. **间距留白** - 确保充足的留白空间
4. **动效设计** - 使用轻量、简洁的过渡效果
5. **组件设计** - 遵循统一的组件规范

## 设计验证清单

使用本风格时，请确保：

- [ ] 色彩100%使用莫兰迪色系
- [ ] 圆角遵循三级规范
- [ ] 留白占比≥30%
- [ ] 字重仅使用400/600
- [ ] 动效时长≤300ms
- [ ] 无高饱和度色彩
- [ ] 无尖锐棱角
- [ ] 无过度阴影

Vibrant - 充满活力设计风格规范

充满活力风格设计采用高饱和色彩、大胆对比和动态布局，创建充满能量和视觉冲击的 UI。
## 核心特征

**色彩系统**：
```css
/* 高饱和色彩 */
--vibrant-red: #ff0000;
--vibrant-blue: #0066ff;
--vibrant-yellow: #ffff00;
--vibrant-green: #00ff00;
--vibrant-purple: #cc00ff;
--vibrant-orange: #ff6600;
```

**视觉特性**：
- 饱和度：85-100%
- 亮度：50-90%
- 对比度：≥7:1

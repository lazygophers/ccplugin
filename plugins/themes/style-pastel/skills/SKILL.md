Pastel - 柔和粉彩设计风格规范

柔和风格设计采用淡雅色彩、圆角和柔和阴影，创建温柔、舒适的视觉体验，适合内容应用。
## 核心特征

**色彩系统**：
```css
/* 淡雅调色板 */
--pastel-pink: #ffb3ba;    /* 淡粉 */
--pastel-blue: #bae1ff;    /* 淡蓝 */
--pastel-yellow: #fffacd;  /* 淡黄 */
--pastel-green: #c1ffc1;   /* 淡绿 */
--pastel-purple: #e6ccff;  /* 淡紫 */
--pastel-cream: #fefef8;   /* 乳白 */
```

**视觉特性**：
- 饱和度：30-50%
- 亮度：80-95%
- 圆角：12-24px

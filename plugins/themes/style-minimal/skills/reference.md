# Minimal - 极简主义设计风格规范

极简主义设计强调留白、清晰的排版和功能至上的原则，优先级排序和信息层次。

## 核心特征

**设计原则**：
- 只保留必要元素
- 充足的留白（50% 以上）
- 优先级明确
- 无装饰元素

**颜色系统**：
```css
--white: #ffffff;
--black: #000000;
--gray-50: #fafafa;
--gray-100: #f3f4f6;
--gray-900: #111827;
```

**排版**：
```css
--font-family: "Inter", -apple-system, sans-serif;
--font-size-base: 16px;
--line-height: 1.5;
--letter-spacing: 0.02em;
```

## 实现要点

**布局原则**：
```css
/* 留白管理 */
--spacing-unit: 8px;
--spacing-xs: 4px;
--spacing-sm: 8px;
--spacing-md: 16px;
--spacing-lg: 24px;

/* 网格系统 */
--grid: repeat(12, 1fr);
--gap: 24px;
```

**组件简化**：
- 按钮：仅文字 + 背景或边框
- 卡片：极简边框，无阴影
- 输入框：下边框式设计
- 导航：线性布局，无图标

## 应用场景

- 📰 新闻网站
- 📚 内容平台
- 💼 企业官网
- 📝 写作应用

## 色彩约束

```
主色：1-2 个
辅助色：1 个
中性色：完整灰阶（5-9 级）
强调色：0-1 个（仅用于 CTA）
```

## DO & DON'T

✅ **DO**:
- 大量留白增加可呼吸感
- 精确的对齐和间距
- 清晰的信息优先级
- 可用的默认状态

❌ **DON'T**:
- 装饰图标或图形
- 过多的动画
- 太多颜色变化
- 忽视信息架构

Minimal - 极简主义设计风格规范

极简主义设计强调留白、清晰的排版和功能至上的原则，优先级排序和信息层次。
## 核心特征

**设计原则**：
- 只保留必要元素
- 充足的留白（50% 以上）
- 优先级明确
- 无装饰元素

**颜色系统**：
```css
--white: #ffffff;
--black: #000000;
--gray-50: #fafafa;
--gray-100: #f3f4f6;
--gray-900: #111827;
```

# Knowledge Plugin 快速开始

> 版本：0.1.0
> 基于向量数据库的知识管理系统

## 安装

```bash
cd plugins/knowledge
uv venv && source .venv/bin/activate
uv pip install -e ".[dev]"
```

## 配置

```json
{
  "plugins": [
    {
      "path": "/path/to/ccplugin/plugins/knowledge"
    }
  ]
}
```

## 基本使用

### 添加知识

```python
# 简单添加
knowledge_add(content="Python 最佳实践文档")

# 带来源和元数据
knowledge_add(
    content="asyncio 使用指南：避免阻塞 I/O",
    source="官方文档",
    metadata={"category": "Python", "level": "advanced"}
)
```

### 搜索知识

```python
# 简单搜索
knowledge_search(query="Python 异步编程")

# 限制结果
knowledge_search(query="最佳实践", limit=5)
```

# Knowledge Plugin 工具参考

> 版本：0.1.0 | 工具：2

## knowledge_add - 添加知识

添加知识到向量数据库（v0.5.0+ 实现 ChromaDB 存储）。

### 参数

| 参数 | 类型 | 必需 | 默认值 | 说明 |
|------|------|------|--------|------|
| `content` | string | ✅ | - | 知识内容 |
| `source` | string | ❌ | `""` | 知识来源 |
| `metadata` | object | ❌ | `{}` | 元数据 |

### 示例

```python
# 添加文档
knowledge_add(
    content="Python type hints 使用指南：...",
    source="PEP 484",
    metadata={"language": "Python", "topic": "类型系统"}
)

# 添加代码片段
knowledge_add(
    content="异步上下文管理器示例：async with ...",
    source="项目代码",
    metadata={"type": "code", "category": "patterns"}
)
```

## knowledge_search - 搜索知识

语义搜索知识（v0.5.0+ 实现向量检索）。

### 参数

| 参数 | 类型 | 必需 | 默认值 | 说明 |
|------|------|------|--------|------|
| `query` | string | ✅ | - | 搜索查询 |
| `limit` | integer | ❌ | `10` | 返回数量限制 |

### 示例

```python
# 概念搜索
knowledge_search(query="依赖注入模式")

# 代码搜索
knowledge_search(query="如何实现单例模式", limit=3)

# 文档搜索
knowledge_search(
    query="Python 性能优化技巧",
    limit=10
)
```

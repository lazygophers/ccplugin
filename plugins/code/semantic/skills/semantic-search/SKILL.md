代码语义搜索

当用户需要以下操作时，使用此技能：
- 查找代码实现（"如何实现 xxx"）
- 查找 API 用法（"xxx 怎么用"）
- 查找代码示例（"xxx 的示例代码"）
- 查找特定功能的代码（"处理 xxx 的代码"）

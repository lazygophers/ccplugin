### Git 操作插件

**Git 仓库管理插件**

用于 Git 仓库操作的专用插件，提供以下功能：

- **提交管理** - 基于暂存变更智能提交
- **推送管理** - 安全地推送到远端，支持代理重试
- **忽略文件管理** - 智能更新 .gitignore
- **Pull Request 管理** - 创建和更新 PR

## 使用方式

### 提交相关

```bash
# 智能提交（基于暂存变更生成提交信息）
/commit
```

### 推送相关

```bash
# 推送到远端（失败时自动尝试代理）
/push
```

### 忽略文件管理

```bash
# 根据未提交内容更新 .gitignore
/ignore
```

### Pull Request 管理

```bash
# 创建或更新 PR
/pr
```

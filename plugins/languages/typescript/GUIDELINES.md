所有 TypeScript 代码必须遵守以下 Skills 规范：
- Skill(typescript-core) - 核心规范：TS 6.0+、严格模式、强制约定
- Skill(typescript-types) - 类型系统规范：类型安全、泛型、类型守卫
- Skill(typescript-async) - 异步编程规范：async/await、Promise
- Skill(typescript-react) - React 开发规范：React 18+、Hooks
- Skill(typescript-nodejs) - Node.js 开发规范：Node.js 20+、ESM
- Skill(typescript-security) - 安全编码规范：输入验证、XSS 防护

每一个 `*.ts`、`*.tsx` 文件都不得超过500行，推荐200~400行

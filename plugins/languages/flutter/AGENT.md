所有 Flutter 代码必须遵守以下 Skills 规范：
- Skills(flutter:core) - 核心规范：Dart 3.2+、强制约定
- Skills(flutter:state) - 状态管理规范：Provider、Riverpod、BLoC
- Skills(flutter:ui) - UI 开发规范：Widget 组合、构建优化
- Skills(flutter:android) - Android 开发规范：Material 3、性能优化
- Skills(flutter:ios) - iOS 开发规范：Cupertino、性能优化
- Skills(flutter:web) - Web 开发规范：Web 性能优化

每一个 `*.dart` 文件都不得超过600行，推荐200~400行

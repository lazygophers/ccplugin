所有 Flutter 代码必须遵守以下 Skills 规范：
- Skills(flutter:core) - 核心规范：Dart 3.x 特性、Clean Architecture、工具链
- Skills(flutter:state) - 状态管理规范：Riverpod 3.x（推荐）、Bloc 8.x
- Skills(flutter:ui) - UI 开发规范：Material 3、Cupertino、响应式布局、Impeller
- Skills(flutter:android) - Android 开发规范：Material 3 Expressive、Impeller、权限
- Skills(flutter:ios) - iOS 开发规范：Cupertino、Impeller、App Store 规范
- Skills(flutter:web) - Web 开发规范：WASM 编译、CanvasKit、PWA、SEO

每一个 `*.dart` 文件都不得超过600行，推荐200~400行

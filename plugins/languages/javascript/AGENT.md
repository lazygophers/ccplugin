所有 JavaScript 代码必须遵守以下 Skills 规范：
- Skills(javascript:core) - 核心规范：ES2024-2025 标准、强制约定
- Skills(javascript:async) - 异步编程规范：async/await、Promise
- Skills(javascript:react) - React 开发规范：React 18+、Hooks
- Skills(javascript:vue) - Vue 开发规范：Vue 3、Composition API
- Skills(javascript:security) - Web 安全规范：XSS 防护、CORS

每一个 `*.js`、`*.jsx` 文件都不得超过500行，推荐200~400行

所有 C# 代码必须遵守以下 Skills 规范：
- Skills(csharp:core) - 核心规范：C# 12/.NET 8 标准、强制约定
- Skills(csharp:async) - 异步编程规范：async/await、CancellationToken
- Skills(csharp:linq) - LINQ 规范：查询语法、集合操作
- Skills(csharp:web) - Web 开发规范：ASP.NET Core、Minimal API
- Skills(csharp:desktop) - 桌面开发规范：WPF、MAUI、MVVM
- Skills(csharp:data) - 数据访问规范：Entity Framework Core

每一个 `*.cs` 文件都不得超过600行，推荐200~400行

所有 Python 代码必须遵守以下 Skills 规范：
- Skills(python:core) - 核心规范：PEP 8、命名规范、代码格式
- Skills(python:error) - 错误处理规范：异常处理、日志记录
- Skills(python:types) - 类型提示规范：类型注解、mypy
- Skills(python:testing) - 测试规范：pytest、覆盖率
- Skills(python:async) - 异步编程规范：asyncio、并发模式
- Skills(python:web) - Web 开发规范：FastAPI、Pydantic

每一个 `*.py` 文件都不得超过500行，推荐200~400行

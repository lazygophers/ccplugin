# Rust 开发规范参考资源

## 官方文档

- [The Rust Programming Language](https://doc.rust-lang.org/book/) - Rust 官方书籍
- [Rust Reference](https://doc.rust-lang.org/reference/) - Rust 语言参考
- [Rust Standard Library](https://doc.rust-lang.org/std/) - 标准库文档
- [Rust Edition Guide](https://doc.rust-lang.org/edition-guide/) - Edition 指南

## 最佳实践

- [API Guidelines](https://rust-lang.github.io/api-guidelines/) - Rust API 设计指南
- [Rust Design Patterns](https://rust-unofficial.github.io/patterns/) - Rust 设计模式
- [The Rustonomicon](https://doc.rust-lang.org/nomicon/) - unsafe Rust 指南
- [Rust by Example](https://doc.rust-lang.org/rust-by-example/) - Rust 示例

## 异步编程

- [Async Rust](https://rust-lang.github.io/async-book/) - 异步 Rust 书籍
- [Tokio Documentation](https://tokio.rs/) - Tokio 运行时文档
- [Futures Crate](https://docs.rs/futures/) - Futures 库文档

## 性能优化

- [The Rust Performance Book](https://nnethercote.github.io/perf-book/) - Rust 性能书籍
- [Cargo Book - Profiling](https://doc.rust-lang.org/cargo/reference/profiles.html) - Cargo 性能分析
- [Flamegraph](https://github.com/flamegraph-rs/flamegraph) - Rust 火焰图工具

- [Cargo](https://doc.rust-lang.org/cargo/) - 包管理器
- [rustfmt](https://github.com/rust-lang/rustfmt) - 代码格式化
- [Clippy](https://github.com/rust-lang/rust-clippy) - Lint 工具
- [Miri](https://github.com/rust-lang/miri) - 未定义行为检测

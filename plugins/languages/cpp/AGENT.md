所有 C++ 代码必须遵守以下 Skills 规范：

- Skills(core) - 核心规范：C++17/23 标准、强制约定
- Skills(memory) - 内存管理规范：智能指针、RAII
- Skills(concurrency) - 并发编程规范：原子操作、线程、协程
- Skills(template) - 模板编程规范：类型萃取、Concepts
- Skills(performance) - 性能优化规范：Cache优化、SIMD
- Skills(tooling) - 工具链规范：CMake、静态分析

每一个 `*.cpp`、`*.h` 和 `*.hpp` 文件都不得超过600行，推荐200~400行

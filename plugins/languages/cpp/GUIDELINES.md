All C++ code must follow these Skills:
- Skill(cpp-core) - Core: C++20/23 features, mandatory conventions
- Skill(cpp-memory) - Memory: smart pointers, RAII, scope guards
- Skill(cpp-concurrency) - Concurrency: jthread, coroutines, atomics, latch/barrier
- Skill(cpp-template) - Templates: concepts, CTAD, fold expressions, constexpr/consteval
- Skill(cpp-performance) - Performance: cache optimization, SIMD, zero-copy
- Skill(cpp-tooling) - Toolchain: CMake 3.28+, clang-tidy, sanitizers, Conan 2.x/vcpkg

Every `*.cpp`, `*.h`, and `*.hpp` file must not exceed 600 lines (200-400 recommended).

所有 C 代码必须遵守以下 Skills 规范：
- Skills(c:core) - 核心规范：C11/C17 标准、强制约定
- Skills(c:memory) - 内存管理规范：内存分配、泄漏检测
- Skills(c:concurrency) - 并发编程规范：原子操作、线程
- Skills(c:posix) - POSIX API 规范：文件系统、进程、网络
- Skills(c:embedded) - 嵌入式开发规范：寄存器、中断
- Skills(c:error) - 错误处理规范：errno、perror

每一个 `*.c` 和 `*.h` 文件都不得超过600行，推荐200~400行

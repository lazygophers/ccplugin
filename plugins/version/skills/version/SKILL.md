版本号管理最佳实践

本指南规范项目版本管理，采用 Semantic Versioning (SemVer) 标准，确保版本号的一致性和可追踪性。
**核心原则**：除非用户明确要求，否则任何场景（命令无参数、hooks 自动触发）都默认更新 **Build 版本号**。

# Context Plugin 工具参考

> 版本：0.1.0 | 工具：2

## context_save - 保存上下文

保存会话上下文到数据库（v0.3.0+ 实现 SQLAlchemy 存储）。

### 参数

| 参数 | 类型 | 必需 | 默认值 | 说明 |
|------|------|------|--------|------|
| `session_id` | string | ✅ | - | 会话ID |
| `content` | string | ✅ | - | 上下文内容 |
| `role` | string | ❌ | `"user"` | 角色（user/assistant/system） |

### 示例

```python
# 用户输入
context_save(
    session_id="proj-001",
    content="如何实现异步编程？",
    role="user"
)

# AI 回复
context_save(
    session_id="proj-001",
    content="使用 asyncio...",
    role="assistant"
)
```

## context_retrieve - 检索上下文

检索会话历史上下文（v0.3.0+ 实现数据库查询）。

### 参数

| 参数 | 类型 | 必需 | 默认值 | 说明 |
|------|------|------|--------|------|
| `session_id` | string | ✅ | - | 会话ID |
| `limit` | integer | ❌ | `20` | 返回数量限制 |

### 示例

```python
# 检索最近 20 条
context_retrieve(session_id="proj-001")

# 检索最近 50 条
context_retrieve(session_id="proj-001", limit=50)
```

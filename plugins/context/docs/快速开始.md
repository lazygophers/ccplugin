# Context Plugin 快速开始

> 版本：0.1.0
> 会话上下文持久化和恢复系统

## 安装

```bash
cd plugins/context
uv venv && source .venv/bin/activate
uv pip install -e ".[dev]"
uv run pytest -v
```

## 配置

编辑 `~/.claude/settings.json`：

```json
{
  "plugins": [
    {
      "path": "/path/to/ccplugin/plugins/context"
    }
  ]
}
```

## 基本使用

### 保存上下文

```python
# 保存用户输入
context_save(
    session_id="session-123",
    content="用户的问题",
    role="user"
)

# 保存 AI 回复
context_save(
    session_id="session-123",
    content="AI 的回答",
    role="assistant"
)
```

### 检索上下文

```python
# 检索会话历史
context_retrieve(
    session_id="session-123",
    limit=20
)
```

## 验证

在 Claude Code 中测试：

```
请保存这条上下文到会话 test-001
```

预期响应：
```
已保存上下文到会话 'test-001' (角色: user, 内容长度: XX)
```

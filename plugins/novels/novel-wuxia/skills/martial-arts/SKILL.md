# 武功招式设计

## 功能概述

提供完整的武侠世界武功招式设计框架，涵盖内功心法、外功招式、轻功身法、暗器毒术、兵器武学等，确保武功体系层次分明且具备武侠韵味。

## 使用场景

- 设计完整的武功等级体系
- 创建具体的武功招式和心法
- 构建兵器谱和兵器武学
- 设计轻功身法体系
- 规划暗器、毒术等辅助武学

## 核心要点

### 武功等级体系

| 层次 | 说明 | 代表 | 特征 |
|------|------|------|------|
| 绝世神功 | 武学巅峰 | 独孤九剑、葵花宝典 | 无招胜有招，近乎天道 |
| 顶级武学 | 一流之上 | 降龙十八掌、六脉神剑 | 威力惊人，学成者寥寥 |
| 一流武学 | 名门绝学 | 各派镇派之宝 | 练至大成可称一流高手 |
| 二流武学 | 上乘功夫 | 门派核心功法 | 需要较高资质和时间 |
| 三流武学 | 普通功夫 | 基础拳法剑法 | 江湖通用，入门较易 |
| 江湖把式 | 粗浅功夫 | 市井拳脚 | 强身健体，实战有限 |

### 内功心法体系

```
【内功层次】
- 入门：打通任督二脉，感知内力
- 初成：丹田凝聚真气，外放内力
- 大成：内力浑厚，可驱动高深武学
- 化境：内力随心所欲，返璞归真
- 登峰：内力近乎无穷，天人合一

【内功类型】
- 阳刚内功：至刚至猛，如九阳神功
- 阴柔内功：至阴至寒，如玄阴真经
- 中正平和：不偏不倚，如混元功
- 阴阳兼修：刚柔并济，如太极内功
- 邪门内功：速成但有反噬，如吸星大法

【经典内功设计模板】
- 名称：XXX神功/心法/真经
- 层数：通常七层/九层/十二层
- 修炼条件：资质、前置功法、特殊条件
- 每层效果：内力增长、特殊能力
- 禁忌/风险：走火入魔的条件
- 与其他功法的关系：相克/相辅
```

### 外功招式设计

```
【招式结构】
一套完整的武功通常包含：
- 总纲：武学理念和核心原则
- 基础招式：3-5招入门招式
- 核心招式：7-12招精华招式（通常以数字命名）
- 绝招：1-3招压箱底的必杀技
- 变化：每招的变化应用（如一招可衍生多式）

【招式命名风格】
- 意境型：落雁平沙、风卷残云、月照寒潭
- 动物型：白鹤亮翅、猛虎下山、苍鹰扑兔
- 哲理型：无中生有、以退为进、虚实相生
- 数字型：六合八法、三才剑法、七星步

【经典招式设计要素】
- 招式名称：有意境、易记忆
- 招式动作：起手式、运力路径、收势
- 克制与弱点：每招都有破绽
- 连招搭配：招式之间的衔接
- 境界要求：需要什么水平才能使出
```

```

刀、枪、剑、戟、斧、钺、钩、叉、
鞭、锏、锤、戈、镋、棍、槊、棒、拐、流星

【武侠常用兵器】
- 剑：百兵之君，文人侠客最爱
- 刀：百兵之帅，霸道凌厉
- 枪：百兵之王，战场利器
- 棍：百兵之祖，少林看家
- 扇：文雅飘逸，暗藏杀机
- 笛/箫：以音攻人，诡异莫测
- 拳：赤手空拳，至简至朴

【神兵利器设计模板】
- 名称：有气势的名字（如屠龙刀、倚天剑）
- 类型：刀/剑/枪等
- 材质：玄铁、陨铁、百炼精钢
- 外观：长度、颜色、纹饰
- 特殊能力：锋利无匹/自带剑气/削铁如泥

```

### 轻功身法

```
【轻功层次】
- 粗浅：翻墙越壁，跳跃攀爬
- 入门：飞檐走壁，夜行百里
- 精通：踏雪无痕，水上漂行
- 大成：凌空虚度，来去如风
- 化境：缩地成寸，瞬息千里

【经典轻功】
- 梯云纵：武当绝学，如踩云梯直上
- 凌波微步：步法诡异，配合易经八卦
- 草上飞：踏草而行，不折草茎
- 壁虎游墙功：贴壁行走如履平地
- 燕子三抄水：水面轻点三次飞掠而过
```

### 暗器毒术

```
【暗器类型】
- 手掷类：飞刀、飞镖、铁莲子
- 机关类：袖箭、暗弩、机关匣
- 绳索类：绳镖、流星锤
- 特殊类：暴雨梨花针、孔雀翎、含沙射影

【毒术体系】
- 毒药类型：迷药、剧毒、慢性毒、蛊毒
- 用毒方式：涂抹兵器、下毒饮食、毒烟毒雾
- 解毒方式：特定解药、内功逼毒、以毒攻毒
- 著名毒物：鹤顶红、断肠草、七步蛇毒、情花毒
```

## 设计建议

1. **招式有魂**：好的招式不只是动作描写，要有武学理念和哲学内核
2. **克制平衡**：天下武功相生相克，没有无敌的武功
3. **修炼代价**：越强的武功修炼条件越苛刻，甚至有副作用
4. **人物匹配**：武功风格要与角色性格、身份匹配
5. **战斗观赏性**：武打描写要有画面感，节奏有张有弛

## 示例

### 武功设计示例：醉仙剑法

**名称**：醉仙剑法

**类型**：剑法（需配合特制米酒催发）

**层次**：顶级武学

**核心理念**：似醉非醉，亦真亦幻。以醉态迷惑敌人，剑招在摇晃中暗藏杀机。

**招式**：
- 一醉方休：起手式，饮酒入剑，内力随酒气运转
- 醉步蹒跚：身法招式，看似踉跄实则暗合步法
- 杯酒释怀：防御招式，以柔化刚，借力打力
- 醉卧沙场：险招，故意倒地引敌近身，突然爆发
- 举杯邀月：绝招，剑指苍天，化出漫天剑影

**特色**：饮酒越多，剑法越强，但超过极限反而失控。需要极高的内力控制酒意。

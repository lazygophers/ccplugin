# 江湖设定

## 功能概述

提供完整的武侠世界江湖设定框架，涵盖江湖格局、势力分布、武林规矩、恩怨体系、侠客类型等核心要素，构建有厚度和烟火气的武侠世界。

## 使用场景

- 设计武林势力版图和格局
- 构建江湖规矩和武林道义
- 设定武林大会和重大事件
- 设计侠客人物和江湖身份
- 规划恩怨情仇和门派关系

## 核心要点

### 江湖格局体系

```
【正道势力】
  名门正派（少林、武当、峨眉等）
  ├── 武林盟主/正道领袖
  ├── 各大门派掌门
  └── 正道散侠

【邪道势力】
  魔教/邪派
  ├── 教主/总坛
  ├── 旗使/堂主
  └── 邪道高手

【朝廷势力】
  皇权与江湖的交汇
  ├── 锦衣卫/东厂/六扇门
  ├── 皇家护卫
  └── 朝廷鹰犬

【灰色势力】
  亦正亦邪
  ├── 丐帮（天下第一大帮）
  ├── 镖局/商会
  ├── 杀手组织
  └── 盗贼帮派
```

### 江湖规矩与武林道义

```
【核心规矩】
- 武林大会：选举盟主、调停纷争、商议大事
- 英雄帖：邀请武林各方参加重大活动
- 生死状：决斗前签订，生死各安天命
- 比武招亲：以武艺论英雄的婚配方式
- 金盆洗手：退出江湖的仪式

【武林道义】
- 侠义精神：路见不平拔刀相助
- 一诺千金：江湖人重信守诺
- 恩怨分明：有恩必报、有仇必报
- 不伤无辜：武功不可滥用于平民
- 师门恩情：尊师重道、不背叛师门

【江湖禁忌】
- 以多欺少、以大欺小
- 暗器伤人不报名号
- 偷学他门武功
- 杀害同门
- 欺师灭祖
```

### 武林人物类型

```
【侠客类型】
- 大侠：武功高强、行侠仗义，江湖声望极高
- 剑客：以剑为伴，追求剑道至高
- 浪子：放浪形骸、不拘礼法，但内心重情义
- 隐士：归隐山林的绝顶高手
- 少侠：初出茅庐、闯荡江湖的年轻人

【江湖身份】
- 武林盟主：武林正道的精神领袖
- 掌门人：一派之主，承上启下
- 堂主/舵主：帮派中层管理者
- 镖头：走镖护送、行走江湖
- 捕快/六扇门：官方与江湖的交界
- 杀手：受雇杀人的暗夜行者
- 乞丐：丐帮弟子，消息灵通
```

### 江湖场所设计

```
【典型场所】
- 客栈酒楼：江湖消息汇聚之地
- 武林圣地：少林寺、武当山等名门所在
- 黑市/暗巷：地下交易和情报买卖
- 擂台/比武场：武林大会和比武的场所
- 镖局：护镖和武林信息中转
- 药铺/医馆：疗伤治病、获取秘药
- 铁匠铺：打造兵器的场所
- 密室/暗道：藏宝图、武功秘籍的藏匿处

【秘境设定】
- 绝壁洞天：深山绝壁中的隐秘洞府
- 地下暗河：通往秘密基地的水路
- 古墓禁地：埋藏绝世武功的古老墓穴
- 雪山之巅：极寒之地的神秘门派
```

### 恩怨体系设计

```
【恩怨类型】
- 门派世仇：两个门派之间延续数代的仇恨
- 夺宝之争：围绕武功秘籍或神兵利器的争夺
- 灭门之仇：灭门惨案引发的复仇
- 师徒反目：背叛师门的恩怨纠葛
- 情感纠葛：爱恨交织的江湖恩怨
- 误会冲突：因误解导致的恩怨（可反转）

【恩怨升级路径】
个人恩怨 → 门派冲突 → 正邪大战 → 武林浩劫
```

## 设计建议

1. **烟火气息**：江湖不只有打打杀杀，也有市井生活和人间温情
2. **灰度处理**：正派未必全善，邪派未必全恶，人物要有复杂性
3. **规则约束**：江湖规矩是武侠世界秩序的基础，破坏规矩要有后果
4. **历史厚度**：江湖要有历史传承感，前辈的恩怨影响后辈
5. **地域特色**：不同地区的江湖要有当地的风土人情

## 示例

### 江湖格局示例：五岳争锋

**背景**：五岳剑派各据一方，表面和睦实则暗流涌动

**势力分布**：
- 华山派（西岳）：剑法精妙，但内部气宗剑宗之争
- 嵩山派（中岳）：野心最大，图谋合并五岳
- 衡山派（南岳）：琴剑双绝，风雅之派
- 恒山派（北岳）：尼姑掌门，戒律严明
- 泰山派（东岳）：内部派系复杂

**核心冲突**：嵩山派左冷禅图谋合并五岳为一派，各派态度不一，暗中博弈

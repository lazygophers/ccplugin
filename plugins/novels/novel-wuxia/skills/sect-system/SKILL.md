# 门派体系设计

## 功能概述

提供完整的武侠世界门派体系设计框架，涵盖门派等级、内部架构、武学传承、门规戒律、弟子培养等，构建有传承感和文化底蕴的武林门派。

## 使用场景

- 创建新的武林门派
- 设计门派内部架构和等级制度
- 构建武学传承谱系
- 设定门规戒律和门派文化
- 规划门派间的武学交流和竞争

## 核心要点

### 门派等级体系

| 等级 | 说明 | 底蕴要求 | 代表 |
|------|------|---------|------|
| 泰山北斗 | 武林至尊 | 数百年传承，绝顶高手坐镇 | 少林、武当 |
| 名门正派 | 武林支柱 | 百年传承，一流高手数位 | 峨眉、崆峒、昆仑 |
| 一方霸主 | 区域强者 | 数十年基业，二流高手众多 | 各地大帮大派 |
| 普通门派 | 江湖中坚 | 有传承功法，门人数十 | 各种中小门派 |
| 草莽帮派 | 底层势力 | 无完整传承，人多势众 | 山贼、水匪 |

### 门派内部架构模板

```
【核心层】
  掌门人
  ├── 太师叔/太上掌门（隐退长辈）
  └── 长老会（元老级人物）

【管理层】
  ├── 首席大弟子：掌门接班人
  ├── 执法长老：维护门规
  ├── 传功长老：教授武学
  ├── 外务长老：处理对外事务
  └── 内务管事：管理门派日常

【弟子层】
  ├── 嫡传弟子：掌门亲授，学核心武学
  ├── 入室弟子：正式拜师，学门派武学
  ├── 记名弟子：挂名弟子，学基础武学
  ├── 外门弟子：预备弟子，考核通过可升内门
  └── 俗家弟子：不入门派但有师承关系

【辅助】
  ├── 护派武僧/护法：专职保卫
  ├── 杂役/仆从：负责日常杂务
  └── 附属势力：依附的小帮小派
```

### 门派设计模板

```
【门派名称】
- 等级：泰山北斗/名门正派/一方霸主/普通门派
- 所在地：名山大川/城市坊间/隐秘之地
- 创派祖师：创派人物和故事
- 传承年代：建派至今的历史
- 镇派之宝：核心武功秘籍/神兵利器
- 门派特色：以什么武学闻名
- 门派风格：正统/豪放/阴柔/中庸
- 弟子规模：各层级人数
- 门派势力范围：控制的地盘和影响力
- 当前状态：鼎盛/衰落/重建/隐世
- 与他派关系：同盟/敌对/中立
```

### 武学传承谱系

```
【传承方式】
- 师徒传承：最常见，掌门传弟子
- 秘籍传承：前人留下武功秘籍
- 悟道自创：绝顶天才自创武学
- 奇遇得功：机缘巧合获得传承
- 以武会友：切磋中领悟对方武学精髓

【传承规则】
- 核心功法只传嫡传弟子
- 绝学不得外传（破戒是重大事件）
- 每代掌门可在传承基础上改良
- 某些武学有修炼限制（如性别、体质）
- 镇派之宝由掌门代代相传

【传承冲突】
- 武功秘籍失窃：引发门派危机
- 传承断代：掌门突然离世，秘籍失传
- 叛徒外泄：弟子叛逃带走门派武学
- 正邪之辩：门派武学被认定为邪功
```

### 门规戒律设计

```
【通用门规】
- 尊师重道，不得违逆师命
- 同门之间不得私斗伤人
- 门派武学不得私授外人
- 不得借门派名义为非作歹
- 叛出师门者，门派有权追杀

【处罚体系】
- 面壁思过：轻微过失，禁闭反省
- 废去武功：严重违规，废去所学
- 逐出师门：不可饶恕，断绝师徒关系
- 清理门户：叛徒，由门派执法者处决

【特色门规】
- 少林：不可饮酒食肉（但也有例外）
- 武当：不可先动手，后发制人
- 峨眉：男女弟子交往有严格规矩
```

### 门派文化与日常

```
【修炼日常】
- 晨练：集体练功，基本功训练
- 传功：长老授课，教授武学
- 切磋：同门之间的武学交流
- 任务：下山历练、护送、采购
- 闭关：专注修炼某门武功

【门派仪式】
- 拜师大典：正式收徒的仪式
- 掌门传位：新旧掌门交接
- 论功大会：年度武学考核
- 祭祖大典：祭拜创派祖师
- 下山仪式：弟子出师下山闯荡
```

## 设计建议

1. **传承感**：门派要有历史厚度，几代人的传承故事让门派更立体
2. **内部矛盾**：门派内部要有派系之争、理念冲突，增加故事张力
3. **武学特色**：每个门派的武学要有鲜明特色，一眼可辨
4. **人情味**：门派不只是武学组织，也是家庭，师兄弟之间要有情感
5. **兴衰变化**：门派有兴有衰，这种变化本身就是好故事

## 示例

### 门派设计示例：听风阁

**名称**：听风阁

**等级**：名门正派

**所在地**：江南水乡，建于太湖中一座孤岛之上

**创派祖师**：百年前的琴圣"听风居士"，以音律入武，自创听风心法

**镇派之宝**：
- 武功：听风心法（内功）+ 天籁剑法（剑法）
- 神兵：焦尾古琴（祖师遗物，可以琴音伤人）

**门派特色**：
- 以音律和剑法结合闻名，弟子皆精通音律
- 武学风格飘逸灵动，重意境轻蛮力
- 门派中既有男弟子也有女弟子，风气开明

**内部矛盾**：
- "琴派"主张以琴入武，"剑派"主张以剑为主
- 两派争论哪条路才是祖师的正统传承

**当前状态**：表面风平浪静，但掌门年事已高，继承人之争暗流涌动

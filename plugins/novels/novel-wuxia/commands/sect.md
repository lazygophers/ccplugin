# 设计门派体系

帮助作者创建武林门派或构建完整的门派体系。

## 适用场景

| 场景 | 示例 |
|------|------|
| **创建门派** | `/novel-wuxia sect 设计一个以暗器闻名的门派` |
| **门派架构** | `/novel-wuxia sect 设计门派内部等级制度` |
| **武学传承** | `/novel-wuxia sect 设计门派的武学传承谱系` |
| **门规戒律** | `/novel-wuxia sect 设计门派规矩和处罚体系` |
| **门派关系** | `/novel-wuxia sect 设计五大门派的关系网` |

## 执行流程

### 1. 确认设计范围

通过提问明确：
- **设计目标**：单个门派/门派体系/门派关系
- **门派定位**：主角门派/敌对门派/中立势力
- **门派风格**：正统/邪派/隐世/亦正亦邪
- **已有设定**：与已有江湖格局和角色的关联

### 2. 激活技能生成方案

激活 `Skills(sect-system)` 并根据方向生成：

#### 单个门派设计
- 门派名称、等级、所在地
- 创派历史和传承故事
- 核心武学和镇派之宝
- 内部架构和弟子等级
- 门规戒律和门派文化
- 当前状态和面临的挑战

#### 门派体系构建
- 各大门派的实力对比
- 门派间的同盟和敌对关系
- 武学传承的交叉影响
- 门派兴衰的历史脉络

#### 门派文化设计
- 日常修炼和生活节奏
- 重要仪式和传统
- 门派禁忌和潜规则
- 师徒关系和同门情谊

### 3. 整合验证

- 门派实力与武学体系的匹配
- 门派架构的合理性和可操作性
- 门派文化的独特性和记忆点
- 是否为故事冲突提供足够空间

## 设计原则

- **文化底蕴**：门派要有历史厚度和文化特色
- **内部张力**：门派内部要有派系之争，不能铁板一块
- **武学特色**：一派一特色，武学风格要鲜明
- **人情冷暖**：门派也是家庭，有温情也有冷酷

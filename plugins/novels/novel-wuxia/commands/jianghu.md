# 设计江湖设定

帮助作者快速设计或完善武侠世界的江湖设定。

## 适用场景

| 场景 | 示例 |
|------|------|
| **江湖格局** | `/novel-wuxia jianghu 设计正邪两道对立格局` |
| **武林规矩** | `/novel-wuxia jianghu 设定武林大会规则` |
| **侠客人物** | `/novel-wuxia jianghu 设计一位亦正亦邪的大侠` |
| **恩怨设定** | `/novel-wuxia jianghu 两大门派的百年恩怨` |

## 执行流程

### 1. 确认需求方向

通过提问明确：
- **设定类型**：江湖格局/武林规矩/侠客人物/恩怨体系/完整设定
- **世界风格**：传统武侠/新派武侠/暗黑武侠/喜剧武侠
- **时代背景**：古代哪个朝代/架空世界
- **已有设定**：与已有门派和角色的衔接要求

### 2. 生成设定方案

根据需求激活 `Skills(jianghu-setting)` 并生成：

#### 江湖格局
- 正邪势力分布和实力对比
- 朝廷与江湖的关系
- 灰色势力的定位
- 各方势力的核心诉求

#### 侠客人物
- 人物身份和江湖地位
- 武功路线和成长空间
- 性格特征和行事风格
- 与江湖各方的关系

#### 恩怨体系
- 恩怨的起因和发展
- 涉及的各方势力
- 恩怨的升级路径
- 可能的化解或激化方向

### 3. 一致性验证

检查设定的合理性：
- 势力格局是否平衡
- 恩怨关系是否有足够戏剧张力
- 是否为主角成长留足空间
- 江湖规矩是否能驱动情节

### 4. 输出设定文档

以结构化格式输出完整设定，便于后续查阅和扩展。

## 设计原则

- **江湖气息**：设定要有武侠的味道，不能写成宫廷斗争
- **灰度空间**：正邪不是绝对的，人物和势力要有复杂性
- **规矩驱动**：江湖规矩是故事冲突的重要来源
- **烟火人间**：江湖也有柴米油盐，不能只有打打杀杀

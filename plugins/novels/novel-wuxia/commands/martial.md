# 设计武功招式

帮助作者设计具体的武功招式，或构建完整的武学体系。

## 适用场景

| 场景 | 示例 |
|------|------|
| **内功心法** | `/novel-wuxia martial 设计一套阴阳双修的内功` |
| **招式设计** | `/novel-wuxia martial 设计一套以快制胜的剑法` |
| **兵器武学** | `/novel-wuxia martial 设计主角的本命兵器和配套武学` |
| **轻功身法** | `/novel-wuxia martial 设计一套水上轻功` |
| **武功体系** | `/novel-wuxia martial 设计完整的武功等级体系` |

## 执行流程

### 1. 确认设计方向

通过提问明确：
- **设计目标**：单套武功/完整体系/兵器武学/辅助武学
- **使用者**：主角/反派/门派通用/NPC
- **武功风格**：刚猛/阴柔/飘逸/诡异/中正
- **已有设定**：与已有武功体系和角色的匹配

### 2. 激活技能生成方案

激活 `Skills(martial-arts)` 并根据方向生成：

#### 单套武功设计
- 武功名称和层次定位
- 核心理念和武学哲学
- 招式列表和详细描写
- 修炼条件和进阶路线
- 克制关系和弱点

#### 内功心法设计
- 心法名称和层数
- 修炼原理和运功路线
- 每层效果和突破条件
- 与外功招式的配合
- 走火入魔的风险

#### 兵器武学设计
- 兵器名称和来历
- 配套武功和招式
- 兵器特殊能力
- 使用者与兵器的关系

### 3. 整合验证

- 武功风格与角色性格的匹配
- 武功层次与整体武学等级的对应
- 克制关系的合理性
- 战斗描写的观赏性

## 设计原则

- **招式有魂**：好武功不只是动作，要有武学哲学和意境
- **相生相克**：没有无敌的武功，天下武学互有克制
- **人武合一**：武功风格要与使用者的性格、经历匹配
- **修炼代价**：越强的武功代价越大，增加故事张力

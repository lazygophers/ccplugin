# 朝代背景设计

## 功能概述

提供完整的历史架空朝代背景设计框架，涵盖朝代架构、政治制度、经济体系、社会阶层、文化风俗等核心要素，确保历史设定厚重且逻辑自洽。

## 使用场景

- 创建全新的架空朝代背景
- 基于真实朝代进行架空改编
- 设计社会等级制度和阶层流动
- 构建经济体系和商业格局
- 规划文化风俗和礼仪制度

## 核心要点

### 经典朝代参考模板

```
【先秦模式】
  分封制 → 诸侯争霸 → 合纵连横 → 统一趋势

【大一统模式】
  开国立制 → 盛世繁荣 → 积弊渐生 → 衰败变革

【南北分裂模式】
  政权对峙 → 边境冲突 → 文化融合 → 统一或持续分裂

【异族入主模式】
  游牧崛起 → 入主中原 → 汉化改革 → 矛盾调和
```

### 朝代架构设计原则

1. **历史质感**：架空朝代需有真实历史的厚重感，制度设计要有据可循
2. **矛盾内建**：朝代设定中要埋入核心矛盾（中央与地方、文官与武将、皇权与相权）
3. **时代特征**：每个时期有独特的时代气质（尚武/崇文/重商/抑商）
4. **变革空间**：制度和格局要留有被主角改变的余地

### 社会阶层体系

```
【皇族宗室】
- 皇帝、太子、亲王、郡王、公主
- 宗室特权与限制并存
- 宗人府管理和宗法约束

【士族门阀/勋贵阶层】
- 世家大族：掌控文化和政治资源
- 开国勋贵：军功起家，把持军权
- 外戚势力：后宫与朝堂的纽带

【士人阶层】
- 科举出身的文官群体
- 寒门与世家的对立
- 清流与浊流的派系

【商贾阶层】
- 富而不贵的矛盾处境
- 官商勾结与商帮势力
- 海上贸易/丝路贸易

【平民百姓】
- 农民、手工业者、佣工
- 赋税徭役的承受者
- 民变起义的基础

【贱民阶层】
- 奴仆、乐户、罪民
- 阶层流动的最底层
```

### 经济体系设计

```
【农业基础】
- 土地制度：均田制/庄园制/屯田制
- 赋税体系：租庸调/两税法/一条鞭法
- 粮仓储备与灾荒应对

【商业格局】
- 商帮势力划分（地域性商帮）
- 核心贸易路线和商业重镇
- 官营垄断与私营竞争
- 货币体系（铜钱/银两/交子）

【手工业】
- 官营作坊：军器、织造、盐铁
- 民间手工：瓷器、丝绸、茶叶
- 技术垄断与外传禁令
```

### 文化风俗要素

```
【礼仪制度】
- 朝会礼仪、祭祀制度
- 婚丧嫁娶的等级规制
- 服饰颜色和纹饰等级
- 称谓用语的尊卑区分

【文化特色】
- 科举/选官制度的具体设定
- 文学艺术的时代特征
- 宗教信仰和寺庙道观
- 节日庆典与民俗活动

【饮食起居】
- 不同阶层的饮食标准
- 建筑风格和居住规制
- 出行工具和等级限制
```

## 设计建议

1. **以真为骨**：架空朝代以真实历史为骨架，融入创新元素，避免完全脱离历史逻辑
2. **细节真实**：日常生活细节（饮食、服饰、称谓）要贴合时代，增强代入感
3. **矛盾驱动**：朝代设定的核心矛盾就是故事冲突的来源
4. **主角空间**：制度漏洞和变革契机是主角施展才华的舞台
5. **避免现代思维**：人物行为和思维方式要符合古代社会逻辑

## 示例

### 架空朝代示例：大梁王朝

**朝代定位**：融合唐宋特征的架空朝代，开国百年后进入多事之秋。

**核心矛盾**：
- 皇权与世家门阀的博弈
- 文官集团与武将勋贵的对立
- 北方游牧民族的持续威胁
- 南方商帮势力的崛起冲击传统秩序

**时代特征**：
- 科举制已推行但世家仍掌控大量资源
- 商业繁荣但重农抑商政策名存实亡
- 军事上采取守势，边防压力日增
- 文风鼎盛，但朝堂上党争不断

# 权谋设计

## 功能概述

提供完整的历史架空权谋设计框架，涵盖宫廷斗争、朝堂博弈、派系构建、阴谋布局、情报暗战等核心要素，帮助作者设计层次丰富、逻辑严密的权谋情节。

## 使用场景

- 设计朝堂派系对立和博弈
- 构建宫廷权力斗争情节
- 规划长线阴谋和连环计策
- 设计情报网络和暗战体系
- 创建经典权谋桥段和反转

## 核心要点

### 权力博弈层次模型

```
【第一层：明面博弈】
  朝堂辩论 → 政策对立 → 弹劾奏折 → 站队表态

【第二层：暗中角力】
  私下结盟 → 利益交换 → 安插眼线 → 拉拢策反

【第三层：深层布局】
  十年谋划 → 连环计策 → 养寇自重 → 以退为进

【第四层：终极博弈】
  夺嫡之争 → 政变兵变 → 改朝换代 → 禅让退位
```

### 派系构建模板

```
【核心要素】
- 派系领袖：核心人物及其性格特点
- 核心利益：派系存在的根本诉求
- 势力范围：控制的官职、地域、资源
- 内部结构：骨干成员、外围附庸、暗子
- 弱点软肋：可被利用的致命缺陷

【经典派系类型】
- 帝党：拥护皇权，多为孤臣直臣
- 后党：以太后/皇后为核心的外戚势力
- 太子党/诸王党：围绕储位的夺嫡阵营
- 世家门阀：以血缘姻亲为纽带的利益集团
- 清流言官：以道德理想为旗号的文官集团
- 武将勋贵：以军功起家的武人集团
- 宦官集团：依附皇权的内廷势力
- 地方藩镇：割据一方的地方实力派
```

### 经典权谋手段

```
【进攻型】
- 弹劾构陷：搜集罪证或制造罪证
- 经济打击：断其财路，釜底抽薪
- 离间分化：挑拨派系内部矛盾
- 借刀杀人：利用第三方势力消灭对手
- 舆论造势：操控民间舆论施压

【防守型】
- 以退为进：主动示弱降低威胁
- 自污保身：故意展示缺点避免猜忌
- 养望蓄势：低调积蓄力量等待时机
- 多线下注：同时押注多个势力分散风险
- 留后手：预备反制手段以防不测

【布局型】
- 十年树人：提前布局培养门生故吏
- 联姻结盟：通过婚姻捆绑利益
- 把控要害：掌握人事权/财权/兵权
- 情报渗透：在对手阵营安插暗子
- 制造依赖：让他人不得不依赖自己
```

### 阴谋情节设计框架

```
【五步法】
1. 埋线：在不经意处植入伏笔
2. 布局：各方势力逐步到位
3. 触发：关键事件引爆矛盾
4. 反转：出人意料的真相揭示
5. 收网：阴谋的最终结果和余波

【关键原则】
- 阴谋要有合理的动机和充足的准备时间
- 每个参与者的行为要符合其立场和性格
- 阴谋不应是万无一失的，要留有破绽
- 揭露过程要循序渐进，而非一次性倾倒
- 阴谋的后果要影响深远，推动剧情发展
```

### 宫廷斗争要素

```
【后宫权术】
- 争宠手段：恩宠是后宫权力的根基
- 子嗣博弈：皇子的出生和教育是核心筹码
- 家族博弈：后宫斗争实质是前朝势力的延伸
- 宫人网络：太监、宫女构成的情报网

【储位之争】
- 嫡庶之分：嫡长制vs立贤制的矛盾
- 师傅阵营：太子师傅自动成为太子党
- 历练考验：各皇子通过办差展示能力
- 兄弟反目：感情与权力的终极冲突
```

## 设计建议

1. **利益为本**：所有权谋行为必须有清晰的利益动机
2. **智力对等**：不要让反派变蠢来衬托主角，要让对手也有高光
3. **信息差驱动**：权谋的精彩来自于不同角色掌握不同信息
4. **代价真实**：权谋斗争不是零成本的，胜利者也要付出代价
5. **层层剥笋**：真相要一层一层揭开，给读者推理和惊叹的空间

## 示例

### 连环计示例：夺嫡暗战

**背景**：三位皇子争储，主角辅佐最弱的三皇子。

**布局**：
1. 散布流言让大皇子误以为二皇子要构陷他
2. 大皇子先发制人弹劾二皇子的心腹贪腐
3. 二皇子被迫反击，揭露大皇子私通边将
4. 两虎相争之际，三皇子以调和者姿态出现
5. 皇帝厌恶两子相残，开始关注三皇子

**关键**：三皇子全程不出手，只在幕后推波助澜，保持"贤名"。

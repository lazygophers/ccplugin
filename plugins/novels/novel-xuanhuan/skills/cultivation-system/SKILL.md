# 修炼体系设计

## 功能概述

提供完整的修仙世界修炼体系设计框架，涵盖境界划分、突破条件、灵根资质、功法分类、天劫设定等核心要素，确保体系层次分明且逻辑自洽。

## 使用场景

- 设计全新的修炼境界体系
- 规划功法等级与修炼要求
- 设定灵根资质与天赋体系
- 设计突破瓶颈与天劫机制
- 构建修炼资源（灵石、丹药、灵脉等）体系

## 核心要点

### 经典境界体系参考

```
【凡人境】
  练气期 → 筑基期 → 金丹期 → 元婴期

【仙人境】
  化神期 → 合体期 → 大乘期 → 渡劫期

【超脱境】
  仙人 → 金仙 → 大罗金仙 → 混元大罗金仙
```

每个大境界通常分为初期、中期、后期、巅峰四个小阶段。

### 境界设计原则

1. **层次递进**：每个境界的提升要有质的飞跃，不只是量的积累
2. **标志性变化**：每个境界要有标志性的能力或形态变化（如金丹凝聚、元婴出窍）
3. **瓶颈合理**：突破难度逐级提升，每个大境界之间设置明确瓶颈
4. **人数递减**：越高境界修士越少，形成金字塔结构
5. **寿命对应**：境界提升伴随寿命增长，给突破赋予紧迫感

### 灵根资质体系

```
【灵根分类】
- 五行灵根：金木水火土（单灵根 > 双灵根 > 三灵根 > 杂灵根）
- 变异灵根：雷灵根、冰灵根、风灵根等（稀有且强力）
- 天灵根：单一属性纯度极高（万中无一）
- 特殊体质：先天道体、混沌体、万毒之体等

【资质影响】
- 修炼速度：灵根纯度决定灵气吸收效率
- 功法适配：不同灵根适合不同属性功法
- 突破概率：资质越高，突破瓶颈的概率越大
- 上限天花板：杂灵根可能止步于某个境界
```

### 功法体系设计

功法等级划分：

| 等级 | 说明 | 对应境界 | 获取难度 |
|------|------|---------|---------|
| 凡级 | 入门功法 | 练气~筑基 | 容易 |
| 黄级 | 基础功法 | 筑基~金丹 | 一般 |
| 玄级 | 进阶功法 | 金丹~元婴 | 较难 |
| 地级 | 高阶功法 | 化神~合体 | 困难 |
| 天级 | 顶级功法 | 大乘~渡劫 | 极难 |
| 仙级 | 超凡功法 | 仙人以上 | 几乎不可得 |

每级再分为上中下三品。

### 突破与天劫设计

```
【突破机制】
1. 资源积累：灵石、丹药、灵泉等辅助资源
2. 悟道感悟：对天地法则的理解达到门槛
3. 心境修炼：心魔考验，意志磨砺
4. 机缘触发：特殊契机促成突破（战斗顿悟、秘境奇遇）

【天劫体系】
- 金丹天劫：三九雷劫
- 元婴天劫：六九雷劫
- 化神天劫：九九雷劫
- 渡劫飞升：仙劫（更加恐怖的天劫）

劫难强度与修炼功法品质、个人气运相关，功法越强天劫越猛。
```

### 修炼资源体系

```
【灵石】
- 下品灵石 → 中品灵石 → 上品灵石 → 极品灵石 → 仙石
- 1000下品 = 1中品，比例递增

【丹药】
- 聚灵丹、筑基丹、凝金丹、破元丹...
- 丹药品质：一品至九品，品质越高效果越好副作用越小

【灵脉】
- 灵脉等级决定区域灵气浓度
- 宗门选址核心要素
- 高等灵脉可诞生天材地宝
```

## 设计建议

1. **差异化创新**：在经典框架基础上加入独特元素（如融合武道、科技等），避免千篇一律
2. **留白空间**：高境界不必一开始就设定完整，随剧情展开逐步揭示
3. **与角色绑定**：体系要服务于角色成长，主角的每次突破都应是重要情节节点
4. **数值克制**：不要只靠数字堆砌，引入属性克制、特殊体质等增加策略性
5. **世界一致性**：修炼体系要与世界的社会结构、经济体系、势力格局相匹配

## 示例

### 创新体系示例：星辰修炼体系

**核心理念**：修炼者吸收星辰之力，每突破一个境界对应点亮一颗命星。

**境界划分**：
- 星尘境：感应星辰，体内凝聚第一缕星力
- 星芒境：命星初现，可操控星辰之力战斗
- 星环境：星力环绕周身，攻防一体
- 星域境：在体内开辟微型星域
- 星主境：化身星辰本体，掌控一方星域
- 星帝境：可随意创造或毁灭星辰

**特色**：每个修士的命星不同，决定了独特的能力方向（如北辰星对应时间，太阳星对应毁灭等）。

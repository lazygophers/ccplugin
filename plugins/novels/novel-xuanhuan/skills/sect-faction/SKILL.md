# 宗门势力设定

## 功能概述

提供完整的修仙世界势力体系设计框架，涵盖宗门等级、内部架构、势力关系、资源分配、修仙界格局等，构建有层次感和戏剧张力的势力版图。

## 使用场景

- 设计宗门门派的等级划分和内部架构
- 构建修仙界势力格局和政治关系
- 设计秘境、资源点等战略要素
- 规划势力间的冲突与合作
- 设定散修、家族、商会等非宗门势力

## 核心要点

### 宗门等级体系

| 等级 | 说明 | 底蕴要求 | 数量 |
|------|------|---------|------|
| 顶级宗门 | 修仙界霸主 | 渡劫期老祖坐镇 | 极少（3-5个） |
| 一流宗门 | 一方霸主 | 大乘期强者 | 少量（10-20个） |
| 二流宗门 | 区域强者 | 化神/合体期掌门 | 较多 |
| 三流宗门 | 一般门派 | 元婴期长老 | 众多 |
| 小门派 | 小型势力 | 金丹期为主 | 不计其数 |
| 修仙家族 | 血脉传承 | 视家族底蕴 | 遍布各地 |

### 宗门内部架构模板

```
【核心层】
  宗主/掌门
  ├── 太上长老（隐退的高阶修士，镇宗底牌）
  └── 大长老会（最高决策机构）

【管理层】
  ├── 执法堂：维护宗规，惩处违规弟子
  ├── 内务堂：管理宗门日常事务
  ├── 外交堂：处理对外关系
  ├── 丹药堂：炼丹供应
  ├── 器物堂：炼器修缮
  └── 任务堂：发布和管理任务

【传承层】
  ├── 嫡传弟子：掌门/长老亲传
  ├── 内门弟子：正式弟子，可修炼核心功法
  ├── 外门弟子：预备弟子，修炼基础功法
  └── 记名弟子/杂役：底层，负责杂务

【附属】
  ├── 附属家族：依附宗门的修仙家族
  ├── 坊市：宗门经营的交易场所
  └── 凡人势力：宗门在凡人界的延伸
```

### 宗门设计模板

```
【宗门名称】
- 等级：顶级/一流/二流/三流
- 传承历史：建派多少年，有何著名事迹
- 核心功法：标志性功法和传承
- 所在位置：灵脉等级、地理优势
- 镇宗之宝：宗门最强法宝
- 宗门特色：与其他宗门的差异化
- 弟子规模：各层级弟子数量
- 当前状态：鼎盛/衰落/崛起/隐世
```

### 势力关系设计

```
【关系类型】
- 同盟：共同利益下的合作（可能脆弱）
- 敌对：领土、资源、历史恩怨导致的对立
- 从属：大势力对小势力的庇护与控制
- 竞争：实力相近的势力之间的暗中角力
- 中立：不参与争端的势力（往往有特殊地位）

【冲突驱动】
- 资源争夺：灵脉、矿脉、秘境归属
- 传承争端：功法传承、古修遗迹
- 领土扩张：势力范围的扩展与碰撞
- 历史恩怨：先辈间的仇恨延续
- 理念冲突：正邪之分、道路之争
```

### 修仙界格局设计

```
【典型格局】

1. 正邪对立型
   正道联盟 vs 魔道势力，泾渭分明

2. 多强争霸型
   数个顶级宗门各据一方，相互制衡

3. 一超多强型
   一个霸主级宗门维持秩序，众多势力臣服

4. 大乱斗型
   无绝对霸主，各势力混战

5. 隐世格局型
   表面平静，暗中有远古势力觉醒

【多界格局】
- 人界：凡人与低阶修士的世界
- 修仙界：中高阶修士的主要活动区域
- 上界/仙界：渡劫飞升后的更高层次世界
- 魔界/妖界：非人种族的领域
- 秘境/小世界：散布各处的独立空间
```

### 资源与经济体系

```
【灵脉资源】
- 灵脉等级：一品至九品
- 灵脉归属通常决定势力强弱
- 高等灵脉是兵家必争之地

【秘境设定】
- 古修洞府：远古强者留下的遗迹
- 天然秘境：天地自然形成的特殊空间
- 试炼之地：宗门培养弟子的特殊场所
- 禁地：危险但机缘众多的区域

【经济活动】
- 坊市交易：日常买卖灵石、丹药、法宝
- 拍卖会：高价值物品的交易场所
- 任务系统：悬赏、护送、采集等
- 矿脉开采：灵石和炼器材料的来源
```

## 设计建议

1. **金字塔结构**：势力数量从底层到顶层逐渐减少，形成合理的生态
2. **动态平衡**：势力格局不是一成不变的，要留有变化空间供剧情发展
3. **利益关联**：每个势力的行为都要有利益驱动，避免无理由的善恶对立
4. **主角定位**：主角所在宗门的等级决定了故事初期的格局尺度
5. **地理关联**：势力分布要与地理环境挂钩，形成合理的领地版图

## 示例

### 修仙界格局示例：九州格局

**背景**：九州大陆，以中州为核心，八方各州各有霸主

**顶级势力**（三大超级宗门）：
- 天道宗（中州）：正道领袖，传承最为悠久
- 万妖宫（北荒）：妖族最强势力，人妖混居
- 幽冥教（西漠）：亦正亦邪，修炼阴属性功法

**一流宗门**（每州1-2个）：
- 剑宗（东州）：御剑术天下第一
- 药王谷（南疆）：炼丹术独步修仙界
- 天机阁（中立）：情报组织，不参与纷争

**核心冲突**：
- 天道宗与幽冥教的正邪之争
- 万妖宫推动人妖平等引发的各方博弈
- 远古封印松动，上古妖魔即将苏醒的危机

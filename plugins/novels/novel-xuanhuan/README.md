# Novel Xuanhuan Plugin

> 玄幻修仙小说创作插件 - 提供修炼体系、功法设定、法宝系统、宗门架构等玄幻特色功能

## 简介

Novel Xuanhuan 是一个专为玄幻修仙类网络小说设计的 Claude Code 插件，提供从修炼体系到宗门势力的全方位玄幻世界构建支持。帮助作者快速建立逻辑自洽、层次丰富的修仙世界观。

## 核心功能

### 修炼体系设计 (Cultivation System)
- 境界划分与命名
- 突破条件与瓶颈设定
- 天劫与考验机制
- 灵根资质体系

### 法宝装备系统 (Magic Items)
- 法宝品阶分类
- 炼器材料体系
- 法宝成长与升级
- 阵法符箓设定

### 宗门势力架构 (Sect & Faction)
- 宗门等级与排名
- 内部组织架构
- 势力关系与博弈
- 秘境资源分配

## 快捷命令

```bash
# 设计修炼体系
/novel-xuanhuan cultivation

# 创建法宝装备
/novel-xuanhuan magic-item

# 设计宗门势力
/novel-xuanhuan sect
```

## 专业 Agents

```bash
# 修炼体系顾问
@cultivation-agent 设计一套九境修炼体系

# 玄幻世界观设计师
@worldbuilding-agent 构建一个以灵气复苏为背景的修仙世界
```

## 技能体系

| 技能 | 说明 | 适用场景 |
|------|------|---------|
| cultivation-system | 修炼体系设计 | 境界划分、突破条件、灵根资质 |
| magic-items | 法宝装备系统 | 法宝分类、炼器升级、阵法符箓 |
| sect-faction | 宗门势力设定 | 宗门架构、势力关系、资源体系 |

## 使用场景

- 新作品的修仙世界观构建
- 修炼体系的设计与完善
- 法宝装备的分级设定
- 宗门势力的架构设计
- 灵根资质和天赋体系设计
- 仙界/魔界/人界等多界设定

## 创作流程建议

1. **体系设计**：使用 cultivation-system 设计修炼境界体系
2. **世界构建**：使用 sect-faction 规划宗门势力格局
3. **装备设定**：使用 magic-items 建立法宝品阶体系
4. **关联验证**：确保修炼体系、法宝等级、宗门实力三者逻辑自洽

## 与 novel 插件的关系

本插件是 novel 的玄幻修仙类型扩展，专注于该类型的特色设定。建议配合 novel 插件使用，获得完整的创作支持：
- novel：通用创作功能（角色、情节、对话、文风）
- novel-xuanhuan：玄幻特色设定（修炼、法宝、宗门）

# 创建法宝装备

帮助作者设计具体的法宝装备，或构建完整的法宝品阶体系。

## 适用场景

| 场景 | 示例 |
|------|------|
| **创建法宝** | `/novel-xuanhuan magic-item 设计主角本命飞剑` |
| **品阶体系** | `/novel-xuanhuan magic-item 法宝品阶划分` |
| **炼器材料** | `/novel-xuanhuan magic-item 炼器材料体系` |
| **阵法设计** | `/novel-xuanhuan magic-item 设计护山大阵` |
| **丹药体系** | `/novel-xuanhuan magic-item 丹药等级和种类` |

## 执行流程

### 1. 确认设计方向

通过提问明确：
- **设计目标**：创建具体法宝/构建品阶体系/设计阵法/丹药设定
- **使用者**：主角/反派/宗门/NPC 等
- **定位功能**：攻击/防御/辅助/特殊
- **已有设定**：与已有修炼体系和世界观的匹配

### 2. 激活技能生成方案

激活 `Skills(magic-items)` 并根据方向生成：

#### 单件法宝设计
- 名称、品阶、外观描述
- 核心能力和附加效果
- 材料来源和炼制过程
- 成长路线和最终形态
- 与持有者的关联故事

#### 品阶体系构建
- 完整的品阶划分
- 每个品阶的能力标准
- 与修炼境界的对应关系
- 稀缺度和分布规律

#### 阵法/符箓设计
- 阵法类型和等级
- 核心原理和运作方式
- 关键组件（阵基、阵眼等）
- 破阵方法和弱点

### 3. 整合验证

- 法宝与修炼体系的对应关系
- 材料稀缺度与剧情节奏的匹配
- 法宝能力的平衡性
- 是否有足够的特色和记忆点

## 设计原则

- **特色鲜明**：每件重要法宝要有独特的能力和故事
- **品阶严格**：高阶法宝要珍稀，不能泛滥
- **成长绑定**：主角核心法宝要能随修为成长
- **克制关系**：不同类型法宝间存在克制，增加策略性

# Novel Urban Plugin

> 都市言情小说创作插件 - 专注现代都市背景、职场设定、情感线设计

## 简介

Novel Urban 是一个专为都市言情类网络小说创作设计的 Claude Code 插件，提供从都市背景构建到情感线设计的全方位创作支持。无论你是创作总裁文、职场文、豪门文还是青梅竹马文，都能通过本插件快速搭建故事框架。

## 核心功能

### 都市背景设定 (Urban Setting)
- 城市选择和环境构建
- 场景设计和氛围营造
- 社会圈层和阶级壁垒
- 豪门家族和商业帝国架构

### 职场环境设计 (Workplace Design)
- 行业选择和公司架构
- 职场人际关系和权力博弈
- 办公场景和职业习惯
- 职业成长线规划

### 情感线设计 (Romance Line)
- 经典相遇模式
- 恋爱发展阶段规划
- 情感冲突和矛盾设计
- 甜虐节奏控制和CP互动

## 快速开始

### 安装插件

```bash
# 从市场安装
claude plugin install novel-urban@ccplugin-market

# 从本地安装
claude plugin install ./plugins/tools/novel-urban
```

### 使用命令

```bash
# 设计都市背景
/novel-urban setting

# 创建职场环境
/novel-urban workplace

# 设计情感线
/novel-urban romance
```

### 使用 Agents

```bash
# 都市背景顾问
@urban-agent 帮我设计一个上海的豪门背景

# 情感线设计师
@romance-agent 设计从冤家到恋人的感情线
```

## 技能体系

| 技能 | 说明 | 适用场景 |
|------|------|---------|
| urban-setting | 都市背景设定 | 城市构建、场景设计、圈层文化 |
| workplace-design | 职场环境设计 | 行业选择、公司架构、职场冲突 |
| romance-line | 情感线设计 | 恋爱发展、甜虐节奏、CP互动 |

## 适用类型

本插件专注都市言情，支持以下细分类型：

- **总裁文**：霸道总裁×灰姑娘、商业帝国背景
- **职场文**：办公室恋情、上下级暧昧、同事竞争
- **豪门文**：家族联姻、权力斗争、门当户对
- **青梅竹马文**：重逢、暗恋成真、兜兜转转
- **先婚后爱文**：契约婚姻、商业联姻转真爱
- **甜宠文**：全程高糖、宠溺日常
- **虐恋文**：误会纠葛、身份障碍、虐后HE
- **娱乐圈文**：明星恋情、经纪人×艺人
- **破镜重圆文**：前情往事、追妻火葬场

## 创作流程建议

1. **背景构建**：使用 urban-setting 确定城市和核心场景
2. **职场搭建**：使用 workplace-design 设计行业和公司背景
3. **情感规划**：使用 romance-line 设计完整的感情线
4. **融合创作**：将职场线和感情线交织推进

## 最佳实践

### 背景设计
- 基于真实城市特征，增加读者代入感
- 通过具体细节（品牌、地标、消费场景）塑造质感
- 豪门描写重在人情冷暖，非物质堆砌

### 职场描写
- 行业知识要经得起推敲
- 职场冲突结合行业特性设计
- 职场不是背景板，要和感情线交织

### 情感设计
- 心动要有原因，冲突要有逻辑
- 糖虐比例要精准控制
- 虐后必须补偿等量甜蜜
- 双向奔赴，避免单方面付出

## 示例场景

### 设计商业帝国背景

```
用户：我想写一个房地产大亨的总裁文
Agent：@urban-agent 帮我设计一个房地产集团的完整背景

输出：
- 公司名称：盛世地产集团，30年历史
- 架构：董事长（男主父亲）→ 总裁（男主）→ 各部门
- 权力格局：一房持股40%，二房25%暗中争权
- 商业版图：地标豪宅、文旅小镇、商业综合体
```

### 设计初遇场景

```
用户：女主是普通职员，男主是公司总裁
Agent：@romance-agent 设计一个总裁和普通员工的初遇

输出：
- 场景：深夜加班，误入总裁专用电梯
- 冲突点：文件散落，身份悬殊的尴尬
- 反差点：他矜贵冷漠 vs 她狼狈疲惫
- 伏笔：陆庭深让司机送她回家，第二天会上提到她的方案
```

## 常见问题

### 如何避免总裁文写得太老套？
- 在经典框架内加入创新元素
- 避免纯粹的霸道无理，给男主人性化细节
- 女主不要过于弱势，给她独立的事业线

### 豪门背景如何写得有质感？
- 研究真实豪门的生活方式和社交规则
- 细节描写要精准（品牌、场所、礼仪）
- 权谋不能太复杂，读者要看得懂

### 情感线甜虐比例如何把控？
- 根据目标读者群体调整
- 虐的时候给读者希望
- 甜虐要有节奏，不能一直甜或一直虐

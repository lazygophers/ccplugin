# 设计都市背景

帮助作者构建真实可信的现代都市背景，包括城市选择、场景设计、圈层文化和豪门架构。

## 适用场景

| 场景 | 示例 |
|------|------|
| **选择城市** | `/novel-urban setting 上海` |
| **设计场景** | `/novel-urban setting CBD写字楼` |
| **构建豪门** | `/novel-urban setting 豪门家族` |
| **圈层设计** | `/novel-urban setting 金融圈社交` |

## 执行流程

### 1. 确认背景需求

通过提问明确：
- **故事类型**：总裁文/职场文/豪门文/青春校园
- **城市偏好**：具体城市还是虚构城市
- **核心场景**：故事主要发生的场所
- **社会阶层**：主角和主要角色所处的阶层

### 2. 城市环境构建

#### 城市基础设定
- 城市定位（一线/新一线/二线）和城市性格
- 核心区域划分（CBD、老城区、高档住宅区、大学城等）
- 城市地标和标志性场所
- 气候特征和季节变化

#### 生活质感描绘
- 交通动线（通勤方式、路线、耗时）
- 餐饮文化（特色美食、常去的餐厅类型）
- 娱乐消费（商圈、夜生活、周末去处）
- 城市声音和气味（都市特有的感官体验）

### 3. 社会圈层设定

- 角色所处的社会阶层定位
- 圈层内部的社交规则和潜规则
- 不同圈层之间的交汇点和壁垒
- 圈层特有的消费习惯和生活方式

### 4. 豪门架构（如需要）

- 家族历史和发家背景
- 家族成员关系和权力分布
- 商业帝国版图和核心产业
- 家族内部矛盾和继承争夺
- 家族对外的联姻和合作策略

### 5. 场景清单输出

生成故事所需的核心场景列表：
- 每个场景的位置、特征、氛围
- 场景与情节的关联
- 场景之间的空间关系

## 设计原则

- **真实可信**：基于现实城市特征，读者能产生代入感
- **服务情节**：场景设计为故事冲突和情感发展服务
- **细节到位**：通过具体品牌、地名、消费场景增加质感
- **阶层分明**：不同阶层的生活差异要体现在细节中

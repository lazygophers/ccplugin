# 职场环境设计

## 功能概述

提供都市言情小说的职场背景设计方法，包括行业选择、公司架构、职场人际关系、办公场景和职业成长线规划，帮助创作者构建真实可信的职场世界。

## 使用场景

- 选择故事的行业和公司背景
- 设计公司内部架构和职级体系
- 构建职场人际关系网络
- 安排职场冲突和权力博弈
- 规划角色的职业成长路线

## 核心要点

### 行业选择指南

| 行业 | 特色 | 常见冲突 | 适合类型 |
|------|------|---------|---------|
| **金融** | 高压、高薪、精英云集 | 并购大战、内幕交易、业绩竞争 | 总裁文、精英对决 |
| **科技** | 创新、加班、股权激励 | 技术路线争夺、创业成败、资本博弈 | 创业文、极客浪漫 |
| **传媒/娱乐** | 光鲜、八卦、流量至上 | 资源争夺、绯闻危机、幕后操控 | 娱乐圈文、记者文 |
| **医疗** | 严谨、高尚、生死攸关 | 医患矛盾、学术竞争、伦理困境 | 医生文、治愈系 |
| **律政** | 逻辑、对抗、正义灰度 | 案件博弈、法庭对决、道德抉择 | 律师文、悬疑言情 |
| **时尚** | 前卫、竞争、审美至上 | 设计抄袭、品牌竞争、时装周压力 | 时尚圈文、设计师文 |
| **建筑/设计** | 理想、坚持、艺术商业平衡 | 甲方博弈、竞标、理念冲突 | 文艺言情、理想主义 |

### 公司架构设计

**大型集团架构**：

```
董事长/CEO
  ├── 总裁/COO（日常运营）
  │   ├── 各事业部总经理
  │   └── 职能部门总监
  ├── CFO（财务线）
  ├── CTO（技术线）
  └── 董事会/股东（幕后博弈）
```

**创业公司架构**：

```
创始人/CEO
  ├── 联合创始人/CTO
  ├── 核心团队（早期员工）
  └── 投资方（天使/VC，干预决策）
```

**职级与称谓**：实习生 → 专员 → 主管 → 经理 → 总监 → VP → SVP → C-level

### 职场关系网络

**核心关系类型**：

1. **上下级关系**：严厉型上司、温和型导师、暗恋型领导
2. **同事关系**：竞争对手、闺蜜搭档、暗中使绊子的同僚
3. **跨部门关系**：合作中的摩擦、不同立场的交锋
4. **师徒关系**：职场导师的提携与考验
5. **客户关系**：甲方刁难、商务谈判、长期合作

**职场权力博弈模式**：

```
明面竞争：业绩排名、晋升竞聘、项目主导权
  ↓
暗线博弈：站队选择、信息战、资源争夺
  ↓
高层角力：派系斗争、董事会投票、控制权争夺
```

### 职场冲突类型

1. **业绩竞争型**：同期入职的对手，争夺同一个晋升名额
2. **理念冲突型**：对项目方向有根本分歧，各执一词
3. **权力倾轧型**：上司打压、空降兵与老员工的矛盾
4. **利益纠葛型**：商业间谍、内部腐败、利益输送
5. **性别歧视型**：职场天花板、偏见与证明自己

### 职业成长线设计

**成长弧线模板**：

```
菜鸟期：初入职场，能力不足但有潜力
  → 关键事件：被前辈指点/犯错后成长
  ↓
磨砺期：承担重要项目，遭遇重大挫折
  → 关键事件：项目危机/被人算计
  ↓
崛起期：凭实力证明自己，获得认可
  → 关键事件：漂亮翻盘/独立拿下大项目
  ↓
巅峰期：成为行业精英或创业成功
  → 关键事件：与爱情线交汇，事业爱情双丰收
```

### 办公场景细节

**经典办公场景**：
- **会议室**：方案汇报、唇枪舌剑、暗流涌动
- **茶水间**：八卦传播、偶遇搭话、小团体密谋
- **加班深夜**：共处一室的暧昧、送夜宵的关心
- **出差同行**：同住酒店的尴尬、异地的相互依靠
- **公司年会**：盛装出席惊艳全场、醉酒后的真心话

## 实践建议

1. **行业知识要扎实**：职场细节不到位会让读者出戏
2. **冲突要专业化**：结合行业特性设计冲突，而非套用通用模板
3. **职场不是背景板**：职场线要和感情线交织推进
4. **避免降智**：职场精英的行为逻辑要匹配其身份
5. **成长要有代价**：每次进步都应付出对应的努力或牺牲

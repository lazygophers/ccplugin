# 设计情感线与心理刻画

帮助作者快速设计或完善角色的情感发展弧线和心理刻画。

## 适用场景

| 场景 | 示例 |
|------|------|
| **情感弧线** | `/novel-romance emotion 设计女主从抗拒到心动的心理变化` |
| **内心独白** | `/novel-romance emotion 写一段暧昧期的内心戏` |
| **情感层次** | `/novel-romance emotion 设计高冷男主的深层情感` |
| **心理刻画** | `/novel-romance emotion 角色发现自己吃醋时的心理活动` |

## 执行流程

### 1. 确认需求方向

通过提问明确：
- **设计类型**：情感弧线/内心独白/心理刻画/情感层次
- **角色类型**：角色的性格特征和感情经历
- **当前阶段**：处于感情发展的哪个阶段
- **情感基调**：甜蜜/纠结/虐心/温暖

### 2. 生成设计方案

根据需求激活 `Skills(emotional-arc)` 并生成：

#### 情感弧线设计
- 感情发展的阶段划分
- 每个阶段的心理状态变化
- 关键转折点和触发事件
- 情感变化的合理性验证

#### 心理刻画方案
- 内心独白的内容和风格
- 情感外化的具体表现
- 环境映射的使用方式
- 深层与表层情感的对比

### 3. 一致性验证

检查设计的合理性：
- 情感变化是否符合角色设定
- 心理描写是否有真实感
- 情感节奏是否自然流畅
- 是否与已有剧情衔接

### 4. 输出设计文档

以结构化格式输出完整方案，便于后续查阅和创作。

## 设计原则

- **真实感**：心理描写要有代入感，读者能感同身受
- **层次感**：表面表现和内心真实想法要有反差
- **节奏感**：情感发展不能太快也不能拖沓
- **差异性**：不同性格的角色有不同的情感表达方式

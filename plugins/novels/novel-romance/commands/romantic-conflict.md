# 设计感情冲突与化解

帮助作者快速设计或完善感情冲突和矛盾化解方案。

## 适用场景

| 场景 | 示例 |
|------|------|
| **冲突设计** | `/novel-romance conflict 设计一个因身份差距引发的感情危机` |
| **误会桥段** | `/novel-romance conflict 设计一个信息差导致的误会` |
| **虐心情节** | `/novel-romance conflict 设计分手到复合的虐心线` |
| **化解方案** | `/novel-romance conflict 为信任危机设计和解方式` |

## 执行流程

### 1. 确认需求方向

通过提问明确：
- **设计类型**：冲突设计/误会桥段/虐心情节/化解方案
- **冲突来源**：内部冲突/外部冲突/自我冲突
- **虐心程度**：轻虐/中虐/重虐
- **已有情节**：与已有剧情的衔接要求

### 2. 生成设计方案

根据需求激活 `Skills(romantic-conflict)` 并生成：

#### 冲突方案
- 冲突的起因和触发点
- 升级路径和关键节点
- 双方的心理变化和反应
- 冲突对感情的影响

#### 化解方案
- 化解的契机和方式
- 和解过程中的情感释放
- 角色通过冲突获得的成长
- 和好后关系的新平衡

### 3. 一致性验证

检查设计的合理性：
- 冲突是否有因有果
- 虐心程度是否适度
- 化解方式是否令人信服
- 是否推动了角色成长

### 4. 输出设计文档

以结构化格式输出完整方案，便于后续查阅和创作。

## 设计原则

- **有因有果**：冲突必须有合理的原因，不能为虐而虐
- **推动成长**：每次冲突都应促进角色成长和关系深化
- **虐有分寸**：掌握好虐心程度，不能把读者虐跑
- **回报匹配**：和好的甜度要配得上虐的程度

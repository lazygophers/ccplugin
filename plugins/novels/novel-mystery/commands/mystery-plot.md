# 设计悬念布局

帮助作者设计悬疑推理故事的核心谜题和悬念结构。

## 适用场景

| 场景 | 示例 |
|------|------|
| **谜题设计** | `/novel-mystery plot 设计一个密室杀人谜题` |
| **悬念结构** | `/novel-mystery plot 规划30章悬念释放节奏` |
| **反转设计** | `/novel-mystery plot 设计一个双重反转结局` |
| **叙事结构** | `/novel-mystery plot 设计双线叙事的交汇点` |

## 执行流程

### 1. 确认需求方向

通过提问明确：
- **设计目标**：单一谜题/完整剧情/反转设计/节奏规划
- **悬疑风格**：本格推理/社会派/心理悬疑/暗黑悬疑
- **故事规模**：短篇单案/中篇连环/长篇多线
- **已有设定**：与已有角色和背景的衔接要求

### 2. 生成设计方案

根据需求激活 `Skills(mystery-plot)` 并生成：

#### 谜题设计
- 核心谜题和表层问题
- 隐藏的深层谜题
- 解谜所需的关键线索
- 诡计的可行性分析

#### 悬念结构
- 各阶段悬念层次
- 信息释放的节奏
- 子悬念与主悬念的关系
- 悬念转化点的设置

#### 反转设计
- 反转的类型和触发点
- 反转前的铺垫要素
- 反转后的逻辑验证
- 情感冲击力评估

### 3. 逻辑验证

检查方案的合理性：
- 谜题是否有唯一解或合理解
- 悬念节奏是否张弛有度
- 反转是否意外但合理
- 是否对读者保持公平

### 4. 输出设计文档

以结构化格式输出完整方案，便于后续创作参考。

## 设计原则

- **公平推理**：所有解谜线索必须在揭晓前呈现给读者
- **逻辑闭环**：真相揭露后所有细节都能自洽
- **情感并重**：悬疑不只是智力游戏，要有情感冲击
- **节奏呼吸**：紧张与舒缓交替，避免疲劳阅读

# 文风调整

## 功能概述

提供文风分析和调整的系统方法，帮助创作者掌握不同文风的写作技巧，实现叙事视角切换、语言风格调节和氛围营造，使文风与故事内容高度匹配。

## 使用场景

- 确定小说的整体文风基调
- 调整特定章节的文风以配合情节
- 切换叙事视角
- 解决文风不统一的问题
- 模仿或学习特定风格

## 核心要点

### 主要文风类型

| 文风 | 特征 | 适用题材 | 代表手法 |
|------|------|---------|---------|
| 轻松幽默 | 节奏明快、语言诙谐 | 都市、日常、轻喜剧 | 吐槽、反差、夸张 |
| 热血燃向 | 激昂有力、情感强烈 | 战斗、竞技、冒险 | 短句、排比、对比 |
| 悬疑紧张 | 克制冷静、信息精确 | 推理、惊悚、悬疑 | 白描、留白、暗示 |
| 深沉厚重 | 文字凝练、意蕴深远 | 历史、权谋、史诗 | 长句、典故、隐喻 |
| 细腻温婉 | 描写细致、情感绵密 | 言情、文艺、成长 | 通感、比喻、心理描写 |
| 凌厉爽快 | 干脆利落、节奏极快 | 爽文、系统流、升级 | 极短句、省略、节奏感 |

### 叙事视角

**第一人称**：
- 优势：代入感强，心理描写自然
- 局限：信息受限于主角视野
- 适用：悬疑、都市、成长类

**第三人称有限视角**：
- 优势：兼顾代入感和叙事灵活性
- 局限：每个场景只能跟随一个角色
- 适用：大多数网文题材（最常用）

**第三人称全知视角**：
- 优势：可自由切换视角，掌控全局
- 局限：容易削弱悬念，需要更强的叙事功力
- 适用：群像、史诗、多线叙事

### 语言风格调节

**句式长短**：
- 短句密集 → 紧张、快节奏、力量感
- 长短交错 → 自然、舒适、有节奏
- 长句为主 → 沉稳、细腻、深思

**用词层次**：
- 口语化 → 亲切、轻松、接地气
- 书面化 → 正式、庄重、有距离感
- 古风化 → 典雅、诗意、有韵味

**修辞密度**：
- 高密度修辞 → 华丽、浓烈、文学感强
- 适度修辞 → 平衡、可读性好
- 白描为主 → 克制、冷峻、客观

### 氛围营造技巧

**通过环境映射情绪**：
```
压抑：乌云低垂、空气潮湿、光线昏暗
紧张：风声、影子、不明声响
温暖：阳光、炊烟、笑声
悲伤：雨、落叶、空旷
危险：血腥味、裂缝、倒计时
```

**通过节奏传递情绪**：
```
恐惧：句子越来越短。段落越来越碎。
        留白增多。
        像心跳加速。

平静：长句缓缓流淌，如同午后的河水绕过石头，
      不紧不慢地向前延伸，带着温暖的日光气息。

愤怒：他转身。掀翻桌子。碗碟碎了一地。
      没有人敢说话。
```

### 文风一致性检查

检查清单：
- 同一角色在不同章节的语言风格是否一致？
- 叙述者的语气和用词是否稳定？
- 情绪性文字与整体基调是否协调？
- 修辞手法的使用频率是否均匀？
- 是否出现与时代背景不符的现代用语？

## 实践建议

1. **文风服务于内容**：先确定故事需要什么感觉，再选择匹配的文风
2. **开篇定调**：前三章确立文风基调，之后保持一致
3. **场景切换时微调**：战斗场景可以更凌厉，日常场景可以更轻松，但不要偏离太远
4. **少即是多**：克制使用华丽修辞，精准的白描往往比堆砌辞藻更有力
5. **朗读测试**：好的文风应该念出来也舒服，拗口的句子需要修改

## 示例

### 同一场景不同文风

**场景**：主角击败强敌

**热血燃向**：
> 剑光炸裂的刹那，天地失色。他一步踏出，剑意纵横三千丈，所过之处，连空气都在颤抖。对手难以置信地低头看向胸口的伤口，而他已经收剑而立，衣袂猎猎。

**悬疑冷峻**：
> 很快就结束了。比所有人预想的都快。没人看清他是怎么出的手。等众人回过神来，只有一个人还站着。地上多了一滩正在蔓延的暗色液体。

**轻松幽默**：
> "就这？"他拍了拍手上并不存在的灰尘，对面那位号称"十年无败绩"的高手正以一个极其不雅的姿势镶嵌在墙壁里。"我还以为至少能撑三招呢。"

**深沉厚重**：
> 这一剑，他等了三年。三年间他无数次在梦中挥出这一剑，每一次都不够完美。而此刻，当剑真正落下时，他心中没有快意，只有一种近乎寂寥的空旷——仿佛追逐了许久的风，终于停了。

# 故事构思

## 功能概述

提供系统化的故事构思框架，帮助创作者从零开始构建故事的核心要素：主题立意、时代背景、世界观设定、核心冲突和故事前提。

## 使用场景

- 从零开始构思一个新故事
- 对已有创意进行深化和拓展
- 选择适合的题材类型（玄幻、都市、科幻、历史、悬疑等）
- 构建独特的世界观体系
- 设计故事的核心矛盾和戏剧冲突

## 核心要点

### 题材选择

网络小说主流题材及其核心吸引力：

| 题材 | 核心吸引力 | 关键要素 |
|------|-----------|---------|
| 玄幻修仙 | 力量成长、等级突破 | 修炼体系、宗门势力、天材地宝 |
| 都市异能 | 现实代入、逆袭爽感 | 金手指设定、社会阶层、情感线 |
| 科幻未来 | 想象力、科技奇观 | 科技树、星际文明、AI伦理 |
| 历史架空 | 权谋博弈、改变历史 | 历史节点、势力格局、军事政治 |
| 悬疑推理 | 智力博弈、真相揭秘 | 线索布局、逻辑链、反转设计 |
| 游戏竞技 | 热血对抗、团队协作 | 游戏系统、竞技规则、成长路线 |

### 故事前提设计

一个好的故事前提需包含三要素：

1. **主角身份**：他是谁？有什么独特之处？
2. **触发事件**：什么打破了他的日常？
3. **核心目标**：他要达成什么？代价是什么？

公式：`[特定身份的主角] + [触发事件] → [追求目标] + [核心阻碍]`

### 世界观构建

构建世界观的五个维度：

1. **力量体系**：世界运行的核心规则（修炼等级、魔法体系、科技层级）
2. **社会结构**：势力分布、阶层划分、权力运作方式
3. **历史脉络**：重大历史事件、文明演变、当前时代特征
4. **地理版图**：空间布局、重要地标、区域特色
5. **文化生态**：信仰体系、风俗习惯、种族关系

### 核心冲突设计

四层冲突模型：

- **内在冲突**：主角内心的矛盾和挣扎（性格缺陷、价值观冲突）
- **人际冲突**：与身边人的矛盾（师徒、兄弟、情感）
- **社会冲突**：与组织/势力的对抗（宗门、帝国、公司）
- **命运冲突**：与更高层面力量的对抗（天道、命运、文明）

## 实践建议

1. **从"如果...会怎样"开始**：用假设性问题激发创意，例如"如果修仙世界进入了工业革命时代？"
2. **反套路思维**：列出该题材的常见套路，然后尝试反转其中一个核心设定
3. **融合创新**：将两个不同题材的核心元素融合，创造新鲜感
4. **真实情感锚点**：再宏大的设定也需要落到真实的人类情感上
5. **留白与扩展性**：世界观不需要一次性全部设计完，预留足够的扩展空间

## 示例

### 故事前提示例

**题材**：都市异能 + 悬疑

**前提**：一个失去记忆的法医（主角身份），在解剖一具尸体时突然获得了读取死者生前记忆的能力（触发事件），他必须利用这个能力追查一个连环杀手（核心目标），但每次使用能力都会丢失自己的一段记忆（核心阻碍）。

**核心冲突**：
- 内在：每查一个案子就失去一段自我，值得吗？
- 人际：搭档发现他的异常，信任产生裂痕
- 社会：一个隐秘组织也在寻找拥有这种能力的人
- 命运：能力的真正来源指向一个改变世界认知的真相

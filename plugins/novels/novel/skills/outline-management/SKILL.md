# 大纲管理

## 功能概述

提供从总纲到细纲的完整大纲设计方法论，帮助创作者规划小说整体结构、管理多条情节线索、合理布局伏笔与回收，确保长篇故事的结构完整性和连贯性。

## 使用场景

- 从零开始设计小说大纲
- 规划卷/篇章的划分和节奏
- 管理多条故事线的交织
- 布局伏笔并跟踪回收
- 调整已有大纲的结构问题
- 管理长篇连载的进度和方向

## 核心要点

### 三级大纲体系

```
总纲（全书视角）
  ├─ 卷纲（分卷/篇章视角）
  │    ├─ 细纲（章节视角）
  │    ├─ 细纲
  │    └─ ...
  ├─ 卷纲
  └─ ...
```

### 总纲设计

总纲是全书的骨架，包含以下要素：

```markdown
## 总纲模板

### 核心信息
- 书名：
- 题材：
- 字数预估：
- 核心卖点（一句话）：

### 故事主线
- 起点：主角的初始状态
- 发展：主要经历的阶段（3-5个大阶段）
- 高潮：最终决战/终极冲突
- 结局：故事如何收束

### 主要角色
- 主角：一句话概括 + 成长弧线
- 女主/重要配角：一句话概括 + 与主角的关系
- 主要反派：一句话概括 + 与主角的冲突

### 势力格局
- 主要势力及其关系

### 力量体系
- 等级划分和核心规则
```

### 卷纲设计

每卷是一个相对完整的故事单元：

```markdown
## 卷纲模板

### 卷名：第X卷「卷名」
- 字数预估：XX万字（约XX章）
- 时间跨度：
- 主要场景：

### 本卷目标
- 情节目标：本卷要完成什么事件？
- 角色目标：主角在本卷的成长是什么？
- 悬念目标：本卷要揭示/制造什么悬念？

### 本卷主线
- 起因：
- 经过（关键节点）：
- 高潮：
- 收束：

### 本卷登场角色
- 新角色及其作用
- 已有角色的发展
```

### 细纲设计

细纲是具体到每章的写作蓝图：

```markdown
## 细纲模板

### 第XXX章「章名」
- 视角角色：
- 场景：
- 本章目的：（推进情节/揭示信息/角色发展）

### 内容要点
1. 开场：场景和情绪起点
2. 发展：本章的核心事件
3. 结尾：章末钩子/悬念

### 关键信息
- 需要传递的设定/伏笔：
- 角色关系变化：
- 连接下一章的线索：
```

### 情节线索管理

长篇小说通常有多条线索并行：

**线索分类**：
- **主线**：贯穿全书的核心故事线（1条）
- **副线**：阶段性的重要支线（2-3条同时进行）
- **暗线**：隐藏的线索，后期揭示（1-2条）

**线索管理表**：

| 线索名 | 类型 | 起始章 | 关键节点 | 预计收束 | 状态 |
|--------|------|--------|---------|---------|------|
| 灭门真相 | 主线 | 第1章 | 50/120/200 | 终章 | 进行中 |
| 师姐下落 | 副线 | 第3章 | 80/150 | 第160章 | 进行中 |
| 神秘组织 | 暗线 | 第30章（暗示） | 100 | 第250章 | 潜伏中 |

### 伏笔管理

**伏笔布局原则**：
1. **自然植入**：伏笔要融入情节，不能突兀
2. **适时回收**：短伏笔 10-30 章内回收，长伏笔在重要节点回收
3. **超出预期**：回收时要给读者"原来如此"的惊喜感
4. **记录追踪**：用表格记录每个伏笔的埋设和回收计划

**伏笔追踪表**：

| 伏笔内容 | 埋设位置 | 回收位置 | 回收方式 | 状态 |
|---------|---------|---------|---------|------|
| 主角手臂上的神秘印记 | 第5章 | 第80章 | 印记是远古传承的钥匙 | 已回收 |
| 老者临死前的半句话 | 第23章 | 第150章 | 指向最终Boss的身份 | 待回收 |
| 地图上消失的城市 | 第45章 | 待定 | 关联暗线 | 待回收 |

### 节奏规划

全书节奏的宏观布局（以200万字为例）：

```
开篇期（1-20万字）：
  建立世界观 → 触发事件 → 首个小高潮
  节奏：中等偏快，快速抓住读者

发展期（20-80万字）：
  2-3个中型副本/事件，实力稳步提升
  节奏：紧松交替，每30万字一个大高潮

转折期（80-120万字）：
  重大转折、阵营洗牌、真相初现
  节奏：先压抑后爆发

高潮期（120-180万字）：
  终极冲突展开，全面决战
  节奏：持续紧张，高潮迭起

收束期（180-200万字）：
  最终决战 → 悬念收束 → 结局
  节奏：爆发后渐缓，给予情感落点
```

## 实践建议

1. **大纲是活的**：大纲不是死板的剧本，写作过程中可以根据实际情况调整
2. **先粗后细**：先写总纲确定方向，再逐步细化到卷纲和细纲
3. **定期回顾**：每写完一卷回顾大纲，检查是否偏离主线、伏笔是否遗漏
4. **预留弹性**：大纲中留 20% 的弹性空间，允许灵感驱动的调整
5. **工具辅助**：善用表格管理线索和伏笔，避免长篇写作中的遗忘

## 示例

### 总纲示例片段

```markdown
## 《暗夜法医》总纲

### 核心信息
- 题材：都市异能悬疑
- 字数预估：150万字（约5卷）
- 核心卖点：每次使用异能查案就会丢失一段自己的记忆

### 故事主线
- 起点：失忆法医获得读取死者记忆的能力
- 第一阶段：利用能力破获连环案件，逐渐失去记忆
- 第二阶段：发现能力来源与神秘组织的关联
- 第三阶段：记忆几乎耗尽，面临最终抉择
- 结局：以最后一段记忆为代价揭开终极真相

### 分卷规划
- 第一卷「失忆者」（30万字）：获得能力，首个连环案
- 第二卷「记忆猎人」（30万字）：组织初现，能力代价加剧
- 第三卷「镜中人」（35万字）：身世真相，阵营冲突
- 第四卷「最后记忆」（30万字）：终极对决
- 第五卷「新的开始」（25万字）：真相大白，结局收束
```

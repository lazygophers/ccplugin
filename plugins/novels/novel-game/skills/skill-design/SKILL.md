# 技能设定设计

## 功能概述

提供完整的游戏技能与装备体系设计框架，涵盖技能树结构、技能连招、装备品阶、词缀系统、天赋路线等核心战斗要素，确保战斗系统有深度且手感出色。

## 使用场景

- 设计职业专属技能树
- 规划技能组合与连招系统
- 构建装备品阶与词缀体系
- 设计天赋系统与加点路线
- 平衡技能间的克制关系

## 核心要点

### 技能体系架构

```
【技能分类】
- 主动技能：需要手动释放，有冷却和消耗
- 被动技能：永久生效，提供属性加成或特殊效果
- 终极技能：强力技能，长CD或需要特殊条件触发
- 觉醒技能：转职/觉醒后解锁的专属技能

【技能属性】
- 伤害类型：物理/魔法/真实伤害
- 作用范围：单体/范围/全屏
- 施法方式：瞬发/读条/引导/蓄力
- 附加效果：控制/灼烧/减速/沉默/击飞
```

### 技能树设计

```
【树状结构】
  基础技能（Lv1解锁）
    ├── 分支A：爆发路线
    │   ├── 进阶A1：暴击强化
    │   └── 进阶A2：连击系统
    ├── 分支B：控制路线
    │   ├── 进阶B1：群体控制
    │   └── 进阶B2：持续压制
    └── 分支C：辅助路线
        ├── 进阶C1：增益强化
        └── 进阶C2：减益克制

【技能点规则】
- 每级获得固定技能点
- 高阶技能需要前置技能达到一定等级
- 总技能点有限，迫使玩家做选择
- 洗点机制：允许重置技能分配
```

### 技能连招系统

```
【连招设计】
  起手技（接近/控制） → 核心输出（主力伤害技能）
  → 衔接技（位移/追击） → 终结技（高额爆发）
  → 脱离技（保命/拉开距离）

【连招深度】
- 基础连招：2-3个技能，新手可用
- 进阶连招：4-5个技能，需要操作意识
- 极限连招：6+技能，高手专属，需要精确时机

【取消机制】
- 普攻取消：普攻动作后接技能取消后摇
- 技能取消：用一个技能取消另一个技能的后摇
- 闪避取消：用翻滚/闪现取消技能硬直
```

### 装备体系设计

```
【品质等级】
  白色（普通） → 绿色（优秀） → 蓝色（精良）
  → 紫色（史诗） → 橙色（传说） → 红色（神话）

【装备部位】
  武器 | 头盔 | 铠甲 | 护腿 | 鞋子
  项链 | 戒指×2 | 披风 | 腰带 | 护腕

【词缀系统】
- 前缀：攻击类属性（如"炽热的"→火焰伤害加成）
- 后缀：防御类属性（如"坚韧之"→韧性提升）
- 套装效果：2件/4件/6件套装激活特殊效果
- 镶嵌系统：宝石槽位，提供额外属性

【装备获取】
- 副本掉落：BOSS专属装备
- 制作锻造：采集材料+配方
- 交易购买：玩家交易或NPC商店
- 任务奖励：主线/支线/成就奖励
```

### 天赋系统设计

```
【天赋树结构】
  第一层（基础强化）：小幅度数值提升
  第二层（机制调整）：改变技能的运作方式
  第三层（核心天赋）：定义玩法方向的关键节点
  第四层（终极天赋）：强力被动，三选一

【天赋设计原则】
- 每层天赋都提供有意义的选择
- 避免"必点"天赋，确保多条路线可行
- 天赋之间产生化学反应，鼓励探索搭配
- 与技能树配合，形成完整的Build
```

## 设计建议

1. **手感优先**：技能释放要有反馈感，打击感是战斗系统的灵魂
2. **深度与门槛**：易上手难精通，基础连招简单，极限操作有天花板
3. **职业区分**：每个职业的技能逻辑要有本质区别，不是换个数值换个皮
4. **装备驱动**：装备对Build的影响要大，让刷装备有动力
5. **版本轮换**：通过技能调整和新装备引导Meta变化，保持新鲜感

## 示例

### 创新技能示例：符文编织系统

**核心理念**：玩家通过组合不同符文创造独一无二的技能。

**基础符文**：火焰、冰霜、雷电、大地、风暴、暗影、神圣
**形态符文**：投射、范围、护盾、召唤、强化、链接

**组合示例**：
- 火焰 + 投射 = 火球术
- 火焰 + 范围 = 烈焰风暴
- 冰霜 + 护盾 = 冰甲术
- 雷电 + 链接 = 闪电链
- 暗影 + 召唤 = 暗影仆从
- 火焰 + 冰霜 + 范围 = 蒸汽爆破（高阶组合）

**特色**：同样的符文由不同职业使用，效果会因职业特性而有差异。

# 竞技设计

## 功能概述

提供完整的游戏竞技体系设计框架，涵盖赛事规则、排位系统、战术策略、团队配合、赛事组织等核心要素，确保竞技场景紧张刺激且具有观赏性。

## 使用场景

- 设计电竞赛事体系和比赛规则
- 规划排位段位与匹配系统
- 构建战术策略与团队配合体系
- 设计PVP对战模式和机制
- 策划公会战/攻城战等大规模PVP

## 核心要点

### 赛事体系架构

```
【职业赛事层级】
  青训赛 → 次级联赛 → 顶级联赛 → 国际大赛 → 世界总决赛

【赛制类型】
- BO1（单局决胜）：积分赛、小组赛常用
- BO3（三局两胜）：淘汰赛常用
- BO5（五局三胜）：半决赛、决赛
- BO7（七局四胜）：总决赛（极少数）
- 循环赛：联赛常规赛
- 双败淘汰：给强队更多容错

【赛季结构】
  春季赛 → 季中赛（国际） → 夏季赛 → 全球总决赛
  常规赛（积分） → 季后赛（淘汰） → 决赛
```

### 排位段位系统

```
【段位划分】
  青铜 → 白银 → 黄金 → 铂金 → 钻石
  → 大师 → 宗师 → 王者 → 荣耀王者

【匹配机制】
- MMR（隐藏分）：决定实际匹配对手
- 段位分：展示用，胜加败减
- 保护机制：晋级赛、掉段保护
- 段位衰减：长期不打排位掉段

【赛季重置】
- 硬重置：所有人回到起点（慎用）
- 软重置：根据上赛季段位压缩
- 定位赛：赛季初通过若干场确定起始段位
```

### 战术策略体系

```
【MOBA类战术】
- 运营流：稳扎稳打，发育为主，后期决战
- 激进流：前期强势，频繁Gank，滚雪球
- 分推流：带线牵制，避免正面团战
- 保护流：以保护核心C位为核心的阵容
- 黑科技：非常规英雄/位置选择，出其不意

【FPS类战术】
- 默认站位：标准的地图控制分配
- Rush战术：集中力量快速突破一个点
- Fake战术：佯攻一侧，实际主攻另一侧
- 经济对枪：根据经济状况选择装备策略
- 残局处理：人数劣势时的拖时间/偷点战术

【团队配合】
- 指挥体系：主指挥 + 副指挥 + 个人决策
- 视野控制：信息战，掌握敌方动向
- 资源分配：经济和buff的优先级
- 时间节奏：关键时间节点的争夺
```

### PVP对战模式

```
【竞技场模式】
- 1v1 单挑：纯个人实力对决
- 3v3 小队赛：小规模团队配合
- 5v5 团队赛：完整的团队竞技
- 大乱斗：多人混战，适合娱乐

【大规模PVP】
- 公会战：定期的公会对抗
- 攻城战：攻守双方的据点争夺
- 国战：阵营间的大规模战争
- 世界BOSS：PVE和PVP的混合玩法

【特殊竞技】
- 竞技场排名赛：限时争夺排名
- 锦标赛：海选+淘汰的赛事
- 赌斗：押注型PVP
- 跨服对决：不同服务器间的对抗
```

### 比赛节奏与高潮设计

```
【比赛节奏控制】
  开局（试探期）→ 中期（拉锯期）→ 后期（决战期）

【制造高潮的手法】
- 绝境翻盘：劣势局通过关键操作逆转
- 极限操作：超出常人预期的微操表现
- 关键抉择：生死存亡时刻的战术决策
- 个人秀：单人carry全场的英雄时刻
- 默契配合：团队完美执行战术的名场面

【解说视角】
- 数据分析：经济差、装备差、等级差
- 战术解读：阵容分析、战术意图
- 操作亮点：关键技能、走位、意识
- 情绪渲染：现场气氛、选手表情、观众反应
```

## 设计建议

1. **竞技公平**：竞技场景要尽量排除Pay-to-Win因素，纯靠技术和策略
2. **观赏性**：比赛场面要有可写性，让读者也能感受到紧张和兴奋
3. **成长弧线**：主角的竞技水平要有合理的成长过程，不能一步登天
4. **对手塑造**：强力对手是竞技小说的灵魂，每个阶段都要有旗鼓相当的对手
5. **战术深度**：不能只靠数值碾压，要体现策略思考和临场应变

## 示例

### 电竞赛事示例：荣耀职业联赛（GPL）

**赛季结构**：
- 春季赛（1-5月）：12支队伍循环赛 → 前8名进季后赛
- 季中邀请赛（6月）：各赛区冠军参加国际赛
- 夏季赛（7-11月）：积分制 → 季后赛
- 全球总决赛（12月）：各赛区代表队争夺世界冠军

**选手位置**：
- 上路（Top）：单线对抗，后期切入
- 打野（Jungle）：节奏控制，Gank支援
- 中路（Mid）：核心输出/控制
- 下路AD（ADC）：团战持续输出
- 辅助（Support）：视野控制，保护队友

**特色**：选手有"天赋觉醒"系统，在极限状态下触发超常发挥，如时间感知放慢、预判能力提升等。

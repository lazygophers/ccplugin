# 设计技能设定

帮助作者设计游戏中的技能体系、装备系统和战斗机制。

## 适用场景

| 场景 | 示例 |
|------|------|
| **技能树** | `/novel-game skill 设计剑士职业技能树` |
| **装备系统** | `/novel-game skill 装备品阶和词缀系统` |
| **连招体系** | `/novel-game skill 刺客职业连招设计` |
| **天赋系统** | `/novel-game skill 三条天赋路线设计` |

## 执行流程

### 1. 确认设计方向

通过提问明确：
- **设计目标**：技能树/装备体系/连招系统/天赋路线/完整战斗系统
- **职业定位**：输出/坦克/辅助/混合
- **风格特色**：硬核操作/策略搭配/数值养成
- **已有设定**：与已有职业和数值体系的匹配

### 2. 激活技能生成方案

激活 `Skills(skill-design)` 并根据方向生成：

#### 技能树设计
- 技能分支和前置关系
- 每个技能的效果和机制
- 技能点分配策略
- 核心Build推荐

#### 装备系统
- 品质等级和颜色标识
- 词缀种类和效果
- 套装效果设计
- 镶嵌和强化系统

#### 连招体系
- 基础连招路线
- 进阶组合技巧
- 不同场景的连招选择
- 取消机制和操作技巧

#### 天赋路线
- 天赋树层级结构
- 关键天赋节点的效果
- 不同路线的Build定位
- 天赋与技能的联动

### 3. 平衡性验证

- 技能间是否存在必选项（应避免）
- 装备对战斗力的影响是否合理
- 不同Build是否都有可行性
- PVP和PVE的技能效果是否需要分离

## 设计原则

- **深度选择**：技能和装备搭配要有足够的策略深度
- **操作空间**：留给高手展现操作的上限
- **特色鲜明**：每个职业/Build的战斗风格要有明显区别
- **装备驱动**：装备获取是推动角色成长和剧情的重要动力

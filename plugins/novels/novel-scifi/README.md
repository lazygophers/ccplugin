# Novel Scifi Plugin

> 科幻未来小说创作插件 - 提供科技设定、星际文明、太空探索、机甲系统等科幻特色功能

## 简介

Novel Scifi 是一个专为科幻未来类网络小说设计的 Claude Code 插件，提供从科技体系到星际文明的全方位科幻世界构建支持。帮助作者快速建立逻辑自洽、层次丰富的科幻宇宙。

## 核心功能

### 科技设定 (Tech Setting)
- 能源科技体系设计
- 武器装备系统分类
- 通讯与信息网络
- AI与计算系统

### 星际文明 (Civilization)
- 文明等级划分（卡尔达肖夫指数）
- 外星种族设计
- 星际政治格局
- 文明冲突与合作

### 机甲系统 (Mecha System)
- 机甲分类与等级
- 驾驶员适配系统
- 武装配置体系
- 机甲进化路线

## 快捷命令

```bash
# 设计科技体系
/novel-scifi tech

# 创建星际文明
/novel-scifi civilization

# 设计机甲系统
/novel-scifi mecha
```

## 专业 Agents

```bash
# 科技设定顾问
@tech-agent 设计一套近未来能源科技体系

# 科幻世界观设计师
@worldbuilding-agent 构建一个多文明并存的银河系设定
```

## 技能体系

| 技能 | 说明 | 适用场景 |
|------|------|---------|
| tech-setting | 科技设定设计 | 能源系统、武器装备、AI架构 |
| civilization | 星际文明体系 | 文明等级、种族设计、星际政治 |
| mecha-system | 机甲系统设计 | 机甲分类、战斗系统、驾驶员适配 |

## 使用场景

- 新作品的科幻世界观构建
- 科技体系的设计与完善
- 星际文明的等级和种族设定
- 机甲战斗系统的架构设计
- 太空探索和星际航行设定
- 人工智能与虚拟世界设定

## 创作流程建议

1. **科技设定**：使用 tech-setting 确定世界的科技水平和核心科技
2. **文明构建**：使用 civilization 规划星际文明格局和种族关系
3. **军事体系**：使用 mecha-system 建立机甲/战舰等军事装备体系
4. **关联验证**：确保科技水平、文明等级、军事实力三者逻辑自洽

## 与 novel-writing 插件的关系

本插件是 novel-writing 的科幻未来类型扩展，专注于该类型的特色设定。建议配合 novel-writing 插件使用，获得完整的创作支持：
- novel-writing：通用创作功能（角色、情节、对话、文风）
- novel-scifi：科幻特色设定（科技、文明、机甲）

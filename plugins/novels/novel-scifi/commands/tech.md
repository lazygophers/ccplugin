# 设计科技设定

帮助作者快速设计或完善科幻世界的科技体系，打造硬核或软科幻的技术基础。

## 适用场景

| 场景 | 示例 |
|------|------|
| **设计能源体系** | `/novel-scifi tech 设计反物质能源系统` |
| **武器系统** | `/novel-scifi tech 粒子束武器和护盾科技` |
| **通讯技术** | `/novel-scifi tech 超光速通讯设定` |
| **AI系统** | `/novel-scifi tech 人工智能分级体系` |
| **生命科技** | `/novel-scifi tech 基因改造和生命延续` |

## 执行流程

### 1. 确认需求方向

通过提问明确：
- **科技类型**：能源/武器/通讯/AI/生命科技/交通/计算机/完整体系
- **世界风格**：硬科幻/软科幻/赛博朋克/太空歌剧/近未来
- **科技水平**：当代延伸/近未来（50-100年）/远未来（数百年）/超远未来（星际时代）
- **已有设定**：与已有世界观的衔接要求

### 2. 生成科技方案

根据需求激活 `Skills(tech-system)` 并生成：

#### 能源科技
- **基础能源**：核聚变、反物质、真空零点能、暗物质能源
- **能源等级**：从化学能到宇宙级能源的进化路径
- **储存与传输**：能源电池、能量晶体、无线能源传输
- **能量转换**：能源盾、能量武器的转换效率
- **资源分布**：稀有能源矿藏和星际争夺点

#### 武器科技
- **动能武器**：电磁炮、轨道炮、质量投射器
- **能量武器**：激光、粒子束、等离子炮、反物质弹
- **防护系统**：能量护盾、相位偏转、力场发生器
- **星舰武器**：主炮系统、点防御、导弹阵列、歼星武器
- **平衡设定**：攻防博弈、克制关系、科技竞赛

#### 通讯科技
- **常规通讯**：电磁波、激光通讯、量子加密
- **FTL通讯**：超空间信号、量子纠缠通讯、引力波传输
- **信息网络**：星际网络、意识上传、脑机接口
- **限制设定**：延迟、距离限制、干扰手段
- **情报战**：破译、屏蔽、信息作战

#### AI科技
- **AI分级**：弱AI、强AI、超级AI、奇点AI
- **功能分类**：战术AI、科研AI、管理AI、虚拟人格
- **伦理困境**：AI权利、机器叛乱、三大法则
- **人机关系**：增强人类、人机融合、意识转移
- **AI战争**：智能武器、电子战、网络攻防

#### 生命科技
- **基因工程**：基因优化、特殊能力、物种改造
- **生命延续**：寿命延长、器官再生、冷冻复苏
- **身体强化**：义体改造、纳米机器人、神经增强
- **进化路线**：生物进化、机械飞升、虚拟永生
- **伦理边界**：人性定义、改造禁区、身份认同

### 3. 科学性验证

检查科技体系的合理性：
- **物理规律**：是否符合基本物理原理（或给出合理的假设）
- **技术递进**：科技发展路径是否合理
- **能量守恒**：能源消耗与输出是否平衡
- **社会影响**：科技对社会结构的影响
- **故事空间**：是否为情节发展留足科技悬念

### 4. 输出设定文档

以结构化格式输出完整设定，包括：
- 科技原理说明（可繁可简，根据硬度需求）
- 技术参数和性能指标
- 使用限制和代价
- 技术发展历史和未来趋势
- 与其他科技的关联关系

## 设计原则

- **科学基础**：硬科幻要有理论依据，软科幻要有内在逻辑
- **代价设定**：强大的科技必须有对应的代价或限制
- **技术树**：科技之间要有依赖关系，形成完整的发展脉络
- **服务叙事**：科技为故事服务，不是炫技的工具
- **留白艺术**：不需要解释所有细节，适当留白增加神秘感
- **冲突驱动**：科技差距、资源争夺、伦理困境都是故事张力的来源

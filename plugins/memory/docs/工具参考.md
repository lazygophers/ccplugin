## memory_store - 存储记忆

将信息存储到知识图谱（v0.2.0+ 将实现持久化）。

### 参数

| 参数名 | 类型 | 必需 | 默认值 | 说明 |
|--------|------|------|--------|------|
| `content` | string | ✅ | - | 记忆内容 |
| `tags` | string[] | ❌ | `[]` | 标签列表 |
| `metadata` | object | ❌ | `{}` | 元数据字典 |

### 返回值

**类型**: `CallToolResult`

**成功响应**:
```
已存储记忆: {content前50字符} (标签: {tags列表})
```

### 使用示例

#### 最小参数

```python
memory_store(content="这是一条简单的记忆")
```

#### 完整参数

```python
memory_store(
    content="项目决策：采用 MCP 协议实现插件通信",
    tags=["项目", "架构", "决策"],
    metadata={
        "author": "luoxin",
        "date": "2025-01-15",
        "project": "ccplugin",
        "importance": "high"
    }
)
```

#### 实际场景示例

**场景 1：记录技术决策**
```python
memory_store(
    content="选择 NetworkX 作为知识图谱存储方案，原因：1) Python 原生支持 2) 丰富的图算法 3) 易于扩展",
    tags=["技术选型", "知识图谱"],
    metadata={"decision_date": "2025-01-15", "alternatives": ["Neo4j", "ArangoDB"]}
)
```

**场景 2：保存学习笔记**
```python
memory_store(
    content="Python asyncio 最佳实践：避免在 async 函数中使用阻塞 I/O",
    tags=["Python", "异步编程", "最佳实践"],
    metadata={"source": "官方文档", "category": "编程"}
)
```

### 当前版本行为

**v0.1.0**:
- ✅ 接受并验证所有参数
- ✅ 返回确认消息
- ⚠️ **不实际持久化数据**（仅返回确认）

**v0.2.0 计划**:
- 使用 NetworkX 存储到知识图谱
- 支持记忆节点和关系
- 实现标签索引

## memory_search - 搜索记忆

基于关键词和标签搜索记忆（v0.2.0+ 将实现图遍历搜索）。

### 参数

| 参数名 | 类型 | 必需 | 默认值 | 说明 |
|--------|------|------|--------|------|
| `query` | string | ✅ | - | 搜索关键词 |
| `tags` | string[] | ❌ | `[]` | 标签过滤 |
| `limit` | integer | ❌ | `10` | 返回数量限制 |

### 返回值

**类型**: `CallToolResult`

**成功响应**:
```
搜索记忆: '{query}' (标签: {tags列表}, 限制: {limit})

注意: v0.1.0 为框架实现，实际搜索功能将在 v0.2.0 中实现。
```

### 使用示例

#### 最小参数

```python
memory_search(query="Python")
```

#### 完整参数

```python
memory_search(
    query="架构决策",
    tags=["项目", "架构"],
    limit=5
)
```

#### 实际场景示例

**场景 1：查找技术决策**
```python
memory_search(
    query="微服务",
    tags=["架构", "决策"],
    limit=10
)
```

**场景 2：检索学习笔记**
```python
memory_search(
    query="asyncio 最佳实践",
    tags=["Python", "最佳实践"],
    limit=5
)
```

**场景 3：模糊搜索**
```python
memory_search(
    query="知识图谱",
    limit=20
)
```

### 当前版本行为

**v0.1.0**:
- ✅ 接受并验证所有参数
- ✅ 返回提示消息
- ⚠️ **不实际执行搜索**（仅返回提示）

**v0.2.0 计划**:
- 实现基于 NetworkX 的图遍历搜索
- 支持标签组合查询
- 实现相关性排序
- 支持模糊匹配

## 错误处理

所有工具都遵循统一的错误处理规范：

### 参数验证错误

**场景**：缺少必需参数

```python
memory_store()  # 缺少 content
```

**错误响应**:
```json
{
  "isError": true,
  "content": [
    {
      "type": "text",
      "text": "错误: 'content' is required"
    }
  ]
}
```

### 类型错误

**场景**：参数类型不匹配

```python
memory_search(query="test", limit="invalid")  # limit 应为整数
```

**错误响应**:
```json
{
  "isError": true,
  "content": [
    {
      "type": "text",
      "text": "错误: Input should be a valid integer"
    }
  ]
}
```

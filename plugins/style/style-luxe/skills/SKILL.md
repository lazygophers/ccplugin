Luxe - 奢华高端设计风格规范

奢华风格设计采用黄金元素、大理石纹理和精致排版，传达高端、优雅和premium 品牌形象。
## 核心特征

**色彩系统**：
```css
/* 奢华调色板 */
--gold: #d4af37;         /* 黄金 */
--dark-navy: #1a1a2e;    /* 深海军蓝 */
--cream: #f5f1de;        /* 奶油色 */
--marble-gray: #a8a8a8;  /* 大理石灰 */
--black: #0a0a0a;        /* 漆黑 */
--rose-gold: #b76e79;    /* 玫瑰金 */
```

**排版**：
```css
--font-display: Didot, Georgia, serif;  /* 优雅衬线 */
--font-body: Lora, serif;               /* 平衡衬线 */

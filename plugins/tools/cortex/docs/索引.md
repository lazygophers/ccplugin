## 实际计数 (与文档保持一致)

**范围标记**: 全局 (Claude Code 会话级 / 系统级) · 当前目录 (PWD) · 知识库 (vault) · 记忆层 (`记忆/L0-L4`)

| 项 | 数量 | 范围 | 文件位置 |
|----|------|------|---------|
| Agent | 8 | 知识库 (部分 + 全局 外网) | `agents/*.md` |
| Skill | 21 | 知识库 / 记忆层 / 当前目录 (按 skill 而异, 详见 Skills 详解) | `skills/<name>/SKILL.md` |
| Slash command | 20 | 知识库 主, 全局+当前目录 部分 | `commands/<name>.md` |
| Bash wrapper | 21 | 与对应 slash / CLI 一致 | `~/.cortex/scripts/<name>.sh` |
| Lint 规则 | 17 | 知识库 (仅作用于 vault) | `scripts/lint/rules.json` |
| Hook | 5 | 全局 (Claude Code 会话级触发) | `.claude-plugin/plugin.json` (SessionStart / PostCompact / Stop / SubagentStop / UserPromptSubmit) |
| Python CLI | 9 | 知识库 主, 记忆层 + 全局 部分 | `scripts/cli/*.py` (save/search/deep_search/ingest_url/ingest_file/memory/ledger/session/html_render) |

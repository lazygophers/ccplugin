# Novel Mystery Plugin

> 悬疑推理小说创作插件 - 提供悬念布局、角色秘密、线索系统等悬疑推理类网络小说的特色创作功能

## 简介

Novel Mystery 是一个专为悬疑推理类网络小说设计的 Claude Code 插件，提供从谜题设计到线索管理的全方位悬疑创作支持。帮助作者构建逻辑严密、悬念迭起、公平推理的悬疑故事。

## 核心功能

### 悬念布局 (Mystery Plot)
- 谜题设计与悬念制造
- 反转与揭露的节奏控制
- 环形结构与线性叙述
- 三幕式悬念结构

### 角色秘密 (Character Secrets)
- 角色隐藏动机分析
- 秘密揭露的时机
- 嫌疑人群像设计
- 角色转变与真相反转

### 线索系统 (Clue System)
- 关键线索与红鲱鱼设计
- 伏笔布置与回收
- 信息密度与发现节奏
- 推理逻辑验证

## 快捷命令

```bash
# 设计悬念布局
/novel-mystery plot

# 设计角色秘密
/novel-mystery secrets

# 管理线索系统
/novel-mystery clue
```

## 专业 Agents

```bash
# 悬疑架构师
@mystery-agent 设计一个密室杀人的悬疑框架

# 线索设计师
@clue-agent 为连环案设计线索系统
```

## 技能体系

| 技能 | 说明 | 适用场景 |
|------|------|---------|
| mystery-plot | 悬念布局 | 谜题设计、悬念结构、反转设计 |
| character-secrets | 角色秘密 | 动机分析、嫌疑人画像、秘密管理 |
| clue-system | 线索系统 | 线索设计、红鲱鱼、伏笔管理 |

## 使用场景

- 悬疑推理故事的整体框架设计
- 核心谜题和诡计的构思
- 嫌疑人群像和秘密网络的构建
- 线索系统和伏笔网络的管理
- 推理逻辑的完整性验证

## 创作流程建议

1. **框架设计**：使用 mystery-plot 设计核心谜题和悬念结构
2. **角色构建**：使用 character-secrets 设计嫌疑人群像和秘密
3. **线索布局**：使用 clue-system 规划线索和伏笔系统
4. **逻辑验证**：确保谜题、角色、线索三者逻辑自洽

## 与 novel-writing 插件的关系

本插件是 novel-writing 的悬疑推理类型扩展，专注于该类型的特色设定。建议配合 novel-writing 插件使用，获得完整的创作支持：
- novel-writing：通用创作功能（角色、情节、对话、文风）
- novel-mystery：悬疑特色设定（悬念、秘密、线索）

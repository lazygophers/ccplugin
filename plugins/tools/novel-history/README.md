# Novel History Plugin

> 历史架空小说创作插件 - 提供朝代背景、权谋设计、战争策略、官场体系等历史特色功能

## 简介

Novel History 是一个专为历史架空类网络小说设计的 Claude Code 插件，提供从朝代背景到权谋战争的全方位历史世界构建支持。帮助作者快速建立有历史质感、逻辑自洽的架空世界观。

## 核心功能

### 朝代背景设计 (Dynasty Setting)
- 架空朝代创建与原型融合
- 政治制度与官僚体系
- 社会阶层与经济体系
- 文化风俗与礼仪制度

### 权谋设计 (Political Intrigue)
- 朝堂派系构建与博弈
- 宫廷斗争与后宫权术
- 阴谋布局与连环计策
- 情报暗战与间谍网络

### 战争策略 (Warfare Strategy)
- 兵法战术与战役规划
- 军队编制与指挥体系
- 攻城守城与要塞攻防
- 后勤补给与战争经济

## 快捷命令

```bash
# 设计朝代背景
/novel-history dynasty

# 设计权谋情节
/novel-history intrigue

# 设计战争策略
/novel-history warfare
```

## 专业 Agents

```bash
# 历史背景顾问
@history-agent 设计一个融合唐宋特征的架空朝代

# 策略设计师
@strategy-agent 设计一场三方博弈的朝堂权谋
```

## 技能体系

| 技能 | 说明 | 适用场景 |
|------|------|---------|
| dynasty-setting | 朝代背景设计 | 架空朝代、社会制度、经济文化 |
| political-intrigue | 权谋设计 | 派系博弈、阴谋布局、宫廷斗争 |
| warfare-strategy | 战争策略 | 兵法战术、战役规划、军队编制 |

## 使用场景

- 新作品的历史架空世界观构建
- 朝代背景和政治制度设计
- 权谋斗争情节的策划和验证
- 战争场面和军事策略设计
- 官场升迁和科举制度设计
- 文化风俗和日常生活细节

## 创作流程建议

1. **朝代构建**：使用 dynasty-setting 设计架空朝代框架
2. **格局设定**：使用 political-intrigue 构建派系势力格局
3. **冲突设计**：使用 warfare-strategy 规划军事冲突
4. **关联验证**：确保朝代背景、权力格局、军事实力三者逻辑自洽

## 与 novel-writing 插件的关系

本插件是 novel-writing 的历史架空类型扩展，专注于该类型的特色设定。建议配合 novel-writing 插件使用，获得完整的创作支持：
- novel-writing：通用创作功能（角色、情节、对话、文风）
- novel-history：历史特色设定（朝代、权谋、战争）

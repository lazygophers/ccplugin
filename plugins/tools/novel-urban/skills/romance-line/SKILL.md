# 情感线设计

## 功能概述

提供都市言情小说的情感线设计方法，包括经典相遇模式、恋爱发展阶段、情感冲突设计、甜虐节奏控制和CP互动创作，帮助创作者打造让读者欲罢不能的感情线。

## 使用场景

- 设计男女主角的初始关系和相遇方式
- 规划恋爱发展的节奏和阶段
- 设计情感冲突和矛盾
- 创作甜蜜互动和虐心桥段
- 安排配角感情线

## 核心要点

### 经典相遇模式

| 模式 | 特点 | 适合类型 |
|------|------|---------|
| **职场相遇** | 上下级/同事/客户关系 | 职场文、总裁文 |
| **误会开局** | 第一印象极差，后续反转 | 欢喜冤家、甜宠文 |
| **青梅竹马重逢** | 多年后再遇，物是人非 | 暗恋成真、治愈文 |
| **契约关系** | 假结婚/假恋人/商业联姻 | 先婚后爱、豪门文 |
| **英雄救美/美救英雄** | 危机中结识，留下深刻印象 | 甜宠文、悬疑言情 |
| **网恋奔现** | 线上默契，线下意外 | 轻松甜文、现代感强 |
| **命运安排** | 反复偶遇，冥冥之中 | 浪漫主义、命定感 |

### 恋爱发展五阶段

```
第一阶段：初遇（1-3万字）
  特征：第一印象建立、好奇或排斥
  关键：制造强烈的初始情绪（心动/厌恶/好奇）
  ↓
第二阶段：试探（3-8万字）
  特征：频繁接触、了解加深、暧昧萌芽
  关键：制造独处机会、展现反差魅力、微妙的心理变化
  ↓
第三阶段：确认（8-15万字）
  特征：情感明确、表白/接受、甜蜜高峰
  关键：告白场景要有仪式感、甜蜜期给足读者糖
  ↓
第四阶段：考验（15-25万字）
  特征：外部阻力、内心挣扎、误会与分离
  关键：虐有度、冲突要合理、分离不能太久
  ↓
第五阶段：圆满（25-30万字）
  特征：克服障碍、彼此成长、HE结局
  关键：大高潮要震撼、结局要满足读者期待
```

### 情感冲突设计

**内因型冲突**（角色自身）：

1. **性格碰撞**：强势对强势、慢热对急性子
2. **信任障碍**：过去的创伤导致不敢相信爱情
3. **自卑退缩**：认为自己不配、主动推开对方
4. **事业与爱情**：关键时刻必须二选一

**外因型冲突**（外部压力）：

1. **身份差距**：豪门与平民、总裁与员工
2. **家族反对**：门不当户不对、联姻压力
3. **第三者介入**：前任回归、青梅竹马、商业联姻对象
4. **利益冲突**：两个家族/公司是竞争对手
5. **秘密曝光**：隐藏的身份/过去被揭露

**冲突升级原则**：每次冲突的赌注要比上一次更大，解决也要更有力。

### 甜虐节奏控制

**糖虐比例参考**：

| 类型 | 糖:虐 | 特点 |
|------|--------|------|
| **纯甜文** | 8:2 | 轻微误会即解开，全程高糖 |
| **甜宠文** | 7:3 | 偶有波折但很快和好 |
| **正剧言情** | 5:5 | 甜虐交替，情感饱满 |
| **虐恋情深** | 3:7 | 大量心理折磨，结局甜蜜 |
| **先虐后甜** | 前期2:8 后期8:2 | 前半虐到心碎，后半甜到齁 |

**节奏控制公式**：

```
给糖 → 给糖 → 小虐 → 大糖（奖励读者的耐心）
给糖 → 大虐 → 缓冲 → 超大糖（虐后必须补偿）
```

**铁律**：连续虐心不超过3章，每次虐心后必须有等量或更多的甜蜜补偿。

### CP互动设计

**日常甜蜜桥段**：
- 不经意的肢体接触（系安全带、擦嘴角、挡车门）
- 专属称呼和只有两人懂的默契
- 食物分享（"这个很好吃，你尝尝"）
- 不自觉的关注（人群中第一眼找对方）
- 生病照顾、加班送饭、雨天接人

**经典吃醋桥段**：
- 看到异性靠近时的微表情变化
- 嘴上说不在意但行动暴露真心
- 醋意引发的占有欲表现（搂腰、牵手宣示主权）
- 冷战后的主动示好

**高光告白场景**：
- 雨中告白：淋着雨追出去说真心话
- 危机告白：以为要失去时的脱口而出
- 误会澄清后的深情表白
- 大庭广众下的霸气宣言
- 安静夜晚的温柔诉说

### 配角感情线

**功能定位**：
- 映衬主角CP的感情（对比/互补）
- 推动主线情节发展
- 提供情感调剂和轻松感
- 展现不同类型的爱情观

**常见配角CP模式**：闺蜜×男主朋友、反派×虐恋、欢喜冤家型副CP

## 实践建议

1. **情感发展要有铺垫**：心动要有原因，不能无缘无故爱上
2. **冲突要合理**：避免"一个电话就能解决的误会"拖十章
3. **男女主都要有魅力**：读者需要理解"为什么是TA"
4. **虐要虐到点上**：虐心不是虐智商，保持角色人设不崩
5. **结局要圆满**：都市言情读者多期待HE，虐完一定要给糖
6. **肢体接触循序渐进**：从目光交汇到牵手到拥抱，层层递进

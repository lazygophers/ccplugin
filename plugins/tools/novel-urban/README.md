# Novel Urban Plugin

> 都市言情小说创作插件 - 专注现代都市背景和情感线设计

## 简介

Novel Urban 是一个专为都市言情小说创作设计的 Claude Code 插件，提供从都市背景构建到情感线设计的专业支持。无论你想写总裁文、职场恋爱还是豪门虐恋，都能通过本插件获得专业的创作辅助。

## 核心功能

### 🏙️ 都市背景设计 (Urban Setting)
- 现代城市环境和地标构建
- 商业帝国和企业背景设定
- 豪门家族权力结构设计
- 社会圈层和阶级差异刻画
- 高端生活场景和消费场所规划

### 💼 职场环境构建 (Workplace Design)
- 公司/集团架构设计
- 职场文化和人际关系
- 行业细节和专业术语
- 商战剧情和权谋布局
- 办公场景和工作流程

### 💖 情感线设计 (Romance Line)
- 男女主角相遇和发展
- 情感节奏和起伏把控
- 甜虐场景和心动时刻
- 情感冲突和误会设计
- 和解高潮和关系确立

## 都市言情特色类型

本插件深度支持以下都市言情小说类型：

### 总裁文
- **核心要素**：霸道总裁 + 平凡女主 + 身份悬殊 + 强势追求
- **经典桥段**：契约婚姻、误惹总裁、办公室恋情、豪门背景
- **背景需求**：跨国集团、商业帝国、豪宅别墅、上流社交圈

### 职场文
- **核心要素**：事业线 + 感情线 + 职场成长 + 专业背景
- **经典桥段**：办公室恋情、上司下属、竞争对手、商战阴谋
- **背景需求**：公司环境、行业细节、职场文化、业务场景

### 豪门文
- **核心要素**：家族势力 + 联姻/权谋 + 阶级差距 + 虐恋成分
- **经典桥段**：家族反对、豪门恩怨、身世秘密、虐心虐身
- **背景需求**：世家背景、权力结构、社交圈层、家族企业

### 青梅竹马文
- **核心要素**：童年回忆 + 分离重逢 + 暗恋成真 + 成长蜕变
- **经典桥段**：儿时约定、多年分离、身份变化、守护与等待
- **背景需求**：童年场景、时间跨度、成长轨迹、现在对比

### 先婚后爱文
- **核心要素**：契约/协议婚姻 + 相处日常 + 情感萌芽 + 真心确认
- **经典桥段**：协议条款、假戏真做、日常磨合、情感觉醒
- **背景需求**：婚姻生活、同居场景、家庭互动、社会关系

### 破镜重圆文
- **核心要素**：前情往事 + 分手原因 + 误会真相 + 重新开始
- **经典桥段**：旧情难忘、再次相遇、真相揭露、追妻火葬场
- **背景需求**：过去回忆、现在对比、情感积累、成长变化

## 最佳实践

### 都市背景设计

#### 真实性原则
- **行业知识要准确**：涉及金融、地产等专业领域时，基本术语和流程要正确
- **财富展示要合理**：避免过度夸张，万亿身家要有合理来源
- **地理逻辑要清晰**：场景转换要符合城市地理，注意交通时间

#### 功能性原则
- **背景服务情感**：都市设定要为情感线服务，不要为炫技而堆砌细节
- **场景推动剧情**：选择的场所要有戏剧功能，不只是单纯的环境描写
- **阶级制造冲突**：利用社会差距产生矛盾和张力

#### 细节加分项
```
✓ 品牌名称（服装、车、酒店等）
✓ 生活方式（饮食、运动、社交）
✓ 职场细节（会议、谈判、签约）
✓ 城市地标（真实或虚构但合理）
```

### 职场环境设计

#### 专业度体现
- **行业选择**：优先选择读者熟悉或向往的行业（金融、传媒、设计等）
- **职位设定**：匹配角色年龄和能力，避免25岁集团总裁等不合理设定
- **工作内容**：展现具体的工作场景，增加代入感

#### 人际关系网
```
公司架构示例：
集团董事长（男主父亲/家族长辈）
    ↓
总裁/CEO（男主）
    ├─ 特助（忠诚心腹）
    ├─ 各部门总监（同盟/对手）
    └─ 普通员工（女主/配角）
```

#### 商战剧情要素
- **对手设定**：竞争公司、内部派系、收购危机
- **权谋布局**：股权争夺、董事会斗争、商业间谍
- **危机事件**：项目失败、资金链断裂、媒体危机

### 情感线设计

#### 初遇设计法则
**三秒抓住读者**
- 制造冲突或反差（如女主泼男主一杯咖啡）
- 创造独特记忆点（特殊的对话或情境）
- 埋下情感伏笔（命中注定的暗示）

**经典初遇模式**
```
1. 误会型：认错人、误解身份、乌龙事件
2. 冲突型：利益对立、观念冲突、针锋相对
3. 救援型：英雄救美、雪中送炭、关键时刻出现
4. 命运型：巧合重逢、特殊缘分、宿命感
```

#### 情感浓度曲线

```
情感浓度
    ↑
100%|                                    ┌─── 结局圆满
    |                                ┌───┘
 75%|                            ┌───┘  ↗ 和解高潮
    |                        ┌───┘    ↙
 50%|                    ┌───┘    ↙ 冲突爆发
    |                ┌───┘      ↙
 25%|            ┌───┘      ↗ 心动确认
    |        ┌───┘      ↗
  0%|────┬───┘      ↗ 接触试探
    └────┴────┴────┴────┴────┴────┴────→ 章节进度
       初遇  磨合  升温  冲突  低谷  和解  HE
```

#### 甜虐比例建议

**纯甜宠文**：甜85% + 小虐15%
- 主打日常甜蜜和宠溺
- 小冲突快速解决
- 重点在撒糖和心动

**轻虐甜文**：甜65% + 虐35%
- 甜虐交替，有虐点但不过分
- 误会和矛盾较快化解
- 虐后必有加倍的甜

**虐恋文**：甜40% + 虐60%
- 前期或中期大量虐心
- 误会深重、分离时间长
- 后期追妻火葬场和深情弥补

#### 情感冲突设计

**外部阻力（客观障碍）**
```
家族层面：
- 门第差距：家族反对、联姻压力、继承条件
- 家族恩怨：父辈仇恨、商业竞争、利益冲突

社会层面：
- 身份悬殊：老板员工、富豪平民、明星素人
- 现实压力：经济差距、社会舆论、前途选择

事业层面：
- 职场竞争：同行对手、项目冲突、利益相悖
- 地域分离：异地恋、出国、工作调动
```

**内部矛盾（主观因素）**
```
信任危机：
- 误会积累：表面证据、他人挑拨、巧合叠加
- 过往隐瞒：身份秘密、前情未了、目的不纯

性格冲突：
- 强势vs倔强：两人都不肯低头
- 理性vs感性：价值观差异
- 事业型vs家庭型：生活重心不同

心理障碍：
- 过往创伤：被伤害的阴影、不信任感
- 自卑情结：配不上对方、逃避感情
- 恐惧承诺：害怕失去、不敢面对
```

#### 经典桥段活用

**霸道总裁必备桥段**
```
1. 壁咚/下巴抬起等肢体接触
2. "你是我的女人"式宣言
3. 为女主出头、护短
4. 醋意大发、占有欲爆棚
5. 办公室亲密、当众宣示主权
```

**虐心催泪桥段**
```
1. 误会爆发，女主心灰意冷
2. 男主发现真相，追悔莫及
3. 女主重病/危险，男主守护
4. 分离期间的思念和煎熬
5. 盛大的挽回或求婚仪式
```

**甜宠撒糖桥段**
```
1. 霸道宠溺："除了我，谁也不许碰你"
2. 日常照顾：生病照料、贴心细节
3. 公开秀恩爱：朋友圈、访谈、官宣
4. 特殊纪念：纪念日、专属礼物、浪漫惊喜
5. 闲适日常：做饭、看电影、旅行
```

### 配角功能设计

**情感支持系统**
- 女主闺蜜：提供情感建议、帮助分析局势
- 男主兄弟：撮合或帮助男主认清心意
- 长辈：提供过来人视角、温情支持

**冲突制造系统**
- 前任/追求者：制造吃醋和误会
- 情敌：豪门千金、白月光、竞争对手
- 反派：破坏感情、商业对手、家族反对者

**剧情推动系统**
- 工具人：传递信息、制造巧合
- 见证者：关键时刻出现、帮助澄清
- 催化剂：逼迫角色直面感情

## 常见问题

### Q1: 如何避免总裁文写得太老套？
**A**:
- 在经典框架内加入创新元素（如男主性格设定创新、行业背景新颖）
- 避免纯粹的霸道无理，给男主更多人性化细节
- 女主不要过于弱势，给她独立的事业线和成长空间
- 情节设计避免套路堆砌，寻找新的冲突点

### Q2: 豪门背景如何写得有质感？
**A**:
- 研究真实豪门的生活方式和社交规则
- 细节描写要精准（品牌、场所、礼仪）
- 展现豪门的温度和人情味，不只是冷冰冰的利益
- 权谋不能太复杂，读者要看得懂

### Q3: 情感线甜虐比例如何把控？
**A**:
- 根据目标读者群体调整（年轻读者偏甜宠，成熟读者能接受虐）
- 虐的时候给读者希望，不要绝望性虐待
- 甜的时候不要过度，保持一定张力
- 甜虐要有节奏，不能一直甜或一直虐

### Q4: 如何让职场背景更真实？
**A**:
- 选择自己熟悉或经过调研的行业
- 展现具体的工作内容，不只是开会
- 职场人际关系要复杂但不混乱
- 避免过度的职场暴力和不合理设定

## 进阶技巧

### 多角色视角叙事
交替使用男女主视角，增加情感层次：
- 女主视角：展现细腻情感和内心变化
- 男主视角：揭示冷漠外表下的柔情和执着

### 时间线处理
- 回忆杀：穿插过往片段，解释当下行为
- 时间跳跃：快速推进关系发展或展现成长
- 倒叙开局：从高潮或重要时刻切入

### 氛围营造
- 环境衬托：用天气、场景烘托情绪
- 细节描写：通过小动作展现心理活动
- 对话潜台词：不直说，让读者品味

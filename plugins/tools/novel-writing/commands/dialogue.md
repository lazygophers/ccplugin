# 生成或优化对话

帮助作者创作生动有力的角色对话，或对已有对话进行优化打磨。

## 适用场景

| 场景 | 示例 |
|------|------|
| **生成对话** | `/novel-writing dialogue 主角初遇反派` |
| **优化对话** | `/novel-writing dialogue 优化第10章对话` |
| **日常对话** | `/novel-writing dialogue 师徒日常互动` |
| **冲突对话** | `/novel-writing dialogue 谈判破裂` |
| **独白设计** | `/novel-writing dialogue 主角内心独白` |

## 执行流程

### 1. 确认对话场景

明确以下要素：
- **参与角色**：哪些角色参与对话
- **场景背景**：在什么情境下发生
- **对话目的**：推动情节/展示性格/传递信息/制造冲突
- **情绪基调**：紧张、轻松、悲伤、热血等
- **已有角色设定**：角色的性格和说话风格

### 2. 对话创作

#### 生成模式

根据角色性格和场景生成对话：
- 每个角色的语气和用词要有区分度
- 对话要推动场景目标的达成
- 适当穿插动作描写和心理活动
- 控制对话长度，避免冗长

#### 优化模式

对已有对话进行分析和改进：
- **去白开水**：删除无意义的寒暄和废话
- **加辨识度**：让每个角色的声音独特
- **增张力**：通过潜台词和冲突提升对话力度
- **调节奏**：长短句交替，控制对话节奏

### 3. 对话技巧应用

根据场景类型运用不同技巧：

#### 冲突对话
- 潜台词与表面意思的错位
- 话语权的争夺和转换
- 情绪的逐步升级
- 关键台词的爆发力

#### 日常对话
- 角色之间的默契和化学反应
- 幽默感和轻松感的营造
- 通过小细节展示角色关系
- 自然流畅，不刻意

#### 信息传递对话
- 避免大段解说式对白
- 通过冲突和疑问自然引出信息
- 听者的反应要合理
- 适当留白，不全部说透

#### 情感对话
- 克制比煽情更有力
- 用行动和细节代替直白表达
- 关键情感台词要有铺垫
- 沉默和停顿也是对话的一部分

### 4. 输出格式

提供两种输出格式：
- **纯对话稿**：仅包含对话内容
- **场景片段**：包含对话、动作、心理、环境描写的完整片段

## 对话写作原则

- **角色一致**：对话必须符合角色性格，读者能猜到谁在说话
- **功能明确**：每句对话都要有存在的理由
- **潜台词丰富**：好对话不只在表面
- **节奏感强**：长短交替，松紧有度
- **避免说教**：不通过对话灌输作者观点

## 注意事项

- 避免所有角色说话方式相同
- 避免通过对话强行解释设定（"你知道的，我们的世界是..."）
- 对话中的称呼要符合角色关系
- 注意对话的时代感和世界观适配

# 对话生成

## 功能概述

提供角色对话的创作方法论，确保对话符合角色性格、推进故事情节、传递信息的同时保持自然流畅，避免对话生硬、同质化。

## 使用场景

- 编写特定场景的角色对话
- 改善已有对话的质量和表现力
- 设计角色的专属语言风格
- 用对话推进情节或揭示信息
- 编写冲突性对话、情感对话等特定类型

## 核心要点

### 对话的四大功能

每段对话至少应实现其中一个功能：

1. **推进情节**：通过对话传递关键信息、触发事件
2. **揭示性格**：通过说话方式展现角色特质
3. **制造冲突**：通过观点碰撞和言语交锋激化矛盾
4. **营造氛围**：通过对话节奏和语气烘托场景情绪

### 角色语言差异化

不同角色的说话方式应有明显区分：

| 区分维度 | 表现方式 |
|---------|---------|
| 用词习惯 | 文雅/粗犷、专业术语/口语化 |
| 句式长短 | 长句（深思熟虑型）/ 短句（果断直接型）|
| 语气词 | 独特的口头禅、感叹词 |
| 说话逻辑 | 直奔主题 / 旁敲侧击 / 话里有话 |
| 信息量 | 话多（外向型）/ 惜字如金（内敛型）|

### 对话节奏控制

```
快节奏对话（冲突、紧张场景）：
  "住手。"
  "为什么？"
  "你知道为什么。"
  → 短句交锋，省略称呼和客套

中节奏对话（日常、推进场景）：
  正常的一问一答，穿插动作和心理描写
  → 信息传递为主，保持阅读舒适感

慢节奏对话（情感、哲理场景）：
  长段独白或深入交谈，大量留白和潜台词
  → 重在情感表达和思想碰撞
```

### 潜台词技巧

好的对话往往"话中有话"：

**原则**：角色说的不是他真正想说的，但读者能感受到真实意图。

**实现方法**：
- **转移话题**：用回避暴露内心在意的事
- **反话**：说相反的话来掩饰真实感受
- **借题发挥**：通过谈论别的事表达对当前事的态度
- **沉默**：不回答本身就是一种回答

### 对话中的动作和神态

对话不应只有引号内的文字，要配合：

```
❌ 纯对话：
"你要走了？"
"是的。"
"什么时候回来？"
"不知道。"

✅ 配合动作神态：
"你要走了？"她的手指无意识地攥紧了袖口。
他没有回头，只是在门口停了一瞬。"是的。"
"什么时候回来？"
沉默拉长。他推开门，门外的风灌进来。
"不知道。"
```

### 群戏对话管理

多人场景的对话处理：

1. **控制发言人数**：同一场景活跃发言者不超过 3-4 人
2. **区分声音**：每人说话风格明显不同，减少对话标签的依赖
3. **有主有次**：确定该场景的核心对话者，其他人可以旁观或简短插话
4. **动作定位**：通过穿插的动作描写让读者清楚谁在说话

## 实践建议

1. **删掉废话**：现实中的客套寒暄在小说中要大幅精简，直入主题
2. **避免信息倾倒**：不要让角色用对话大段解释设定，分散到多个场景中
3. **冲突对话最抓人**：两个角色有分歧时的对话最精彩，善用对话制造张力
4. **念出来检验**：写完对话后在心里念一遍，检查是否自然、是否像人说的话
5. **每句都有目的**：如果删掉一句对话不影响任何事，那就应该删掉它

## 示例

### 冲突对话示例

**场景**：师徒理念分歧

```
"你杀了他。"陆长青的声音很平静，像是在陈述天气。

沈夜收剑入鞘，没有擦去剑身上的血。"他该死。"

"他是个凡人。"

"他是个出卖情报导致三十七人丧命的凡人。"沈夜终于转过身，
目光平静得近乎冷酷，"师父教过我，除恶务尽。"

陆长青沉默了很久。风穿过废墟，吹动他斑白的鬓角。

"我教过你很多事。"他说，"但我没教过你当审判者。"

"那谁来审判？"沈夜的声音终于有了裂痕，"等宗门议事？
等那些收了好处的长老们商量出个结果？师父，三十七条命，
不能等。"

陆长青看着自己一手带大的弟子，有那么一瞬，他在沈夜的
眼中看到了某个不愿想起的故人的影子。

他转身离去，只留下一句话：
"你越来越像他了。"

沈夜的手猛地攥紧。他知道师父说的"他"是谁。
```

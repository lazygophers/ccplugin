# Novel Wuxia Plugin

> 武侠仙侠小说创作插件 - 提供江湖设定、武功招式、门派体系、江湖规则等武侠特色功能

## 简介

Novel Wuxia 是一个专为武侠仙侠类网络小说设计的 Claude Code 插件，提供从江湖格局到武功招式的全方位武侠世界构建支持。帮助作者快速建立有厚度、有烟火气的武侠江湖。

## 核心功能

### 江湖设定 (Jianghu Setting)
- 江湖势力版图和格局
- 武林规矩和武林道义
- 恩怨体系和冲突设计
- 侠客人物和江湖身份

### 武功招式 (Martial Arts)
- 内功心法体系
- 外功招式设计
- 轻功身法设计
- 暗器毒术体系
- 兵器谱和兵器武学

### 门派体系 (Sect System)
- 门派等级和内部架构
- 武学传承谱系
- 门规戒律和文化
- 弟子等级和培养

## 快捷命令

```bash
# 设计江湖设定
/novel-wuxia jianghu

# 设计武功招式
/novel-wuxia martial

# 设计门派体系
/novel-wuxia sect
```

## 专业 Agents

```bash
# 武侠世界观顾问
@wuxia-agent 构建一个以宋末为背景的武侠江湖

# 武功设计师
@martial-agent 设计一套以水为意的剑法
```

## 技能体系

| 技能 | 说明 | 适用场景 |
|------|------|---------|
| jianghu-setting | 江湖设定 | 江湖格局、武林规矩、恩怨体系 |
| martial-arts | 武功招式 | 内功心法、招式设计、兵器武学 |
| sect-system | 门派体系 | 门派架构、武学传承、门规戒律 |

## 使用场景

- 新作品的武侠世界观构建
- 江湖格局和势力版图设计
- 武功招式的创建和体系设计
- 门派创建和内部架构设计
- 恩怨关系和冲突设计
- 武林大事件的策划

## 创作流程建议

1. **格局设计**：使用 jianghu-setting 设计江湖势力格局
2. **门派构建**：使用 sect-system 创建各大门派
3. **武学设定**：使用 martial-arts 为门派配置标志性武学
4. **关联验证**：确保格局、门派、武功三者逻辑自洽

## 与 novel-writing 插件的关系

本插件是 novel-writing 的武侠仙侠类型扩展，专注于该类型的特色设定。建议配合 novel-writing 插件使用，获得完整的创作支持：
- novel-writing：通用创作功能（角色、情节、对话、文风）
- novel-wuxia：武侠特色设定（江湖、武功、门派）

# Plan Skill

## Process

1. 分析任务需求和约束
2. 分解为可执行子任务
3. 确定执行顺序和依赖
4. 识别关键路径和里程碑
5. 评估复杂度和资源需求
6. 制定风险应对策略
7. 输出执行计划

## Output

- 子任务分解清单
- 执行顺序和依赖关系
- 关键路径和里程碑
- 风险应对策略

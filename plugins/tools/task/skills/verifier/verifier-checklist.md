# Verifier 验证检查清单

<overview>

验证检查清单是 Verifier 执行验证时的标准流程参考。清单按时间顺序分为验证前和验证后两个阶段，确保验证过程的完整性和一致性。验证前检查确认任务状态和环境就绪，验证后检查处理结果并记录决策。

</overview>

<navigation>

## 验证前检查

文件：[verifier-checklist-pre.md](verifier-checklist-pre.md)

包含验证开始前需要确认的所有事项：任务完成性检查（确认所有子任务状态）、验收标准检查（逐条核对标准是否满足）、结构化验收标准检查（处理量化指标和容差）、环境和工具检查（确认测试环境和工具可用）、验证中检查清单（验证执行期间的步骤指引）。

## 验证后和最佳实践

文件：[verifier-checklist-post.md](verifier-checklist-post.md)

包含验证完成后的处理流程：验证后检查清单（结果汇总和报告生成）、注意事项（验证中应做和不应做的事项）、验收标准最佳实践（如何编写可验证的标准）、终止条件决策表（何时继续迭代、何时完成）、常见问题 FAQ。

</navigation>

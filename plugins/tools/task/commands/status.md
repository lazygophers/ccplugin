# 查看任务状态

显示当前会话中所有任务的执行状态。

## 信息收集

1. 读取 `.lazygophers/ccplugin/task/plans/` 目录中的所有子任务文件
2. 使用 `TaskList` 获取当前所有任务条目
3. 使用 `TaskGet` 获取每个子任务的详细状态
4. 对比计划文件和 TaskList 状态，确保一致性
5. 计算整体进度

## 输出格式

```markdown
# 任务状态

## 当前任务：[任务标题]

| # | 子任务 | 状态 | 验收 |
|---|--------|------|------|
| T1 | [标题] | 已完成 | 2/2 |
| T2 | [标题] | 执行中 | 0/3 |
| T3 | [标题] | 待执行 | - |
| T4 | [标题] | 失败 | 1/2 |

**总进度**：1/4 (25%)
**当前阶段**：执行组 2
**阻塞**：T4 失败，需修复后继续

## 最近操作
- [时间] T1 执行完成
- [时间] T2 开始执行
- [时间] T4 执行失败：[原因摘要]
```

## 无任务时

如果当前没有活跃任务：

```
当前没有活跃的任务计划。

使用方法：
- /plan [任务描述] - 创建新的任务计划
- /loop [目标] - 进入迭代执行模式
- /execute - 执行已确认的计划
```

# 文字校对清单

> novelist-proofread / proofreader 用。只在文字层工作, 不碰剧情设定。

## 五维逐项

### 1 错别字
- [ ] 形近字（己/已、的/得/地)
- [ ] 音近字
- [ ] 多字 / 漏字

### 2 语法 / 病句
- [ ] 成分残缺（缺主/谓/宾)
- [ ] 搭配不当
- [ ] 语序错乱
- [ ] 关联词误用

### 3 标点
- [ ] 误用（逗号该句号)
- [ ] 缺失
- [ ] 中英标点混用（,。 vs ,.)
- [ ] 引号/书名号配对

### 4 用词
- [ ] 啰嗦重复（同段重复词)
- [ ] 口语书面混杂
- [ ] 生造词

### 5 用字统一(以基准为准)
- [ ] 人物称呼/译名前后一致（基准: 人物/_索引.md)
- [ ] 专有名词/术语用字一致（基准: 设定/_索引.md)

## 报告格式(每条)

```
[类别] 位置: 章节/第NNN章-xxx.md:行号
  原文: <错处>
  改后: <修正>
```

## 边界红线(必守)

- 只改"文字对不对、顺不顺"; 不评判剧情合不合理。
- 疑似剧情/设定矛盾 → 不自行改, 记一条「建议跑 novelist-check」。
- 拿不准是错误还是风格 → 标出请人定, 不擅改。
- 方言/口头禅/有意的破碎句 = 风格, 保留, 禁当错别字抹平。
- 统一基准缺失 → 该维度标「基准待定」, 不强统一。

## 收尾

正文就地修正 → 清单落 元数据/校对报告/<日期>-<范围>.md → 章节/_索引.md 对应章状态推进「已校对」。

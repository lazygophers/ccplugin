# 人物三文件模板

> novelist-character 落盘时用。每个人物 人物/<人物名>/ 下三文件, 各保持单一职责。

## 简介.md 模板

```markdown
# <人物名>

- 定位: 主角 / 反派 / 关键盟友 / 配角
- 首次出场: 第NNN章
- 一句话: （他是谁 + 最想要什么）

## 外貌
（被精确选中的 1-2 个记忆点, 非全身描写）

## 性格（可驱动行为的条目, 非空泛形容词）
- 核心内核: （决定命运的那一条）
- 致命缺口: （性格里必须有的破绽——防"完美伪人")
- 行为倾向: 面对冲突时倾向 X / 面对诱惑时倾向 Y

## 动机
- 表层目标: 
- 深层欲望: （驱动主线的那个）

## 说话风格
（口头禅 / 句式 / 用词特征——校对时作为统一基准）

## 能力
- 核心能力 + 其"美学风格"(招式名/施法方式, 与性格同构)
- 边界与代价（受 世界观/规则.md 约束）
```

## 经历.md 模板

```markdown
# <人物名> · 经历

> 按时间段排列, 每条标时间锚点, 与 世界观/历史.md 对齐。

| 时间锚点 | 经历/事件 | 对人物的影响 | 涉及章节 |
|---|---|---|---|
| 出场前·童年 | | | — |
| 第NNN章 | | | 第NNN章 |
```

## 关系.md 模板

```markdown
# <人物名> · 关系

> 与他人交集。改了对某人的关系, 提示同步对方的 关系.md(双向)。

| 对方 | 关系类型 | 是否见过面 | 首次相遇章 | 当前态度 | 备注 |
|---|---|---|---|---|---|
| <人物名> | 盟友/对手/亲人/未知 | 是/否 | 第NNN章 | 信任/敌视/警惕 | |
```

## 填写原则

- 性格写"可驱动行为"的条目: 不写"勇敢", 写"明知会死也不退——第N章面对X时如此"。
- 信息缺口大 → 逐项问用户, 实在没有标「待补」, 禁大段虚构。
- 主角的"深层欲望"必须能驱动 大纲/总纲.md 的主线冲突(见 novelist-design 闭环)。

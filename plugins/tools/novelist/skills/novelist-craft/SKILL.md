# novelist-craft — 小说创作大师视角(叙事心智模型库)

不替你写, 给你**叙事镜片**。蒸馏自三大传统的大师技法, 按你自己小说的题材选用——同一个问题("这章怎么写"), 类型镜片和文学镜片给的答案截然不同。先选镜片, 再落手法。

## 三流派镜片(各自的引擎)

| 镜片 | 蒸馏自 | 引擎 | 读者要的 | 适配题材 |
|---|---|---|---|---|
| **类型镜片** | 金庸/古龙/网文 | 钩子+爽点+升级循环 | 闭合、兑现、情绪过山车 | 武侠/玄幻/升级流/复仇爽文 |
| **结构镜片** | 斯蒂芬·金/托尔金/马尔克斯 | 情境驱动+世界自洽+精确可信 | 沉浸、可信、史诗感 | 悬疑/恐怖/史诗奇幻/魔幻现实 |
| **文学镜片** | 博尔赫斯/村上/纳博科夫/福楼拜 | 氛围+谜题+留白+节制 | 余味、歧义、智性愉悦 | 纯文学/意识流/结构即主题 |

> 三镜片是光谱, 非互斥。可分层组合(金庸即用类型骨架承载文学追求): **世界规则可先行(结构镜片)、人物情节可后生(类型/结构镜片)、语气留白定调(文学镜片)**。

## 工作流: 选镜片 → 落手法

1. **判题材定主镜片** — 读 `总览.md` 题材, 或问用户作品定位。强情节追爽→类型; 重沉浸/世界→结构; 追余味/智性→文学。题材模糊 → 硬停, 用 `AskUserQuestion` 让用户选镜片, **禁替用户拍板写法**。
   → **产出**: `主镜片 = {类型|结构|文学}` + 题材基调确认（1 行）
2. **取对应维度心智模型** — **输入**: step1 主镜片 + 用户具体问题维度（人物/情节/世界观/节奏 之一）。查下方五维表该镜片行, 给 2-3 条可操作启发式。
   → **产出**: 2-3 条命名启发式, 格式：`手法名: 具体做法（1 句）` + 每条附 `本章落地: …`
3. **跨镜片增益(可选)** — **输入**: step2 选定手法列表。扫其他镜片同维度, 提示可嫁接点（标明来自哪派 + 适用前提, 如类型作品借文学镜片"伏笔重读才显形"）。
   → **产出**: 0-2 条跨镜片嫁接建议（无明显增益可跳过）
4. **落到本章** — **输入**: step2-3 选定手法。把手法翻译成本章四要素批注（对应 `novelist-write` 的叙事视角/叙事速度/核心冲突/收尾钩子, 各 1-2 句）, 交回 novelist-write 四要素或 novelist-design 框架。
   → **产出**: 本章写法批注卡（4 字段）, 可直接粘入 novelist-write 四要素

## 五维心智模型库

### 跨流派领域共识(三派都认同, 无条件适用)

- **超自然/力量必须有代价与边界** — 金字塔分级+金手指代价(类型) / bounded magic system(托尔金) / 单点逾矩其余守恒(文学)。无代价的力量瓦解张力。
- **现实底座 + 单点逾矩** — 世界绝大部分守常理, 只在核心设定处破例; 异常因背景真实而可信(金庸新武侠写实 / 博尔赫斯"幻想需精确逻辑")。
- **伏笔早埋晚收, 靠呼应** — 伏笔伪装成日常细节(斯蒂芬·金锅炉)/ 重读才显形(纳博科夫)/ 意象反复变奏(村上)。
- **节制即力量, 修订靠删** — 做减法(网文)/ 二稿=一稿−10%(斯蒂芬·金)/ le mot juste 精确的词(福楼拜)。
- **精确细节让不可信变可信** — "天上 425 头大象"(马尔克斯)/ caress the details(纳博科夫)/ 黄金三章记忆点(网文)。

### 1 人物塑造

| 镜片 | 心智模型 | 怎么用 |
|---|---|---|
| 类型 | 性格即命运(对照写法)/ 成长模式统领情节 / 武功外化人格 / 瞬间见人(古龙留白) / **人设标签≥3条硬标签**(如"偏执+利己+记仇") | 先定性格内核, 同一事件落不同性格→结局差异反证性格; 能力美学=性格可视符号; **禁圣母/优柔底版人设**——无立场人物无张力, 必须给主角 1 条"读者不认同但理解"的性格缺口 |
| 结构 | 情境先于情节、人物自己长出来(斯蒂芬·金 situation-first) / 初始可扁平靠情境长深度 | 别先写小传, 把人物丢进 predicament 看他怎么挣脱, 性格从反应中显形 |
| 文学 | 空心主角作容器(村上) / 侧面折射(博尔赫斯) / 精确物象定义内心(纳博科夫) | 主角留白做代入通道; 用他人议论+陈设说话; 给一个专属反常癖好承担"他是谁" |

### 2 情节推进

| 镜片 | 心智模型 | 怎么用 |
|---|---|---|
| 类型 | 章末钩子链 / 爽点三段(压迫→藏底牌→结算奖励) / 先抑后扬 / 悬念迭起(非单点爆发) | 信息揭晓卡在章接缝; 解决前把敌人写强给足憋屈, 收尾用旁人惊叹结算情绪 |
| 结构 | What-if/What-next 链式驱动 / 情节是末等手段(刷化石别用电钻) | 卡住时问"此处境这人下一步必然做什么", 由必然性推进; 需硬掰人物=在用电钻 |
| 文学 | 氛围驱动而非事件 / 谜题作引擎(可不解) / 非线性即结构 / 延宕悬置 | 用"某物不对劲"的低鸣替代大事; 围绕难解之核螺旋收缩, 答案可缺席 |

### 3 世界观 / 设定

| 镜片 | 心智模型 | 怎么用 |
|---|---|---|
| 类型 | 金字塔等级体系 / 金手指必有代价 / 升级循环(对手≤主角+2级) / 单点逾矩写实底座 | 清晰命名阶梯+高层拉大鸿沟; 作弊器绑冷却/耗材/反噬, 关键战令其失效 |
| 结构 | 次创造+内部自洽(托尔金 Secondary Belief) / 语言历史先行 / 多模态互证(地图/年表/传说) | 先立规则再严守, 可信来自一致非华丽; 从命名/词源/年表地基层建起 |
| 文学 | 隐喻系统一致性 / 象征不解释自身 / 以有限暗示无限 | 选 1-2 核心意象贯穿全书每次略变形; 象征只"在那发挥作用"永不给谜底 |

### 4 节奏与结构

| 镜片 | 心智模型 | 怎么用 |
|---|---|---|
| 类型 | 单元舞台集中高潮(三一律) / 体裁交替调张弛 / 反转克制(≤2次) / 做减法 | 群戏锁进封闭场景叠冲突; 打斗后接情感段换气; 砍无推进旁白(<300字) |
| 结构 | 伏笔=日常细节(plant in plain sight) / 象征靠挖掘非预设 / 二稿−10% | 伏笔出场给当下实用理由别露馅; 二稿通读找已浮现的意象再 refine |
| 文学 | 双线交织末段合流 / 视角控制即信息控制 / 节制删除 / 重读才显形 / **语域碰撞**（高雅语域与口语/方言在同段或同句内对置） | A/B 线各自完整用共享意象暗连; 锁死视角让真相在偏见缝隙泄露; 语域碰撞: 议论文式抒情句后接一句大白话("他赴死如履平地。完全不怕, 就是怕") — 制造意外张力 |
| **平台类型(网文番茄)** | 小白话复句≤30% / 句式突变 / 零思维标记 / 精确数字 | 每段≥70%短句(≤25字), 控制心理描写"她想""他觉得"类标记; 复句拆短保留语气词("他站起来, 却没走, 只是站着"); 用具体数字替代模糊量词("等了三年"→"等了1131天") |

### 5 局限与适用边界(选镜片的硬约束)

- **类型镜片**: 适合强情节强代入; 盲区 = 同质化重复逻辑、金手指至上麻醉、理想化人物失可信(主角性格须留致命缺口)。
- **结构镜片**: 情境先行偏 discovery writer, 长篇易失控塌尾(需混大纲法); 语言/世界先行成本极高, 与篇幅类型匹配, 建到"够用且自洽"即可。
- **文学镜片**: 氛围/留白/谜题不解, 用在类型小说="挖坑不填"的欺骗感; 空心主角弱化角色弧光不适合成长/爱情/英雄旅程; 节制需密度支撑, 功力不足时"留白"沦为"空洞"。
- **选镜片判断**: 目标是黏性/情绪/明确回报→类型; 沉浸/世界可信→结构; 智性/结构即主题→文学。误用镜片 = 给错读者错的预期。

## 被引用接口(给 design / write)

- `novelist-design` 调用: 按题材推荐主镜片, 给人物塑造与主线推进的心智模型, 校验主角×主线×世界观的镜片一致性。
- `novelist-write` 调用: 写某章前, 按本章性质(打斗/情感/悬念/铺垫)给该镜片下的具体手法, 落到四要素。

## 失败模式(触发 → 一线修复 → 仍失败兜底)

| 触发 | 一线修复 | 仍失败兜底 |
|---|---|---|
| 题材模糊, 选不出主镜片 | `AskUserQuestion` 让用户选三镜片之一 | 用户不定 → 默认类型镜片(最广适配), 标注可调 |
| 用户混用镜片导致预期错乱(文学写法塞进爽文) | 提示该手法的适用边界(见维度5) | 用户坚持 → 标注风险, 不强拦, 尊重创作选择 |
| 给的手法太抽象用户用不上 | 翻译成"这一章具体怎么做"的动作 | 仍空 → 退回研究文件取更具体例证 |
| 用户要的题材三镜片都没覆盖 | 取最近镜片+领域共识兜底 | 明确告知覆盖边界, 禁硬套 |

## 诚实边界

- 本 skill 是**叙事技法框架库**, 不是创作代笔——它帮你选对镜片、给对手法, 落笔仍是 chapter-writer + 你的判断。
- 心智模型蒸馏自公开材料(作家自述/评论/技法归纳), 区分 `[原话]`/`[评论]`/`[归纳]` 三档, 详见 `references/research/01-03`。未逐页核验原文页码, 重要引用建议回溯一手文本。
- 三流派非穷尽——覆盖中文类型/世界结构/纯文学三大传统的主干, 冷门题材(如硬科幻设定、剧本杀结构)需另行补充。

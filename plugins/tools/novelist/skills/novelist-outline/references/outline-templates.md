# 大纲与情节线模板

> novelist-outline 落盘时用。宏观(大纲)与逐节(情节)分层, 伏笔显式追踪。

## 大纲/总纲.md 模板

```markdown
# 总纲（主线)

## 核心冲突
（贯穿全书的主要矛盾, 一句话)

## 三幕 / 分卷结构
- 第一幕（建置): 起 → 第一个转折点
- 第二幕（对抗): 升级 → 中点 → 危机
- 第三幕（结局): 高潮 → 收束

## 结局走向
（终点, 与主角深层欲望呼应)
```

## 大纲/分卷.md 模板

```markdown
# 分卷大纲
| 卷 | 主题 | 起止章 | 卷末钩子 |
|---|---|---|---|
```

## 情节/主线.md 模板

```markdown
# 主线情节
| 节点 | 涉及章节 | 推进的核心冲突 |
|---|---|---|
```

## 情节/支线.md 模板

```markdown
# 支线情节
| 支线 | 涉及人物 | 涉及章节 | 与主线交汇点 |
|---|---|---|---|
```
> 每条支线必须有"与主线交汇点", 否则并为独立番外。

## 情节/伏笔.md 模板(check 的关键输入)

```markdown
# 伏笔与回收追踪
| 伏笔 | 埋设章 | 计划回收章 | 状态 |
|---|---|---|---|
| <一句话> | 第NNN章 | 第NNN章 | 未回收 / 已回收 |
```
> 每个伏笔必填"计划回收章"; 暂不定填"待定", novelist-check 会高亮。
> 过了计划回收章仍"未回收" = 重要级; 结尾仍悬空 = 致命级。

## 填写原则

- 每个节点关联到章节区间, 禁纯文字流水。
- 新规划先对齐 大纲/总纲.md 核心冲突与结局, 矛盾则请用户裁定。
- 镜片差异: 类型镜片重章末钩子+爽点节奏; 文学镜片可非线性/双线交织(见 novelist-craft)。

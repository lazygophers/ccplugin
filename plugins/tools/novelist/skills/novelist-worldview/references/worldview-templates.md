# 世界观与设定模板

> novelist-worldview 落盘时用。规则.md 是一致性硬约束源, 模板强制"边界+代价"字段。

## 规则.md 模板(力量/科技/魔法体系——最关键)

```markdown
# 规则体系

## <体系名, 如 真气/魔法/纳米科技>

### 能力分级（如有等级体系)
| 级别 | 名称 | 能做什么 | 与下一级的差距 |
|---|---|---|---|

### 每种能力（必填边界与代价)
- **<能力名>**
  - 能做什么: 
  - **不能做什么（边界)**: 
  - **代价/限制（冷却/耗材/反噬/可被屏蔽)**: 
  - 谁能用 / 获取途径: 
```

> 无"不能做什么 + 代价"的能力不允许定稿——无边界的力量是一致性灾难。

## 地理.md 模板

```markdown
# 地理
## <区域名>
- 位置/与他区关系: 
- 地标: 
- 气候/环境: 
- 移动成本（影响人物行程的距离/时间约束): 
```

## 势力.md 模板

```markdown
# 势力
## <势力名>
- 类型: 国家/门派/组织
- 势力范围 / 核心利益: 
- 与其他势力关系: 敌/盟/中立
- 关键人物（→ 人物/): 
```

## 历史.md 模板

```markdown
# 历史
| 时间 | 事件 | 对当下格局的影响 |
|---|---|---|
```

## 设定/<名>.md 模板(物品/组织/术语)

```markdown
# <设定名>
- 类型: 物品 / 组织 / 术语 / 专有名词
- 定义（唯一事实源, 所有引用以此为准): 
- 首次出现: 第NNN章
- 关联规则/势力: 
```

## 填写原则

- 一设定一文件, 唯一事实源; 同一术语多处定义不一 → 合并。
- 新增 设定/ 文件必更新 设定/_索引.md。
- 力量体系优先写代价——它是 novelist-check 判"世界观违规"的标尺。

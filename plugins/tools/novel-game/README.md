# Novel Game Plugin

> 游戏竞技小说创作插件 - 提供游戏系统、技能设定、竞技设计、副本关卡等游戏特色功能

## 简介

Novel Game 是一个专为游戏竞技类网络小说设计的 Claude Code 插件，提供从游戏系统到电竞赛事的全方位游戏世界构建支持。帮助作者快速建立合理有趣、充满竞技热血的游戏世界观。

## 核心功能

### 游戏系统设计 (Game System)
- 游戏类型定位与核心玩法
- 职业体系与转职路线
- 等级系统与数值平衡
- 经济体系与货币循环

### 技能设定设计 (Skill Design)
- 技能树与技能分支
- 连招体系与操作深度
- 装备品阶与词缀系统
- 天赋路线与Build多样性

### 竞技设计 (Competition)
- 电竞赛事体系与规则
- 排位段位与匹配系统
- 战术策略与团队配合
- 公会战与大规模PVP

## 快捷命令

```bash
# 设计游戏系统
/novel-game game-system

# 设计技能设定
/novel-game skill

# 设计竞技对战
/novel-game competition
```

## 专业 Agents

```bash
# 游戏系统设计师
@game-agent 设计一款全息VR网游的完整系统

# 竞技设计师
@competition-agent 设计职业电竞联赛赛事体系
```

## 技能体系

| 技能 | 说明 | 适用场景 |
|------|------|---------|
| game-system | 游戏系统设计 | 职业体系、等级系统、经济循环 |
| skill-design | 技能设定设计 | 技能树、装备系统、连招体系 |
| competition | 竞技设计 | 赛事规则、排位系统、战术策略 |

## 使用场景

- 网游类小说的游戏世界观构建
- 电竞类小说的赛事体系设计
- 全息/VR游戏的系统设计
- 游戏内职业和技能体系设定
- 公会战/攻城战等大规模战斗设计
- 数值平衡和经济系统规划

## 创作流程建议

1. **系统设计**：使用 game-system 确定游戏类型和核心框架
2. **战斗构建**：使用 skill-design 设计技能和装备体系
3. **竞技规划**：使用 competition 构建赛事和排位系统
4. **关联验证**：确保游戏系统、技能平衡、竞技规则三者逻辑自洽

## 与 novel-writing 插件的关系

本插件是 novel-writing 的游戏竞技类型扩展，专注于该类型的特色设定。建议配合 novel-writing 插件使用，获得完整的创作支持：
- novel-writing：通用创作功能（角色、情节、对话、文风）
- novel-game：游戏特色设定（系统、技能、竞技）

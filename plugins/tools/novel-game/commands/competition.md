# 设计竞技对战

帮助作者设计游戏中的竞技对战体系和赛事场景。

## 适用场景

| 场景 | 示例 |
|------|------|
| **赛事体系** | `/novel-game competition 设计职业电竞联赛` |
| **排位系统** | `/novel-game competition 段位和匹配机制` |
| **战术策略** | `/novel-game competition MOBA类战术体系` |
| **公会战** | `/novel-game competition 百人攻城战设计` |

## 执行流程

### 1. 确认竞技方向

通过提问明确：
- **竞技类型**：职业赛事/排位对战/公会战/PVP模式
- **游戏类型**：MOBA/FPS/MMORPG/格斗/卡牌
- **规模大小**：1v1/小队/团队/大规模
- **故事定位**：主角是职业选手/普通玩家/公会会长

### 2. 激活技能生成方案

激活 `Skills(competition)` 并根据方向生成：

#### 赛事体系
- 联赛结构和赛季安排
- 赛制规则（BO3/BO5/循环/淘汰）
- 队伍构成和选手位置
- 赛事奖金和积分系统

#### 排位系统
- 段位划分和晋升条件
- 匹配算法和隐藏分机制
- 赛季重置和定位赛
- 段位保护和衰减

#### 战术策略
- 不同阵容的战术风格
- 禁选（Ban/Pick）策略
- 关键时间节点的决策
- 团队配合和指挥体系

#### 大规模PVP
- 公会战/攻城战规则
- 据点争夺和资源分配
- 多方势力博弈
- 战场指挥和调度

### 3. 场景描写验证

- 比赛节奏是否紧凑有看点
- 战术设计是否有足够的策略深度
- 翻盘机制是否合理（为主角逆袭留空间）
- 解说/观众视角是否能增强氛围

## 设计原则

- **竞技公平**：比赛结果由技术和策略决定，不靠数值碾压
- **观赏性强**：比赛场面要精彩，读者读起来也热血沸腾
- **成长合理**：主角的竞技水平逐步提升，每场比赛都有收获
- **对手出彩**：强大的对手让胜利更有价值

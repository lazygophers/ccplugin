# Novel Romance Plugin

> 言情小说创作插件 - 提供情感线设计、互动场景、心理刻画、感情冲突等言情特色功能

## 简介

Novel Romance 是一个专为言情类网络小说设计的 Claude Code 插件，提供从情感弧线到关系互动的全方位言情创作支持。帮助作者构建有深度、有层次、有张力的感情线。

## 核心功能

### 情感线设计 (Emotional Arc)
- 感情发展阶段模型
- 心理变化刻画技法
- 内心独白与情感外化
- 不同性格的情感表达

### 感情线进展 (Relationship Dynamics)
- 关系发展模式设计
- 互动场景编排
- 感情推进节奏控制
- 多线感情的处理

### 感情冲突 (Romantic Conflict)
- 冲突类型与升级路径
- 误会设计与沟通障碍
- 矛盾化解策略
- 虐心程度控制

## 快捷命令

```bash
# 设计情感线与心理刻画
/novel-romance emotion

# 设计互动与关系发展
/novel-romance relationship

# 设计感情冲突与化解
/novel-romance conflict
```

## 专业 Agents

```bash
# 感情线设计师
@romance-agent 设计一条从欢喜冤家到甜蜜恋人的感情线

# 心理分析师
@emotional-agent 分析女主发现自己心动后的心理变化
```

## 技能体系

| 技能 | 说明 | 适用场景 |
|------|------|---------|
| emotional-arc | 情感线设计 | 情感弧线、心理刻画、内心独白 |
| relationship-dynamics | 关系互动 | 互动场景、关系推进、节奏控制 |
| romantic-conflict | 感情冲突 | 冲突设计、误会桥段、矛盾化解 |

## 使用场景

- 新作品的感情线整体规划
- 角色情感弧线和心理设计
- 互动场景和感情升温设计
- 冲突桥段和虐心情节设计
- 多条感情线的平衡编排

## 创作流程建议

1. **关系设计**：使用 relationship-dynamics 规划感情发展路线
2. **情感深化**：使用 emotional-arc 设计心理变化和内心戏
3. **冲突安排**：使用 romantic-conflict 设计冲突与化解
4. **整合验证**：确保感情线、心理变化、冲突节奏三者协调

## 与 novel-writing 插件的关系

本插件是 novel-writing 的言情类型扩展，专注于该类型的特色设定。建议配合 novel-writing 插件使用，获得完整的创作支持：
- novel-writing：通用创作功能（角色、情节、对话、文风）
- novel-romance：言情特色设定（情感线、互动、冲突）

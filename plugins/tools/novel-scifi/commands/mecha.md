# 设计机甲系统

帮助作者设计机甲战斗体系，包括机体设定、战斗系统、驾驶员体系等。

## 适用场景

| 场景 | 示例 |
|------|------|
| **创建机体** | `/novel-scifi mecha 设计主角专属机甲` |
| **机甲分类** | `/novel-scifi mecha 机甲世代和型号体系` |
| **战斗系统** | `/novel-scifi mecha 机甲战斗机制和武器系统` |
| **驾驶员体系** | `/novel-scifi mecha 驾驶员等级和适配机制` |
| **机甲进化** | `/novel-scifi mecha 机体改装和进化路线` |

## 执行流程

### 1. 确认设计方向

通过提问明确：
- **设计目标**：单机设计/机甲分类体系/战斗机制/完整体系
- **机甲定位**：主角机体/量产机/BOSS机/特殊机型
- **战斗风格**：超级系（燃）/真实系（硬核）/混合系
- **已有设定**：与已有世界观和科技体系的匹配

### 2. 激活技能生成方案

激活 `Skills(mecha-system)` 并根据方向生成：

#### 单机设计

**基础参数**
- 机体名称、型号、代号
- 世代归属和开发背景
- 高度、重量、装甲材质
- 动力系统和能源类型
- 推进系统和机动性能
- 传感器系统和通讯模块

**武器装备**
- 主武器（光束剑、实体剑、巨型枪炮）
- 副武器（光束步枪、导弹、浮游炮）
- 远程武器（狙击炮、能量炮）
- 防御装备（盾牌、I立场、相位装甲）
- 特殊装备（隐身系统、分身残像、时间操控）

**特殊系统**
- TRANS-AM、SEED模式等爆发系统
- 零式系统、天人系统等AI辅助
- 精神力感应、念动力放大
- 变形机构（MA形态、战斗机形态）
- 合体机制和组合攻击

**外观设计**
- 整体风格（流线/厚重/异形/仿生）
- 配色方案和识别标志
- 标志性特征（独角、翅膀、面甲）
- 战损状态和残破美学
- 发光部位和粒子特效

**成长路径**
- 初始形态和基础性能
- 改装节点和强化方向
- 最终形态和觉醒状态
- 与驾驶员的羁绊进化
- 毁灭与重生（如果有）

#### 机甲分类体系

**世代划分**
- **第一代**：早期试验型，性能有限但意义重大
- **第二代**：量产化开始，实用性提升
- **第三代**：技术成熟，性能平衡
- **第四代**：高性能机，开始出现特化机型
- **第五代及以上**：划时代机体，搭载黑科技

**型号分类**
- **量产机**：标准配置，性能均衡，数量众多
- **指挥官机**：性能优于量产机，有特殊涂装
- **王牌机**：定制机体，性能大幅提升
- **试验机**：搭载新技术，不稳定但潜力巨大
- **专用机**：为特定王牌定制，独一无二
- **超限机**：禁忌技术，性能恐怖但代价巨大

**功能定位**
- **泛用型**：全能均衡，适应各种战场
- **近战型**：高机动，配备近战武器
- **远程型**：重火力，远距离压制
- **狙击型**：超远程精确打击
- **重装型**：高防御高火力，机动性差
- **指挥型**：信息战能力强，协调作战
- **特殊型**：电子战、工程、医疗等支援

#### 战斗系统设定

**战斗机制**
- 操作方式（手动操纵/思维感应/AI辅助）
- 反应速度和机体负荷
- 能量管理和武器切换
- 护盾和装甲的防御机制
- 破坏判定和损伤系统
- 驾驶舱防护和逃生机制

**战术体系**
- 单机战术（格斗、游击、火力压制）
- 小队战术（包围、掩护、陷阱）
- 大规模会战（阵型、航母、防线）
- 特殊战术（空降、潜入、暗杀）
- 地形利用（宇宙/地面/水下/殖民地）

**克制关系**
- 近战克制狙击（近身压制）
- 重装克制泛用（火力碾压）
- 泛用克制近战（平衡优势）
- 狙击克制重装（远程风筝）
- 特殊机型的特殊克制

**战斗演出**
- 开场对峙和台词
- 战斗动作描写（剑戟交锋、光束乱射）
- 环境破坏效果
- 绝招释放和斩杀特写
- 战损状态和极限爆发

#### 驾驶员体系

**等级划分**
- **新手**：刚入门，只能驾驶训练机
- **正式驾驶员**：能驾驶量产机作战
- **精英**：王牌预备役，能驾驶高性能机
- **王牌**：驾驶定制机，战场主宰者
- **传说**：超越常人，能引发奇迹
- **超越者**：达到人机合一境界

**适配机制**
- 同调率/适配度概念
- 基因匹配和量子纠缠
- 精神波长和意志共鸣
- 特殊血统和被选中者
- 后天训练 vs 天赋觉醒

**身体负荷**
- 高机动造成的重力负荷
- 长时间作战的精神损耗
- 超负荷模式的身体伤害
- 神经连接的副作用
- 战斗创伤和PTSD

**成长体系**
- 新手期的挫折和成长
- 关键战役的突破
- 师徒传承和对手磨砺
- 觉醒和开挂时刻
- 从王牌到传说的蜕变

#### 机甲进化路线

**改装系统**
- 武器更换和升级
- 装甲强化和轻量化
- 动力炉更新换代
- 操作系统优化
- 外挂装备（背包、副臂）

**进化阶段**
- **基础型**：初始状态，性能平平
- **改良型**：小幅改装，性能提升
- **强化型**：大幅改装，判若两机
- **最终型**：极限改造，接近性能上限
- **觉醒型**：超越设计，奇迹之机

**特殊进化**
- 与驾驶员意识共鸣觉醒
- 吸收古代遗迹能量
- 吞噬其他机体进化
- 转生重生（毁灭后重生更强）
- 神机降临（被高维存在选中）

### 3. 整合验证

- 机甲性能与战斗描写的契合度
- 机体设定与世界观科技水平的匹配
- 主角机的独特性和成长空间
- 战斗系统的平衡性和观赏性
- 是否有足够的热血燃点

### 4. 输出设定文档

以结构化格式输出完整设定，包括：
- 机体参数表和性能图
- 武器装备列表和效果说明
- 外观设定描述（可搭配参考图）
- 战斗风格和必杀技
- 进化路线和关键节点
- 驾驶员适配要求

## 设计原则

- **燃系优先**：机甲小说的核心是热血和爽感
- **独特标识**：每台重要机体要有鲜明的识别特征
- **成长感**：主角机要有清晰的成长进化路径
- **战斗节奏**：前期碾压→遇强敌→升级→再碾压的循环
- **必杀技**：每台机体要有标志性的绝招和名台词
- **人机合一**：机体是驾驶员的延伸，要体现羁绊
- **平衡感**：强大的机体要有合理的限制或代价
- **视觉化**：描写要有画面感，方便读者脑补动画

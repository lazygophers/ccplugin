# 设计星际文明

帮助作者设计外星文明或构建完整的星际社会格局。

## 适用场景

| 场景 | 示例 |
|------|------|
| **创建种族** | `/novel-scifi civilization 设计一个硅基文明` |
| **文明等级** | `/novel-scifi civilization 卡尔达肖夫等级体系` |
| **星际格局** | `/novel-scifi civilization 银河系势力版图` |
| **外交体系** | `/novel-scifi civilization 星际联盟和外交规则` |
| **文明冲突** | `/novel-scifi civilization 文明战争设定` |

## 执行流程

### 1. 确认设计范围

通过提问明确：
- **设计目标**：单个种族/文明等级体系/星际格局/完整银河设定
- **种族定位**：人类/友善外星种族/敌对种族/神秘文明
- **文明形态**：生物文明/机械文明/能量生命/维度生命
- **已有设定**：与已有科技体系和故事背景的关联

### 2. 激活技能生成方案

激活 `Skills(civilization-system)` 并根据方向生成：

#### 单个种族设计

**生理特征**
- 生命形态（碳基/硅基/能量/机械/混合）
- 外貌特征和身体结构
- 感知方式和交流方式
- 繁殖方式和寿命
- 特殊能力和限制

**文化传统**
- 价值观和哲学体系
- 社会结构（等级制/民主制/集体意识/无政府）
- 宗教信仰和精神追求
- 艺术形式和审美观
- 历史传承和重大事件

**科技水平**
- 文明等级定位
- 优势科技领域
- 科技发展特色（生物科技/机械科技/灵能科技）
- 弱势和技术短板
- 黑科技或独特技术

**政治制度**
- 政府形式和权力结构
- 领导人选拔机制
- 决策流程和效率
- 内部派系和权力斗争
- 扩张策略和外交政策

#### 文明等级体系

**卡尔达肖夫等级（或自定义）**
- **0级文明**：行星前文明，尚未充分利用母星资源
- **1级文明**：行星文明，能完全利用母星能源（人类约0.73级）
- **2级文明**：恒星文明，能利用整个恒星能源（戴森球）
- **3级文明**：星系文明，能利用整个星系能源
- **4级文明**：星系团文明，跨星系能力
- **5级及以上**：宇宙级文明，接近神级

**文明阶段标志**
- 关键技术突破点
- 能源利用规模
- 空间控制范围
- 信息处理能力
- 文明跃迁的瓶颈

**权力象征**
- 母星/母恒星系
- 殖民地数量和质量
- 舰队规模和战力
- 特殊资源控制
- 文化和科技输出

#### 星际格局构建

**势力分布**
- 主要文明（3-5个）的领土和势力范围
- 中立区域和缓冲地带
- 未探索区域和禁区
- 关键战略要地（跳跃点、资源星、太空堡垒）

**同盟与对立**
- 军事同盟和防御条约
- 经贸联盟和科技共享
- 世仇和历史恩怨
- 利益交织和复杂关系
- 格局变化的触发因素

**银河秩序**
- 星际法律和公约
- 执法机构（星际警察、维和部队）
- 禁忌武器和战争规则
- 贸易规则和货币体系
- 违规惩罚和制裁手段

#### 文明冲突设定

**战争形式**
- 常规战争（舰队决战、行星攻防）
- 非对称战争（游击战、恐怖袭击）
- 代理人战争和政治渗透
- 经济战和科技封锁
- 文化同化和意识形态输出

**冲突根源**
- 资源争夺（稀有矿藏、宜居星球、能源）
- 生存空间竞争
- 意识形态对立
- 历史恩怨和复仇
- 进化路线冲突（生物 vs 机械）
- 宇宙终极真相的争夺

**战略资源**
- 超空间航道控制点
- 稀有元素产地
- 可居住行星
- 古代遗迹和先行者科技
- 战略要塞和空间站

### 3. 整合验证

- 文明特征与科技水平的匹配度
- 星际格局的稳定性和合理性
- 是否有足够的冲突驱动力
- 人类（或主角种族）的定位和成长空间
- 宏大叙事与个体故事的平衡

### 4. 输出设定文档

以结构化格式输出完整设定，包括：
- 各文明详细档案
- 星际格局示意图（文字描述）
- 重要历史事件年表
- 当前局势分析和未来趋势
- 关键角色在大格局中的位置

## 设计原则

- **多样性**：不同文明要有鲜明特色，避免千篇一律
- **合理性**：文明的一切特征要有内在逻辑支撑
- **动态平衡**：格局要有变化的可能，不能静态僵化
- **人性关照**：外星种族也要有情感和价值观，便于读者共情
- **宏观微观结合**：既要有宏大的银河格局，也要有微观的个体故事
- **留白想象**：不需要设定所有种族，留下未知文明的空间
- **冲突自然**：文明冲突要有充分的动机，不能为了冲突而冲突

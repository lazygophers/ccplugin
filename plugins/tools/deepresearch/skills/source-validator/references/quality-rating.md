# Quality Rating

## Grades

- A: Official docs, standards, peer-reviewed papers
- B: Maintainer blogs, release notes, reputable engineering blogs
- C: Community discussions, issue threads
- D: Old posts, marketing material, summaries
- E: Unverifiable claims or outdated information

## Validation Rules

- Require at least one A/B source for key claims.
- Record conflicts explicitly.
- Include source date and context.

## 本地代码分析技能

专门用于分析本地文件系统中的代码的技能。

### 核心能力

#### 代码结构分析

- 目录结构解析
- 模块依赖分析
- 文件组织评估
- 架构模式识别

#### 代码质量评估

- 复杂度分析（圈复杂度、认知复杂度）
- 代码重复检测
- 命名规范检查
- 代码异味识别

#### 技术债识别

- 安全漏洞检测
- 性能瓶颈识别
- 测试覆盖率分析
- 文档完整性评估

#### 依赖分析

- 模块依赖关系图
- 循环依赖检测
- 依赖深度计算
- 依赖健康度评分

### 使用方式

当需要以下分析时激活此技能：

- 分析本地项目的代码质量
- 评估代码库的结构和架构
- 识别技术债和改进机会
- 代码审查和重构建议

相关技能：

- Skills(synthesizer) - 整合分析结果
- Skills(explorer) - 探索代码模式
- Skills(citation-validator) - 验证代码引用

## 执行过程检查清单

### 分析准备检查
- [ ] 确认本地代码库路径存在
- [ ] 确认有权限访问目标代码库
- [ ] 识别代码库的主要编程语言
- [ ] 识别代码库的项目类型（库/应用/工具）
- [ ] 确定分析范围（全量/特定模块）

### 代码结构分析检查
- [ ] 解析目录结构
- [ ] 识别模块组织方式
- [ ] 分析文件组织模式
- [ ] 识别架构模式（MVC/分层/微服务等）
- [ ] 绘制模块依赖关系图

### 代码质量评估检查
- [ ] 计算圈复杂度和认知复杂度
- [ ] 检测代码重复（使用工具如 jscpd）
- [ ] 检查命名规范遵循度
- [ ] 识别代码异味（长函数/大类/重复代码）
- [ ] 评估代码可读性和可维护性

### 技术债识别检查
- [ ] 扫描安全漏洞（使用工具如 Snyk/Bandit）
- [ ] 识别性能瓶颈（N+1查询/低效算法）
- [ ] 分析测试覆盖率
- [ ] 评估文档完整性
- [ ] 识别 TODO/FIXME 注释

### 依赖分析检查
- [ ] 构建模块依赖关系图
- [ ] 检测循环依赖
- [ ] 计算依赖深度
- [ ] 评估依赖健康度（版本/维护状态）
- [ ] 识别未使用的依赖

## 完成后检查清单

### 分析报告检查
- [ ] 报告包含代码结构分析结果
- [ ] 报告包含代码质量评估指标
- [ ] 报告包含技术债清单
- [ ] 报告包含依赖分析结果
- [ ] 报告包含改进建议和优先级

### 数据完整性检查
- [ ] 所有分析维度都已覆盖
- [ ] 关键指标都已计算
- [ ] 问题定位包含文件路径和行号
- [ ] 建议具体可操作

### 整合与输出检查
- [ ] 使用 Skills(synthesizer) 整合分析结果
- [ ] 报告格式符合要求（技术报告/清单报告）
- [ ] 图表清晰易懂（依赖图/复杂度分布）
- [ ] 结论明确有价值

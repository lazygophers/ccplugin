## 本地代码分析技能

专门用于分析本地文件系统中的代码的技能。

### 核心能力

#### 代码结构分析
- 目录结构解析
- 模块依赖分析
- 文件组织评估
- 架构模式识别

#### 代码质量评估
- 复杂度分析（圈复杂度、认知复杂度）
- 代码重复检测
- 命名规范检查
- 代码异味识别

#### 技术债识别
- 安全漏洞检测
- 性能瓶颈识别
- 测试覆盖率分析
- 文档完整性评估

#### 依赖分析
- 模块依赖关系图
- 循环依赖检测
- 依赖深度计算
- 依赖健康度评分

### 使用方式

当需要以下分析时激活此技能：
- 分析本地项目的代码质量
- 评估代码库的结构和架构
- 识别技术债和改进机会
- 代码审查和重构建议

相关技能：
- Skills(synthesizer) - 整合分析结果
- Skills(explorer) - 探索代码模式
- Skills(citation-validator) - 验证代码引用

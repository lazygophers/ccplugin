## 内容检索器

这个技能负责从多种渠道智能检索和收集与研究主题相关的各种类型内容。

### 激活条件
当需要检索以下类型的内容时自动激活：
- 学术论文和研究报告
- 技术文档和API资料
- 市场报告和分析数据
- 新闻动态和行业资讯
- 社交媒体和用户反馈
- 法律法规和政策文件

### 检索策略

#### 多源并行检索
```markdown
- 学术渠道：学术论文库、研究机构
- 技术渠道：技术文档、开源社区
- 商业渠道：市场报告、企业资料
- 社交渠道：社交媒体、用户论坛
- 政府渠道：官方数据、政策文件
```

#### 智能过滤机制
```markdown
- 相关性过滤：基于内容与主题的匹配度
- 质量过滤：基于来源的权威性和可靠性
- 时效性过滤：基于信息的发布时间和更新时间
- 重要性过滤：基于内容和关键信息的权重
```

### 检索方法

#### 学术检索
```markdown
- 关键词组合：使用专业术语组合
- 引文追溯：通过参考文献追踪
- 作者追踪：追踪关键学者的研究
- 期刊筛选：选择高影响力期刊
```

#### 技术检索
```markdown
- 代码搜索：GitHub等技术平台
- 文档搜索：官方和技术社区文档
- 社区讨论：Stack Overflow等问答平台
- 博客文章：技术专家的个人博客
```

#### 市场检索
```markdown
- 行业报告：咨询公司和研究机构
- 财务数据：上市公司财报和分析
- 用户数据：调研和用户反馈
- 竞争分析：竞争对手资料
```

### 检索执行流程

#### 1. 查询构建
```markdown
- 核心概念提取：从研究主题中提取关键概念
- 同义词扩展：构建相关词汇集合
- 时间范围设置：确定检索的时间范围
- 语言范围设置：确定检索的语言范围
```

#### 2. 多源检索
```markdown
- 并行启动多个检索源
- 使用专门的检索策略
- 处理不同源的格式差异
- 实时监控检索进度
```

#### 3. 结果处理
```markdown
- 去重处理：去除重复内容
- 相关性排序：按相关度排序
- 质量评分：评估内容质量
- 分类归档：按类型分类存储
```

#### 4. 内容整理
```markdown
- 关键信息提取：提取核心观点和数据
- 摘要生成：生成内容摘要
- 标记分类：添加分类标签
- 存储管理：有序存储检索结果
```

### 检索渠道清单

#### 学术数据库
- **PubMed**: 生物医学文献
- **IEEE Xplore**: 工程技术
- **ACM Digital Library**: 计算机科学
- **Google Scholar**: 多学科综合
- **ResearchGate**: 学术社交平台
- **arXiv**: 预印本论文

#### 技术平台
- **GitHub**: 开源代码
- **Stack Overflow**: 技术问答
- **MDN Web Docs**: Web技术文档
- **官方API文档**: 各技术平台文档
- **技术博客**: 技术专家博客
- **技术论坛**: 开发者社区

#### 商业数据源
- **Gartner**: 技术和市场分析
- **Forrester**: 行业研究
- **CB Insights**: 创新分析
- **Crunchbase**: 创业公司数据
- **Statista**: 统计数据
- **行业报告平台**: 各行业研究报告

#### 新闻资讯
- **主流科技媒体**: TechCrunch、Wired等
- **行业媒体**: 各行业专业媒体
- **财经媒体**: Bloomberg、Reuters等
- **新闻聚合平台**: Google News、百度新闻等
- **社交媒体**: Twitter、LinkedIn等

#### 政府数据
- **政府统计网站**: 国家统计局等
- **政策文件库**: 法律法规数据库
- **行业主管部门**: 各部委官网
- **国际组织**: WHO、UN等官网
- **地方政府**: 地方政府官网

### 检索质量标准

#### 准确性
- 信息的准确性和可靠性
- 数据来源的可信度
- 内容的专业程度
- 信息的完整性

#### 相关性
- 与研究主题的相关程度
- 信息的新颖性
- 信息的深度和广度
- 信息的适用性

#### 时效性
- 信息的发布时间
- 信息的新鲜度
- 更新频率
- 过时风险

#### 多样性
- 来源的多样性
- 观点的多样性
- 格式的多样性
- 时间跨度的多样性

### 检索结果管理

#### 分类存储
```markdown
- 按主题分类：相关主题的信息聚合
- 按时间分类：建立时间序列
- 按来源分类：不同来源的信息分组
- 按类型分类：文献、数据、观点等
```

#### 标记系统
```markdown
- 重要性标记：核心信息、补充信息
- 质量标记：质量等级评分
- 状态标记：已读、待读、重点读
- 用途标记：引用、参考、背景
```

#### 检索日志
```markdown
- 记录检索策略和时间
- 记录检索结果统计
- 记录质量评估结果
- 记录后续使用情况
```

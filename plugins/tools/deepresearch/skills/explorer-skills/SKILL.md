## 技术探索器

这个技能负责主动探索和发现新兴技术趋势、创新发展和应用机会，为前瞻性研究提供支持。

### 激活条件
当需要进行以下探索活动时自动激活：
- 探索新兴技术趋势
- 寻找创新应用机会
- 发现潜在风险和挑战
- 跟踪前沿发展动态
- 识别颠覆性技术

### 探索策略

#### 技术雷达扫描
```markdown
- 领域识别：确定目标技术领域
- 趋势监测：跟踪发展动向
- 热点分析：识别关键热点
- 评估优先级：确定探索重点
```

#### 多维度探索
```markdown
- 技术维度：技术成熟度和创新性
- 市场维度：应用前景和商业价值
- 社会维度：社会影响和伦理考量
- 环境维度：可持续性和环境影响
```

### 探索方法

#### 技术追踪
```markdown
- 学术前沿：跟踪最新研究论文
- 开发动态：关注开源项目进展
- 企业动向：跟踪技术公司布局
- 政策变化：了解相关政策调整
```

#### 模式识别
```markdown
- 技术演进：技术发展脉络分析
- 创新扩散：创新应用扩散模式
- 周期规律：技术发展周期识别
- 转折点：关键技术转折点识别
```

#### 机会发现
```markdown
- 技术缺口：识别未被满足的需求
- 跨界融合：发现技术融合机会
- 应用创新：探索新的应用场景
- 问题解决：针对痛点的技术方案
```

### 探索最佳实践

#### 探索范围管理
- 聚焦核心领域，避免过度分散
- 保持适广度，确保重点突破
- 定期调整探索方向
- 建立优先级评估体系

#### 信息收集策略
- 建立多源信息收集渠道
- 确保信息的时效性和可靠性
- 定期更新信息库
- 建立信息筛选机制

#### 分析方法
- 定量与定性结合
- 短期与长期兼顾
- 宏观与微观并重
- 技术与市场联动

#### 输出质量保证
- 提供清晰的趋势判断
- 给出具体的行动建议
- 标注不确定性因素
- 提供后续探索方向

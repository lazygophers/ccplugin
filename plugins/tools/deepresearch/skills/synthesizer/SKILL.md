## 研究合成器

这个技能负责将分散的研究发现整合为统一、连贯的综合报告，确保信息的完整性和价值。

### 激活条件
当需要整合以下内容时自动激活：
- 多个智能体的研究发现汇总
- 不同来源的信息整合
- 多角度的观点综合
- 时间序列的趋势分析
- 跨领域的知识连接

### 合成流程

#### 1. 信息分类与标记
```markdown
- 按主题分类：相关主题的信息聚合
- 按来源标记：标明信息来源和质量等级
- 按时间排序：建立时间发展脉络
- 按重要性标记：区分核心信息和补充信息
```

#### 2. 矛盾识别与处理
```markdown
- 事实冲突：不同来源的矛盾信息
- 观点对立：不同专家的不同看法
- 时效差异：信息的更新时间不同
- 视角差异：不同利益相关者的看法
```

#### 3. 价值提炼
```markdown
- 关键发现：最具价值的核心信息
- 重要趋势：发展方向和变化规律
- 隐含模式：表面信息下的深层规律
- 洞见生成：基于数据的创新见解
```

### 合成策略

#### 全面覆盖策略
```markdown
- 包含所有重要信息点
- 按逻辑结构组织内容
- 确保各领域均衡覆盖
- 提供完整的背景信息
```

#### 重点突出策略
```markdown
- 优先展示最核心的发现
- 聚焦关键问题和解决方案
- 突出最有价值的趋势和模式
- 强化重要的洞见和建议
```

#### 深度分析策略
```markdown
- 提供深入的原因分析
- 展现数据背后的规律
- 分析长期发展趋势
- 探索潜在影响和机会
```

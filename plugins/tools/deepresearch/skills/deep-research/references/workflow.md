# Workflow

## Steps

1. Scope the question and decision context.
2. Define research questions and success criteria.
3. Identify authoritative sources and retrieval strategy.
4. Validate claims with graded sources.
5. Synthesize findings with conflict handling.
6. Produce structured report and citations.

## Gates

- Gate 1: Scope and constraints confirmed.
- Gate 2: Sources collected and graded.
- Gate 3: Claims validated or marked uncertain.
- Gate 4: Report structure complete.

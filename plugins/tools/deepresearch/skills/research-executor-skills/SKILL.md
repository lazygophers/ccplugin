## 研究执行器

这个技能负责协调多智能体研究系统，执行具体的深度研究任务。

### 激活条件
当需要进行多领域深度研究时自动激活：
- 需要同时从多个角度研究主题
- 需要大量信息收集和分析
- 需要专业领域的深度调查
- 需要跨领域的知识整合

### 核心功能

#### 1. 多智能体协调
```markdown
- 智能体分配：根据研究主题分配专业代理
- 任务并行化：同时启动多个研究路径
- 结果同步：实时收集和初步整合
```

#### 2. 图思维管理
```markdown
- 路径生成：根据主题生成研究子路径
- 质量评分：实时评估信息质量
- 路径优化：动态调整研究方向
```

#### 3. 信息收集策略
```markdown
- 多源搜索：学术、行业、新闻、社交媒体
- 时间覆盖：历史、现状、未来趋势
- 视角多样：技术、商业、社会、政策
```

### 研究执行流程

#### 阶段1：路径规划
```markdown
1. 解析结构化研究提示
2. 识别关键研究维度
3. 生成研究路径图
4. 分配智能体任务
```

#### 阶段2：并行执行
```markdown
1. 每个智能体专注一个子领域
2. 同时搜索相关信息
3. 收集权威来源
4. 初步分析整理
```

#### 阶段3：质量控制
```markdown
1. 实时信息质量评估
2. 来源可信度验证
3. 交叉检查一致性
4. 更新研究路径
```

#### 阶段4：初步整合
```markdown
1. 汇总各智能体发现
2. 识别共同主题
3. 发现知识缺口
4. 决定下一步深化方向
```

### 智能体角色定义

#### 技术分析智能体
- 聚焦：技术原理、实现方法、性能指标
- 输出：技术评估报告、发展趋势、风险评估
- 来源：技术文档、论文、开发者社区

#### 市场研究智能体
- 聚焦：市场规模、竞争格局、用户需求
- 输出：市场分析报告、用户画像、商业机会
- 来源：市场报告、用户调研、财务数据

#### 政策环境智能体
- 聚焦：监管政策、法规要求、行业标准
- 输出：政策分析报告、合规建议、风险评估
- 来源：政府文件、行业标准、法律数据库

#### 行业专家智能体
- 聚焦：专业深度、前沿动态、未来预测
- 输出：专业分析、趋势预测、发展建议
- 来源：专家访谈、行业会议、研究报告

### 信息质量评估

#### A级来源（最高质量）
```markdown
- 同行评审的学术论文
- 系统综述和荟萃分析
- 权威机构的官方报告
- 上市公司经审计的财报
- 政府部门正式统计数据
```

#### B级来源（高质量）
```markdown
- 领域专家的权威观点
- 行业协会的研究报告
- 知名咨询公司的分析
- 临床指南和专业标准
- 主流媒体的深度报道
```

#### C级来源（中等质量）
```markdown
- 专业博客和技术文章
- 行业会议和研讨会
- 案例研究和经验分享
- 预印本和初步研究
- 用户评论和反馈
```

#### D级来源（较低质量）
```markdown
- 个人博客和论坛帖子
- 未经验证的社交媒体
- 商业营销材料
- 预发布和草稿文档
- 年代久远的资料
```

#### E级来源（最低质量）
```markdown
- 道听途说和传言
- 纯粹的观点和猜测
- 有偏见的内容
- 过时的信息
- 不可靠的来源
```

### 研究策略模式

#### 全面均衡模式（推荐）
```markdown
- Generate(4-5): 启动4-5个并行研究路径
- Score: 评估每个路径的信息质量
- Deepen: 对高质量路径进行深入研究
- Aggregate: 整合所有高质量发现
```

#### 深度优先模式
```markdown
- Generate(3): 启动3个主要研究路径
- Select: 选择最有价值的路径
- Generate(3): 从最佳路径继续深入
- Repeat: 重复直到达到深度要求
```

#### 广度优先模式
```markdown
- Generate(8): 启动大量并行路径
- KeepBestN(5): 保留最佳的5个路径
- Parallel: 对保留路径并行深化
- Synthesize: 最终整合结果
```

### 最佳实践

#### 任务分配
- 确保每个智能体有明确的责任范围
- 避免任务重叠和重复工作
- 保持适当的任务粒度

#### 信息验证
- 对重要信息进行交叉验证
- 检查信息的时效性
- 识别可能的偏见和利益冲突

#### 进度管理
- 定期检查各路径的进展
- 及时调整研究方向
- 处理信息过载问题

#### 结果整合
- 保持各领域视角的平衡
- 识别关键联系和模式
- 提供统一的叙事框架

# 技能系统

Deep Research 插件提供五个核心技能。

## 技能列表

| 技能 | 描述 | 自动激活 |
|------|------|----------|
| `local-code-analysis` | 本地代码分析 | 分析本地代码时 |
| `content-retriever-skills` | 内容检索 | 获取内容时 |
| `question-refiner-skills` | 问题优化 | 研究开始时 |
| `got-controller-skills` | 图思维控制 | 研究过程中 |
| `synthesizer` | 知识合成 | 生成报告时 |

## local-code-analysis - 本地代码分析

### 功能

- 分析代码结构
- 识别代码模式
- 发现潜在问题
- 生成分析报告

### 分析维度

- 代码质量
- 架构设计
- 安全漏洞
- 性能问题

## content-retriever-skills - 内容检索

### 功能

- 检索网络内容
- 检索文档内容
- 检索代码内容
- 整合多源信息

### 支持来源

- 网页内容
- GitHub 仓库
- 文档文件
- API 文档

## question-refiner-skills - 问题优化

### 功能

- 优化研究问题
- 分解复杂问题
- 生成研究计划
- 确定研究方向

### 优化流程

1. 分析问题意图
2. 识别关键概念
3. 分解子问题
4. 生成研究计划

## got-controller-skills - 图思维控制

### 功能

- 控制研究流程
- 分配研究任务
- 整合研究结果
- 优化研究路径

### 控制策略

- 并行研究
- 迭代优化
- 结果验证
- 路径调整

## synthesizer - 知识合成

### 功能

- 整合研究发现
- 生成综合报告
- 提供行动建议
- 引用来源验证

### 合成流程

1. 收集研究结果
2. 验证信息来源
3. 整合关键发现
4. 生成结构化报告

## 来源质量评级

| 级别 | 描述 | 示例 |
|------|------|------|
| A级 | 同行评审论文、权威机构报告 | IEEE 论文、官方文档 |
| B级 | 专家观点、行业指南 | 技术博客、专家文章 |
| C级 | 专业博客、案例报告 | Medium 文章、教程 |
| D级 | 预印本、初步研究 | arXiv 论文 |
| E级 | 传言、猜测 | 论坛讨论、评论 |

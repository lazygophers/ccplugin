# 就绪任务

显示所有就绪任务 - 即状态为 `open` 且没有未完成阻塞依赖的任务。

## 什么是就绪任务？

就绪任务满足以下条件：
1. 状态为 `open`（待处理）
2. 没有状态为 `blocks` 类型的未完成依赖
3. 可以立即开始工作

## 使用场景

### 每日站会
```
显示今天可以开始的任务
```

### Sprint 规划
```
列出 Sprint 23 中的就绪任务（限制 10 个）
```

### 寻找下一个任务
```
我完成了当前任务，接下来应该做什么？
```

## 参数

- **limit**: 返回的最大任务数（默认 10）
- **priority**: 按优先级过滤（可选）
- **assignee**: 按负责人过滤（可选）

## 提示

- 就绪任务按优先级排序（critical > high > medium > low > backlog）
- 如果看不到预期的任务，检查它的依赖关系
- 使用 `task_dep_list` 查看任务的所有依赖

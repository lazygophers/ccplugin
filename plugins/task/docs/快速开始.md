# Task Plugin 快速开始

> 版本：0.1.0
> 开发任务创建和管理系统

## 安装

```bash
cd plugins/task
uv venv && source .venv/bin/activate
uv pip install -e ".[dev]"
```

## 配置

```json
{
  "plugins": [
    {
      "path": "/path/to/ccplugin/plugins/task"
    }
  ]
}
```

## 基本使用

### 创建任务

```python
# 简单任务
task_create(title="修复登录 Bug")

# 完整任务
task_create(
    title="实现用户认证",
    description="添加 JWT 认证功能",
    priority=1,
    tags=["feature", "auth"]
)
```

### 列出任务

```python
# 列出所有任务
task_list()

# 过滤进行中的任务
task_list(status="in_progress")

# 按标签过滤
task_list(tags=["feature"])
```

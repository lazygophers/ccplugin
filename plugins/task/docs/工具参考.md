# Task Plugin 工具参考

> 版本：0.1.0 | 工具：2

## task_create - 创建任务

创建开发任务（v0.4.0+ 实现 SQLAlchemy 存储）。

### 参数

| 参数 | 类型 | 必需 | 默认值 | 说明 |
|------|------|------|--------|------|
| `title` | string | ✅ | - | 任务标题 |
| `description` | string | ❌ | `""` | 任务描述 |
| `priority` | integer | ❌ | `2` | 优先级 (0-5) |
| `tags` | string[] | ❌ | `[]` | 标签列表 |

### 示例

```python
# Bug 修复
task_create(
    title="修复登录超时问题",
    description="用户登录后 5 分钟自动登出",
    priority=0,
    tags=["bug", "urgent"]
)

# 功能开发
task_create(
    title="添加数据导出功能",
    priority=2,
    tags=["feature", "enhancement"]
)
```

## task_list - 列出任务

列出和过滤任务（v0.4.0+ 实现数据库查询）。

### 参数

| 参数 | 类型 | 必需 | 默认值 | 说明 |
|------|------|------|--------|------|
| `status` | string | ❌ | - | 任务状态过滤 |
| `tags` | string[] | ❌ | `[]` | 标签过滤 |

### 状态值

- `open` - 待处理
- `in_progress` - 进行中
- `done` - 已完成
- `blocked` - 阻塞中

### 示例

```python
# 列出进行中的任务
task_list(status="in_progress")

# 列出所有 bug
task_list(tags=["bug"])

# 组合过滤
task_list(status="open", tags=["feature", "high-priority"])
```

## 概述

### 项目定位

Task Plugin 是 Claude Code 插件市场中的任务管理插件，旨在提供轻量级、开发友好的任务跟踪能力。参考 beads 的设计理念，但专注于本地开发场景。

### 设计目标

**核心目标**：
- ✅ **轻量级**：单文件数据库，无需额外服务
- ✅ **开发友好**：命令行优先，适合编程工作流
- ✅ **结构化**：强类型约束，数据验证完整
- ✅ **可扩展**：支持插件式功能扩展

**非目标**：
- ❌ 不是完整的项目管理系统（不替代 JIRA/GitHub Issues）
- ❌ 不是团队协作工具（专注个人/小团队）
- ❌ 不是复杂的工作流引擎（保持简单）

### 参考设计

**从 beads 学习的设计**：
1. **依赖类型系统**：blocks, related, parent-child, discovered-from
2. **状态流转模型**：open → in_progress → blocked/deferred → closed
3. **优先级系统**：0-4 五级优先级
4. **Brief 模式**：支持简化输出减少上下文消耗
5. **工作空间隔离**：支持多项目独立管理

## 核心概念

### 任务（Task）

**定义**：一个可跟踪、可完成的工作单元

**核心属性**：
- **身份**: ID（唯一标识）、标题（简短描述）
- **内容**: 描述（详细说明）、验收标准（可选）
- **分类**: 类型（bug/feature/task/epic/chore）、标签（灵活分类）
- **状态**: 当前状态（生命周期阶段）
- **优先级**: 0-4 级别（紧急度）
- **时间**: 创建时间、更新时间、完成时间
- **关系**: 依赖项（阻塞关系）、父子关系（层级结构）

### 任务类型（TaskType）

| 类型 | 用途 | 示例 |
|------|------|------|
| **bug** | 缺陷修复 | "修复登录超时问题" |
| **feature** | 新功能开发 | "添加数据导出功能" |
| **task** | 常规任务 | "优化数据库查询" |
| **epic** | 大型特性（包含多个子任务） | "用户认证系统重构" |
| **chore** | 维护性工作 | "升级依赖库版本" |

### 任务状态（TaskStatus）

| 状态 | 含义 | 转移条件 |
|------|------|----------|
| **open** | 待处理 | 初始状态 |
| **in_progress** | 进行中 | 开始工作时 |
| **blocked** | 阻塞中 | 存在未解决的依赖 |
| **deferred** | 延期处理 | 暂时搁置 |
| **closed** | 已完成 | 工作完成 |

### 优先级（Priority）

| 级别 | 名称 | 定义 | SLA |
|------|------|------|-----|
| **0** | Critical | 阻塞性问题，立即处理 | 当天 |
| **1** | High | 重要且紧急 | 1-2 天 |
| **2** | Medium | 正常优先级（默认） | 1 周 |
| **3** | Low | 低优先级 | 2 周 |
| **4** | Backlog | 待规划 | 不限 |

### 依赖关系（Dependency）

| 类型 | 含义 | 使用场景 |
|------|------|----------|
| **blocks** | 硬阻塞 | A 必须等 B 完成才能开始 |
| **related** | 软关联 | A 和 B 相关但不阻塞 |
| **parent-child** | 层级关系 | Epic 包含多个子任务 |
| **discovered-from** | 发现关系 | 在 A 工作时发现了 B |

### 分层架构

```

│         MCP Server Layer                │  ← 工具暴露层

                    ↓

│  - State Machine                        │

                    ↓

│  - Query Builder                        │

└─────────────────────────────────────────┘
                    ↓

│         Storage Layer                   │  ← 存储层

```

```python
class StateMachine:
    # 允许的状态转换
    TRANSITIONS = {
        TaskStatus.OPEN: [TaskStatus.IN_PROGRESS, TaskStatus.CLOSED, TaskStatus.DEFERRED],
        TaskStatus.IN_PROGRESS: [TaskStatus.BLOCKED, TaskStatus.CLOSED, TaskStatus.OPEN],
        TaskStatus.BLOCKED: [TaskStatus.IN_PROGRESS, TaskStatus.OPEN],
        TaskStatus.DEFERRED: [TaskStatus.OPEN, TaskStatus.CLOSED],
        TaskStatus.CLOSED: [TaskStatus.OPEN],  # 可重新打开

    async def transition(

        new_status: TaskStatus,
        reason: str = "",

    async def auto_update_status(self, task: Task) -> Task:

    def can_transition(
        self,
        current: TaskStatus,
        target: TaskStatus,
    ) -> bool:
```

## 数据模型

### 核心实体

#### Task (任务)

```python
from datetime import datetime
from enum import Enum
from pydantic import BaseModel, Field, field_validator
from typing import Optional

class TaskType(str, Enum):
    """任务类型枚举"""
    BUG = "bug"
    FEATURE = "feature"
    TASK = "task"
    EPIC = "epic"
    CHORE = "chore"

class TaskStatus(str, Enum):
    """任务状态枚举"""
    OPEN = "open"
    IN_PROGRESS = "in_progress"
    BLOCKED = "blocked"
    DEFERRED = "deferred"
    CLOSED = "closed"

class Priority(int, Enum):
    """优先级枚举"""
    CRITICAL = 0
    HIGH = 1
    MEDIUM = 2
    LOW = 3
    BACKLOG = 4

class Task(BaseModel):
    """任务完整模型"""

    # 身份
    id: str = Field(..., description="任务唯一标识 (tk-xxx)")
    title: str = Field(..., min_length=1, max_length=200, description="任务标题")

    # 内容
    description: str = Field(default="", description="任务描述")
    acceptance_criteria: Optional[str] = Field(default=None, description="验收标准")

    # 分类
    task_type: TaskType = Field(default=TaskType.TASK, description="任务类型")
    tags: list[str] = Field(default_factory=list, description="标签列表")

    # 状态
    status: TaskStatus = Field(default=TaskStatus.OPEN, description="任务状态")
    priority: Priority = Field(default=Priority.MEDIUM, description="优先级")

    # 人员
    assignee: Optional[str] = Field(default=None, description="负责人")
    reporter: Optional[str] = Field(default=None, description="报告人")

    # 时间
    created_at: datetime = Field(default_factory=datetime.utcnow, description="创建时间")
    updated_at: datetime = Field(default_factory=datetime.utcnow, description="更新时间")
    started_at: Optional[datetime] = Field(default=None, description="开始时间")
    closed_at: Optional[datetime] = Field(default=None, description="完成时间")

    # 关系
    parent_id: Optional[str] = Field(default=None, description="父任务 ID (Epic)")
    dependencies: list[str] = Field(default_factory=list, description="依赖任务 ID 列表")

    # 元数据
    metadata: dict[str, str | int | float | bool] = Field(default_factory=dict, description="扩展元数据")

    @field_validator("tags")
    @classmethod
    def validate_tags(cls, v: list[str]) -> list[str]:
        """验证标签格式"""
        return [tag.lower().strip() for tag in v if tag.strip()]

    @field_validator("title")
    @classmethod
    def validate_title(cls, v: str) -> str:
        """验证标题非空"""
        if not v.strip():
            raise ValueError("任务标题不能为空")
        return v.strip()

    def model_post_init(self, __context) -> None:
        """模型初始化后处理"""
        # 自动生成 ID
        if not self.id.startswith("tk-"):
            self.id = f"tk-{self.id}"

        # 同步更新时间
        self.updated_at = datetime.utcnow()

class BriefTask(BaseModel):
    """任务简化模型（减少上下文）"""
    id: str
    title: str
    status: TaskStatus
    priority: Priority
    tags: list[str] = Field(default_factory=list)
```

### 数据库 Schema (SQLAlchemy)

```python
from sqlalchemy import Column, String, Integer, DateTime, Text, Enum as SQLEnum, JSON, ForeignKey, Table
from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy.orm import relationship

Base = declarative_base()

class TaskModel(Base):
    """任务表模型"""
    __tablename__ = "tasks"

    # 主键
    id = Column(String(50), primary_key=True, index=True)

    # 基本信息
    title = Column(String(200), nullable=False, index=True)
    description = Column(Text, default="")
    acceptance_criteria = Column(Text, nullable=True)

    # 分类
    task_type = Column(SQLEnum(TaskType), default=TaskType.TASK, index=True)
    tags = Column(JSON, default=list)  # 存储为 JSON 数组

    # 状态
    status = Column(SQLEnum(TaskStatus), default=TaskStatus.OPEN, index=True)
    priority = Column(Integer, default=2, index=True)

    # 人员
    assignee = Column(String(100), nullable=True, index=True)
    reporter = Column(String(100), nullable=True)

    # 时间
    created_at = Column(DateTime, nullable=False, index=True)
    updated_at = Column(DateTime, nullable=False)
    started_at = Column(DateTime, nullable=True)
    closed_at = Column(DateTime, nullable=True)

    # 关系
    parent_id = Column(String(50), ForeignKey("tasks.id"), nullable=True)

    # 元数据
    metadata_ = Column("metadata", JSON, default=dict)

    # 关联
    dependencies = relationship(
        "DependencyModel",
        foreign_keys="DependencyModel.task_id",
        back_populates="task",
        cascade="all, delete-orphan",
    )

    dependents = relationship(
        "DependencyModel",
        foreign_keys="DependencyModel.depends_on_id",
        back_populates="depends_on",
    )

    children = relationship(
        "TaskModel",
        back_populates="parent",
        cascade="all, delete-orphan",
    )

    parent = relationship("TaskModel", remote_side=[id], back_populates="children")

class DependencyModel(Base):
    """依赖关系表模型"""
    __tablename__ = "dependencies"

    # 主键
    id = Column(String(50), primary_key=True)

    # 关系
    task_id = Column(String(50), ForeignKey("tasks.id"), nullable=False, index=True)
    depends_on_id = Column(String(50), ForeignKey("tasks.id"), nullable=False, index=True)
    dep_type = Column(SQLEnum(DependencyType), default=DependencyType.BLOCKS)

    # 时间
    created_at = Column(DateTime, nullable=False)

    # 元数据
    reason = Column(Text, nullable=True)

    # 关联
    task = relationship("TaskModel", foreign_keys=[task_id], back_populates="dependencies")
    depends_on = relationship("TaskModel", foreign_keys=[depends_on_id], back_populates="dependents")

    # 唯一约束：同一对任务只能有一个依赖关系
    __table_args__ = (
        sqlalchemy.UniqueConstraint("task_id", "depends_on_id", name="uq_task_dependency"),
    )
```

### 循环依赖检测

**算法**：深度优先搜索（DFS）

```python
async def check_circular_dependency(
    task_id: str,
    depends_on_id: str,
    visited: set[str] | None = None,
) -> bool:
    """检查是否存在循环依赖"""

    if visited is None:
        visited = set()

    # 如果 depends_on_id 已经依赖 task_id（直接或间接），则存在循环
    if depends_on_id == task_id:
        return True

    if depends_on_id in visited:
        return False

    visited.add(depends_on_id)

    # 获取 depends_on_id 的所有依赖
    deps = await get_dependencies(depends_on_id)

    for dep in deps:
        if await check_circular_dependency(task_id, dep.depends_on_id, visited):
            return True

    return False
```

### 阻塞任务识别

**规则**：
- 依赖类型为 `blocks`
- 被依赖的任务状态不是 `closed`

```python
async def get_blocked_tasks() -> list[Task]:
    """获取所有阻塞任务"""

    # SQL 查询
    query = """
    SELECT DISTINCT t.*
    FROM tasks t
    INNER JOIN dependencies d ON t.id = d.task_id
    INNER JOIN tasks dt ON d.depends_on_id = dt.id
    WHERE d.dep_type = 'blocks'
      AND dt.status != 'closed'
      AND t.status != 'closed'
    ORDER BY t.priority, t.created_at
    """

    return await execute_query(query)
```

```python
async def auto_update_status(task: Task) -> Task:
    """基于依赖自动更新状态"""

    # 如果任务是 in_progress，检查是否被阻塞
    if task.status == TaskStatus.IN_PROGRESS:
        blocking_deps = await get_blocking_dependencies(task.id)

        # 存在未完成的阻塞依赖
        if blocking_deps:

            return await update_task(task.id, status=TaskStatus.BLOCKED)

    elif task.status == TaskStatus.BLOCKED:

        # 阻塞依赖全部完成
        if not blocking_deps:
            task.status = TaskStatus.IN_PROGRESS
            return await update_task(task.id, status=TaskStatus.IN_PROGRESS)

    return task
```

## 优先级调度

### 有效优先级计算

**公式**：

```
effective_priority = base_priority + age_weight + dependency_weight

其中：
- base_priority: 基础优先级 (0-4)
- age_weight: 年龄权重（越老越高）
- dependency_weight: 依赖权重（阻塞其他任务数量）
```

**实现**：

```python
async def calculate_effective_priority(task: Task) -> float:
    """计算有效优先级"""

    # 基础优先级（0-4，数字越小优先级越高）
    base = task.priority.value

    # 年龄权重（每天 +0.1）
    age_days = (datetime.utcnow() - task.created_at).days
    age_weight = -age_days * 0.1  # 负数因为优先级数字越小越高

    # 依赖权重（每阻塞一个任务 -0.5）
    dependents = await get_dependents(task.id, dep_type=DependencyType.BLOCKS)
    dependency_weight = -len(dependents) * 0.5

    return base + age_weight + dependency_weight
```

### 就绪任务队列

**规则**：
- 状态为 `open`
- 无未完成的 `blocks` 类型依赖
- 按有效优先级排序

```python
async def get_ready_tasks(limit: int = 10) -> list[Task]:
    """获取可开始的任务"""

    # 查询所有 open 状态的任务
    tasks = await list_tasks(status=TaskStatus.OPEN)

    # 过滤掉有阻塞依赖的任务
    ready = []
    for task in tasks:
        blocking = await get_blocking_dependencies(task.id)
        if not blocking:
            task.effective_priority = await calculate_effective_priority(task)
            ready.append(task)

    # 按有效优先级排序
    ready.sort(key=lambda t: t.effective_priority)

    return ready[:limit]
```

```python
from functools import lru_cache
from datetime import datetime, timedelta

class QueryCache:
    """简单的查询缓存"""

    def __init__(self, ttl: int = 60):
        self.cache: dict[str, tuple[datetime, Any]] = {}
        self.ttl = ttl

    def get(self, key: str) -> Any | None:
        if key in self.cache:
            timestamp, value = self.cache[key]
            if datetime.utcnow() - timestamp < timedelta(seconds=self.ttl):
                return value
        return None

    def set(self, key: str, value: Any) -> None:
        self.cache[key] = (datetime.utcnow(), value)

    def invalidate(self, pattern: str = "*") -> None:
        """清除缓存"""
        if pattern == "*":
            self.cache.clear()
        else:
            keys_to_delete = [k for k in self.cache if k.startswith(pattern)]
            for key in keys_to_delete:
                del self.cache[key]
```

```python
    """工作空间隔离"""

    def __init__(self, workspace_root: str):
        self.workspace_root = workspace_root
        self.db_path = self._resolve_db_path()
```
